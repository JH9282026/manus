# Tools And Resources

Detailed reference content for the Video Seo Optimization skill.

---

## Tools and Resources


### YouTube SEO Tools

**Free Tools**:
- YouTube Studio (native analytics)
- YouTube Search Suggest
- Google Trends (YouTube search)
- Keyword.io (YouTube suggestions)

**Premium Tools**:
- TubeBuddy (keyword research, optimization scoring, A/B testing)
- VidIQ (SEO tools, analytics, trend alerts)
- Morningfame (growth analytics)
- Social Blade (tracking and statistics)


### Technical SEO Tools

- Google Search Console (video indexing)
- Schema Markup Validators
- Video Sitemap Generators
- PageSpeed Insights (embedded video performance)


### Design Tools

- Canva (thumbnail creation)
- Adobe Photoshop (professional thumbnails)
- Photopea (free Photoshop alternative)
- Remove.bg (background removal)


### Caption and Transcript Tools

- YouTube Auto-Captions (free, starting point)
- Rev (professional transcription)
- Otter.ai (AI transcription)
- Descript (transcription and editing)
