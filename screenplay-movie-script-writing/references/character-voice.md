### Character Voice

Each character should have a distinct voice—vocabulary, rhythm, and speech patterns that reflect their background, personality, and emotional state.

**Generic dialogue:**
```
                         CHARACTER A
          I'm really angry about what happened.

                         CHARACTER B
          I understand why you feel that way.
```

**Distinctive voices:**
```
                         TONY
          You think you can pull that garbage 
          with me? After everything I done for 
          you?

                         MARGARET
          What you've done for me? Is that what 
          we're calling it now?
```
