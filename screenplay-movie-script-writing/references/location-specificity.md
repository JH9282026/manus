### Location Specificity

Be specific enough for production planning but not overly detailed:

**Too vague:**
```
INT. HOUSE - DAY
```

**Appropriate:**
```
INT. SARAH'S HOUSE - LIVING ROOM - DAY
```

**Too specific:**
```
INT. SARAH'S HOUSE - LIVING ROOM - BY THE FIREPLACE NEAR THE WINDOW - DAY
```
