# Microsoft Audience Network Campaigns

Create native advertising campaigns across Microsoft's premium publisher network including MSN, Outlook, and Microsoft Edge.

## Overview

This skill covers Microsoft Audience Network campaign creation, native ad formats, audience targeting, and performance optimization. Use it for brand awareness and consideration campaigns across Microsoft's owned and operated properties.

## Audience Network Overview

### Where Ads Appear

| Property | Monthly Users | Ad Format |
|----------|---------------|-----------|
| **MSN** | 500M+ | Native, display |
| **Microsoft Edge** | 600M+ | Native |
| **Outlook.com** | 400M+ | Native |
| **Microsoft Start** | 500M+ | Native |

### Ad Formats

| Format | Specifications | Use Case |
|--------|----------------|----------|
| **Native** | Image + headline + description | Content discovery |
| **Display** | Standard IAB sizes | Brand awareness |
| **Video** | In-feed video | Engagement |

## Campaign Setup

### Requirements

- **Images**: Landscape (1.91:1), Square (1:1)
- **Headlines**: 25-90 characters
- **Descriptions**: 90 characters
- **Business Name**: 25 characters
- **Logo**: Square, 128x128px minimum

### Audience Targeting

| Targeting Type | Options | Use Case |
|----------------|---------|----------|
| **Demographics** | Age, gender, location | Demographic targeting |
| **Interests** | LinkedIn profile data | Professional targeting |
| **In-Market** | Purchase intent signals | Conversion campaigns |
| **Remarketing** | Website visitors | Re-engagement |

## Performance Benchmarks

| Metric | Good Benchmark |
|--------|----------------|
| **CTR** | 0.3-0.6% |
| **CPC** | $0.50-$2.00 |
| **CPM** | $2-$5 |
