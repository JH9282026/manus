# Music Theory Workflows and Best Practices

Industry standards for composition and analysis.

## Composition Workflow

### Pre-Composition Planning

**Define Objectives:**
- Determine purpose (film cue, song, concert work)
- Identify target audience and genre
- Establish duration and instrumentation
- Consider performance constraints

**Gather Inspiration:**
- Create reference playlist
- Analyze similar works
- Sketch melodic and harmonic ideas
- Develop rhythmic motifs

### Sketching and Development

**Melodic Development:**
- Create strong, memorable themes
- Use motivic development (repetition, sequence, variation)
- Balance stepwise motion with leaps
- Consider phrase structure and cadences

**Harmonic Planning:**
- Establish key and tonal plan
- Map out modulations and key areas
- Choose chord progressions supporting melody
- Plan harmonic rhythm (rate of chord change)

**Rhythmic Structure:**
- Establish meter and tempo
- Create rhythmic motifs
- Balance repetition with variation
- Consider syncopation and rhythmic interest

### Orchestration and Arrangement

**Instrumentation Choices:**
- Select instruments appropriate for style
- Consider range and technical limitations
- Plan for blend and balance
- Use doublings strategically

**Texture Development:**
- Vary between homophonic, polyphonic, and monophonic textures
- Create contrast between sections
- Build and release tension through texture
- Consider register and spacing

## Analysis Workflow

### Initial Listening

**First Impression:**
- Listen without score or analysis
- Note overall form and structure
- Identify key sections and transitions
- Observe emotional arc and character

**Repeated Listening:**
- Focus on different elements (melody, harmony, rhythm)
- Listen for details and subtleties
- Compare different performances
- Take detailed notes

### Harmonic Analysis

**Identify Key:**
- Determine tonic through cadences
- Note key signature and accidentals
- Identify any modulations

**Roman Numeral Analysis:**
- Label each chord with Roman numerals
- Indicate chord quality and inversions
- Mark non-chord tones
- Identify functional harmony (T, PD, D)

**Advanced Harmonic Features:**
- Note secondary dominants
- Identify borrowed chords
- Mark augmented sixth chords
- Observe chromatic harmony

### Melodic Analysis

**Phrase Structure:**
- Identify phrases and periods
- Note antecedent and consequent relationships
- Observe phrase lengths and symmetry
- Mark cadence types

**Motivic Development:**
- Identify primary motifs
- Track motivic transformations
- Note sequence and repetition
- Observe thematic relationships

### Formal Analysis

**Section Identification:**
- Label major sections (A, B, C)
- Identify form type (binary, ternary, sonata, rondo)
- Note transitions and retransitions
- Observe proportions and balance

**Tonal Plan:**
- Map key areas throughout piece
- Identify modulation techniques
- Note relationship between keys
- Observe return to tonic

## Best Practices for Composers

### Melodic Writing

**Singability:**
- Create melodies that are easy to remember and sing
- Balance stepwise motion (60-70%) with leaps
- Resolve large leaps by step in opposite direction
- Use repetition and sequence for coherence

**Contour:**
- Create interesting melodic shapes
- Avoid excessive repetition of same pitch
- Use climax points strategically
- Balance ascending and descending motion

**Rhythm:**
- Vary note values for interest
- Use syncopation judiciously
- Align strong melodic notes with strong beats
- Create rhythmic motifs for unity

### Harmonic Writing

**Voice Leading:**
- Move voices smoothly with minimal motion
- Resolve tendency tones (leading tone to tonic, 7th down by step)
- Avoid parallel fifths and octaves in outer voices
- Use contrary motion between bass and soprano

**Chord Progression:**
- Follow functional harmony principles (T-PD-D-T)
- Use strong progressions (V-I, IV-I, ii-V)
- Avoid weak progressions (I-iii, vi-iii) unless intentional
- Create harmonic rhythm appropriate for style

**Non-Chord Tones:**
- Use passing tones to connect chord tones
- Employ neighbor tones for embellishment
- Apply suspensions for expressive dissonance
- Use anticipations to create forward motion

### Rhythmic Writing

**Metric Clarity:**
- Establish clear meter through accent patterns
- Use strong beats for important harmonic changes
- Align phrase beginnings with strong beats
- Create metric hierarchy

**Rhythmic Variety:**
- Vary note values and patterns
- Use syncopation for interest
- Employ hemiola for metric ambiguity
- Balance activity with rest

**Rhythmic Motifs:**
- Develop characteristic rhythmic patterns
- Use rhythmic augmentation and diminution
- Create rhythmic unity through repetition
- Vary rhythms to maintain interest

## Genre-Specific Applications

### Classical Composition

**Form and Structure:**
- Follow established forms (sonata, rondo, theme and variations)
- Observe period practice for historical styles
- Use appropriate harmonic language
- Consider performance traditions

**Orchestration:**
- Understand instrument capabilities and limitations
- Follow orchestration principles (balance, blend, color)
- Use doublings for reinforcement
- Create clear, playable parts

### Jazz Composition and Arranging

**Harmonic Language:**
- Use extended chords (9ths, 11ths, 13ths)
- Employ tritone substitutions
- Apply modal interchange
- Use chromatic approach chords

**Voicing:**
- Use drop 2 and drop 3 voicings
- Employ rootless voicings for piano
- Create voice leading between chords
- Balance open and closed voicings

**Rhythm Section:**
- Write clear chord symbols
- Indicate specific voicings when needed
- Allow space for improvisation
- Create rhythmic hits and figures

### Pop and Commercial Music

**Song Structure:**
- Use verse-chorus form
- Create memorable hooks
- Build energy toward chorus
- Include bridge for contrast

**Harmonic Simplicity:**
- Use diatonic progressions primarily
- Employ common progressions (I-V-vi-IV)
- Add color with borrowed chords
- Keep harmony supporting melody

**Production Considerations:**
- Write for specific instrumentation
- Consider arrangement density
- Plan for mixing and production
- Create space for vocals

### Film and Media Scoring

**Dramatic Function:**
- Support narrative and emotion
- Use leitmotifs for characters and themes
- Create appropriate mood and atmosphere
- Time music to picture

**Orchestration:**
- Use full orchestra for epic moments
- Employ solo instruments for intimacy
- Create textural variety
- Balance with dialogue and sound effects

**Harmonic Language:**
- Use modal harmony for ambiguity
- Employ chromaticism for tension
- Apply functional harmony for resolution
- Use ostinatos for momentum

## Notation Best Practices

### Score Preparation

**Clarity:**
- Use appropriate note spacing
- Indicate all dynamics and articulations
- Mark tempo changes clearly
- Include rehearsal numbers or letters

**Consistency:**
- Maintain consistent notation style
- Use standard symbols and terminology
- Follow engraving conventions
- Proofread carefully

**Performance Indications:**
- Mark bowings for strings
- Indicate breathing for winds
- Note pedaling for piano
- Include expression markings

### Part Preparation

**Readability:**
- Use appropriate page turns
- Include cues for entrances after rests
- Mark important dynamic changes
- Indicate tempo changes clearly

**Completeness:**
- Include all necessary information
- Add measure numbers
- Mark rehearsal letters
- Include title and composer
