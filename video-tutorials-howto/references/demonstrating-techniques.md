### Demonstrating Techniques

**Show Before You Tell:**
- Demonstrate the action first
- Then explain what happened
- Repeat with narration if needed
- Show common variations

**Hand and Tool Visibility:**
- Keep hands in frame
- Don't block with body
- Use angles that show grip/technique
- Consider mirror shots

**Pacing Physical Demonstrations:**
- Normal speed first
- Slow motion for complex actions
- Repeat difficult steps
- Allow practice pauses
