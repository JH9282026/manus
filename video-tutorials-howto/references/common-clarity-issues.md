### Common Clarity Issues

**Problem: Moving too fast**
- Solution: Build in pauses, slow down
- Edit to add time if needed
- Include "pause here" moments

**Problem: Assuming knowledge**
- Solution: Explain prerequisites
- Define terms first use
- Link to foundational content

**Problem: Non-linear explanation**
- Solution: Follow logical order
- Use step numbers consistently
- Reframe if you backtrack
