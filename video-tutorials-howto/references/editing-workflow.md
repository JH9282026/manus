### Editing Workflow

**1. Assembly Edit:**
- Import all footage
- Arrange in sequence
- Rough cuts only
- Identify usable content

**2. Rough Cut:**
- Remove mistakes and tangents
- Tighten pacing
- Add B-roll and screen recordings
- Basic audio cleanup

**3. Fine Cut:**
- Refine every cut
- Add annotations and graphics
- Polish audio (EQ, compression)
- Add music if appropriate

**4. Final Polish:**
- Color correction
- Audio mix
- Export review
- Final adjustments
