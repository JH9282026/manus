### Handling Complex Procedures

**Break Into Phases:**
```
Phase 1: Preparation
  Step 1, 2, 3

Phase 2: Main Process
  Step 4, 5, 6, 7

Phase 3: Finishing
  Step 8, 9, 10
```

**Provide Overview First:**
- Show what we're building toward
- Preview all phases/steps
- Set expectations for complexity
- Help viewers assess if right for them

**Offer Checkpoints:**
- "At this point, you should see..."
- Compare with expected result
- Opportunity to pause
- Common issues at this stage
