### Structural Patterns

**Linear Step-by-Step:**
```
Step 1 → Step 2 → Step 3 → Step 4 → Result
```
Best for: Sequential processes, recipes, assembly

**Problem-Solution:**
```
Problem → Solution 1 → Solution 2 → Solution 3 → Best Choice
```
Best for: Troubleshooting, optimization, decisions

**Before/After Transformation:**
```
Before State → Process → Transformation → After State
```
Best for: Makeovers, improvements, skill development

**Comparison/Options:**
```
Method A → Method B → Method C → Recommendations
```
Best for: Tool comparisons, approach decisions
