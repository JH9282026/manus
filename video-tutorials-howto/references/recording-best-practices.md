### Recording Best Practices

**Pre-Recording Checklist:**
- [ ] Clean desktop and browser
- [ ] Close unnecessary apps/notifications
- [ ] Set resolution (1920×1080 recommended)
- [ ] Test audio levels
- [ ] Clear browser history/autofill
- [ ] Disable system notifications
- [ ] Close chat/email applications
- [ ] Prepare any needed tabs/windows

**Recording Settings:**
- Resolution: 1920×1080 or 2560×1440
- Frame rate: 30fps (60fps for smooth motion)
- Audio: 48kHz, mono or stereo
- Format: High quality (compress later)
