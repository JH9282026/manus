### Planning a Series

**Series Structure:**
```
Series: [Topic]
├── Intro/Overview
├── Module 1: Foundations
│   ├── Lesson 1.1
│   ├── Lesson 1.2
├── Module 2: Core Skills
│   ├── Lesson 2.1
│   ├── Lesson 2.2
├── Module 3: Advanced
│   ├── Lesson 3.1
│   ├── Lesson 3.2
└── Conclusion/Next Steps
```

**Consistency Elements:**
- Intro/outro consistency
- Visual style matching
- Numbering system
- Cross-references between videos
