
### Building Accessibility into Production

**Pre-Production**:
- Include accessibility in project scope
- Budget for captions and descriptions
- Plan production for accessibility
- Select accessible platforms and tools

**Production**:
- Clear audio recording
- Adequate pauses for description
- Read on-screen text aloud
- Consider visual accessibility
- Document visual elements for description

**Post-Production**:
- Create captions during editing
- Produce audio descriptions
- Test with accessible player
- Quality assurance checks

### Caption Production Workflow

**Internal Caption Production**:
1. Edit video to near-final
2. Generate auto-captions (if using AI assist)
3. Review and correct transcription
4. Add sound effects and music cues
5. Time synchronization
6. Style according to standards
7. Export to required formats
8. Quality review

**External Caption Services**:
- Send final video to service
- Provide context and terminology
- Review returned captions
- Request corrections if needed
- Import into video platform

### Audio Description Workflow

**Planning for Description**:
- Review video for description opportunities
- Identify critical visual information
- Note timing constraints
- Plan description approach

**Production**:
- Write description script
- Review script timing
- Record narration
- Mix with original audio
- Quality review
- Export as separate track or mixed version
