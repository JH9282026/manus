# El Toro Ads Management

Precision IP targeting and programmatic advertising through El Toro platform for household-level digital advertising.

## Overview

El Toro specializes in IP address targeting, allowing advertisers to target specific households with digital ads across devices. This skill covers IP targeting strategies, campaign setup, match-back attribution, and integration workflows.

## Platform Overview

### What is El Toro?
- **IP Targeting**: Match physical addresses to IP addresses
- **Household Precision**: Target specific households
- **Multi-Channel**: Display, video, CTV, mobile
- **Match-Back**: Offline conversion tracking

### Use Cases
- Political campaigns (voter targeting)
- B2B (account-based marketing)
- Local businesses (geographic precision)
- Real estate (neighborhood targeting)
- Automotive (dealership proximity)

## Campaign Setup

### IP List Upload
1. **Prepare Address List**
   - Physical addresses (street, city, state, ZIP)
   - CSV format
   - Minimum 1,000 addresses recommended

2. **Upload to Platform**
   - El Toro matches addresses to IP addresses
   - Match rate typically 60-80%
   - Matched IPs become targetable audience

3. **Campaign Configuration**
   - Select ad formats (display, video, CTV)
   - Set budget and schedule
   - Upload creatives

## Match-Back Attribution

### Offline Conversion Tracking
1. **Campaign Runs**: Ads served to targeted IPs
2. **Action Occurs**: Vote cast, store visit, purchase
3. **Match-Back**: El Toro matches action to exposed household
4. **Reporting**: Attribution and lift analysis

### Metrics

- Lift vs control group
