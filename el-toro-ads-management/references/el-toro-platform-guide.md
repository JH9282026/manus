# El Toro Platform Guide

## Platform Access
- Web-based platform
- Login: https://platform.eltoro.com
- API access available (contact support)

## Campaign Setup Workflow
1. Upload address list
2. Wait for IP matching (24-48 hours)
3. Create campaign
4. Upload creatives
5. Set budget and schedule
6. Launch campaign

## Address List Format
CSV with columns:
- First Name
- Last Name
- Address
- City
- State
- ZIP Code

## Match Rates
- Typical: 60-80%
- Factors: Address quality, recency, location
- Minimum recommended: 1,000 addresses
