# SEO, Outreach & Measurement

## SEO
- Keyword in file name, alt text, title, H1
- Meta description 150-160 chars
- 500+ word contextual post
- Schema markup; internal links
- Submit sitemap

## Outreach
1. Build list of 50-100 relevant publishers
2. Personalize pitch referencing their coverage
3. Offer exclusive data or custom versions
4. Provide embed code and attribution
5. Follow up after 3-5 days

## Repurposing
- Individual social graphics (one stat per image)
- Slide deck for LinkedIn/SlideShare
- Video walkthrough narrating findings
- Twitter thread of key stats
- Blog series expanding each section

## ROI Metrics
| Metric | Target |
|--------|--------|
| Page views | Track 90-day trend |
| Backlinks | 20-50+ quality links |
| Social shares | 500+ total |
| Email signups | 2-5% of visitors |
| Time on page | 2+ minutes |
