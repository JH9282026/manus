# CTV Creative Production Guide

Comprehensive guide to producing high-quality video ads for Connected TV platforms.

---

## Technical Specifications by Platform

### Universal CTV Specs (Safe for All Platforms)

**Video Specifications**
- **Duration**: 15s or 30s (most universally accepted)
- **Resolution**: 1920x1080 (1080p Full HD)
- **Aspect Ratio**: 16:9 (required)
- **File Format**: MP4 container with H.264 codec
- **Frame Rate**: 29.97 fps (NTSC) or 23.976 fps (film)
- **Bitrate**: 8-12 Mbps (1080p)
- **Color Space**: Rec. 709 (HD standard)
- **Scan Type**: Progressive (not interlaced)

**Audio Specifications**
- **Codec**: AAC (Advanced Audio Coding)
- **Channels**: Stereo (2.0)
- **Sample Rate**: 48 kHz
- **Bit Depth**: 16-bit
- **Loudness**: -24 LUFS (±2 LU) per ATSC A/85 standard
- **True Peak**: -2 dBTP maximum

**File Specifications**
- **Max File Size**: 500MB (safe for all platforms)
- **Naming Convention**: `Brand_Campaign_Duration_Version.mp4`
- **Metadata**: Include title, description, advertiser name

### Platform-Specific Variations

**Roku**
- Accepts up to 3840x2160 (4K)
- Max file size: 2GB
- Supports 6s, 15s, 30s, 60s durations
- Bitrate: Up to 20 Mbps for 4K

**Amazon Fire TV**
- Accepts up to 3840x2160 (4K)
- Max file size: 500MB
- Supports 6s, 15s, 30s, 60s durations
- Bitrate: 8-20 Mbps

**Samsung Ads**
- Accepts up to 3840x2160 (4K)
- Max file size: 1GB
- Supports 15s, 30s durations primarily
- Bitrate: 10-20 Mbps for 4K

**Hulu**
- 1920x1080 standard
- Max file size: 1GB
- Supports 15s, 30s, 60s durations
- Bitrate: 8-15 Mbps

**YouTube TV**
- Accepts up to 3840x2160 (4K)
- Max file size: 1GB
- Supports 6s, 15s, 30s, 60s durations
- Bitrate: 8-20 Mbps

## Creative Best Practices for CTV

### Design for the Big Screen

**Visual Hierarchy**
- Use bold, high-contrast visuals
- Minimum text size: 36pt (readable from 10 feet)
- Avoid small details that won't be visible
- Use 1/3 to 1/2 of screen for key messages
- Ensure 10% safe zone around edges

**Typography**
- Sans-serif fonts (better readability)
- Bold or semi-bold weights
- High contrast with background (white on dark, dark on light)
- Limit to 2-3 font families maximum
- Avoid thin or condensed fonts

**Color & Contrast**
- High contrast ratios (minimum 4.5:1 for text)
- Avoid pure white (#FFFFFF) — use off-white (#F0F0F0)
- Avoid pure black (#000000) — use near-black (#0A0A0A)
- Test on actual TV screens, not just computer monitors
- Consider color blindness (8% of males)

### Optimize for Sound-On Viewing

**Audio Strategy**
- 95%+ of CTV viewers have sound enabled
- Use voiceover to reinforce key messages
- Include music to set emotional tone
- Ensure dialogue is clear and intelligible
- Avoid background music that competes with voiceover

**Audio Mixing**
- Voiceover should be 6-10 dB louder than music
- Use compression to maintain consistent levels
- Avoid sudden volume spikes
- Test on TV speakers (not studio monitors)
- Meet -24 LUFS loudness standard (required)

**Sound Design**
- Use sound effects to enhance key moments
- Create audio branding (sonic logo, signature sound)
- Ensure audio quality matches video quality
- Avoid harsh or jarring sounds

### Front-Load Key Information

**First 3 Seconds**
- Show brand logo or product
- Establish visual hook
- Begin voiceover or music
- Capture attention immediately

**First 5 Seconds**
- Communicate primary message
- Show key product benefit
- Establish emotional tone
- Assume some viewers may skip after this point

**Full Duration**
- Reinforce message throughout
- Show brand multiple times (3-5 times in 30s)
- Build to clear call-to-action
- End with strong brand presence

### Call-to-Action Strategies

**QR Codes**
- Minimum size: 20% of screen width
- High contrast (black on white or white on black)
- Display for minimum 5 seconds
- Include text instruction ("Scan to learn more")
- Test scannability from 10 feet away
- Place in corner or bottom third

**URLs**
- Use short, memorable URLs (brand.com/offer)
- Display for minimum 5 seconds
- Large, readable text (minimum 48pt)
- Avoid complex URLs with hyphens or numbers

**Voice Commands**
- "Ask Alexa..." (Amazon Fire TV)
- "Hey Roku..." (Roku devices)
- "Hey Google..." (YouTube TV)
- Keep commands simple and memorable

**Promo Codes**
- Short, memorable codes (6-8 characters)
- Display prominently for minimum 5 seconds
- Repeat in voiceover
- Use brand-relevant codes (BRAND20, SAVE15)

## Production Workflow

### Pre-Production

**1. Creative Brief**
- Define campaign objectives
- Identify target audience
- Determine key message and benefits
- Establish brand guidelines
- Set budget and timeline

**2. Concept Development**
- Brainstorm creative concepts
- Create storyboards
- Write scripts
- Plan shot list
- Identify talent and locations

**3. Platform Planning**
- Determine target CTV platforms
- Review platform-specific requirements
- Plan for multiple durations (15s, 30s)
- Consider interactive elements (QR codes, etc.)

### Production

**4. Video Shoot**
- Shoot in 4K (3840x2160) for future-proofing
- Use 16:9 aspect ratio
- Shoot at 24 fps or 30 fps
- Capture high-quality audio
- Get multiple takes for editing flexibility

**5. Post-Production**
- Edit in 1080p or 4K timeline
- Color grade for Rec. 709 color space
- Mix audio to -24 LUFS
- Add graphics, text, and effects
- Create multiple duration versions

**6. Quality Control**
- Review on actual TV screen
- Check audio levels with loudness meter
- Verify text readability from 10 feet
- Test QR code scannability
- Get stakeholder approvals

### Post-Production

**7. Export & Delivery**
- Export in platform-specific specs
- Create backup versions (different durations)
- Generate thumbnails/preview images
- Prepare metadata (title, description, advertiser)
- Upload to ad platforms or trafficking system

**8. Trafficking**
- Upload creative assets to ad platforms
- Tag with tracking pixels
- Set up QR code or URL tracking
- Verify playback on each platform
- Conduct test campaigns

## Creative Testing & Optimization

### A/B Testing Framework

**Elements to Test**
- Opening hook (first 3 seconds)
- Messaging approach (benefit-focused vs. feature-focused)
- Voiceover style (male vs. female, tone)
- Music style (upbeat vs. calm)
- Call-to-action type (QR code vs. URL vs. promo code)
- Duration (15s vs. 30s)

**Testing Methodology**
- Test one variable at a time
- Run tests for minimum 2 weeks
- Ensure statistical significance (minimum 1,000 impressions per variant)
- Measure completion rate, engagement, and conversions
- Implement winning variant, test next variable

### Performance Metrics

**Creative Quality Indicators**
- **Completion Rate**: 85%+ is good; 90%+ is excellent
- **Engagement Rate**: Clicks, QR scans, or interactions
- **Brand Lift**: Awareness, consideration, or purchase intent increase
- **Conversion Rate**: Website visits, sign-ups, or purchases

**Optimization Actions**

| Metric | Benchmark | If Below Benchmark |
|--------|-----------|--------------------|
| Completion Rate | 85%+ | Shorten duration, improve opening hook, reduce ad length |
| QR Code Scans | 2-5% | Increase QR size, extend display time, add voiceover CTA |
| Website Visits | Varies | Simplify URL, increase display time, test different CTAs |
| Conversions | Varies | Improve landing page, test different offers, refine targeting |

## Creative Refresh Strategy

### When to Refresh Creative

**Frequency-Based**
- Refresh every 4-6 weeks for ongoing campaigns
- Refresh when frequency exceeds 5 impressions per household
- Refresh when completion rate drops 10%+ from baseline

**Performance-Based**
- Refresh when engagement drops 20%+ from baseline
- Refresh when conversion rate declines significantly
- Refresh when brand lift scores plateau

**Seasonal/Event-Based**
- Refresh for holidays and seasonal events
- Refresh for product launches or updates
- Refresh for competitive responses

### Refresh Approaches

**Minor Refresh (Every 4-6 Weeks)**
- Update opening hook or closing CTA
- Change background music
- Swap out b-roll footage
- Update offer or promo code

**Major Refresh (Every 3-6 Months)**
- New creative concept
- New messaging approach
- New talent or spokesperson
- Complete re-shoot

## Interactive CTV Creative

### QR Code Implementation

**Design Guidelines**
- Minimum size: 300x300 pixels (20% of screen width)
- High contrast: Black on white or white on black
- Include border/padding for scannability
- Add text instruction ("Scan for more info")
- Test from 10 feet with multiple devices

**Placement**
- Bottom right corner (most common)
- Bottom left corner (alternative)
- Center bottom (for emphasis)
- Avoid top corners (may be cut off on some TVs)

**Timing**
- Display for minimum 5 seconds
- Ideal: 7-10 seconds
- Pause other motion when QR is on screen
- Include voiceover instruction

**Tracking**
- Use unique QR codes per campaign
- Track scans by platform, creative, and time
- Measure scan-to-conversion rate
- A/B test QR placement and design

### Shoppable Video Ads

**Amazon Fire TV**
- Direct links to Amazon product pages
- "Add to Cart" functionality
- Product images and pricing overlays
- Voice command integration ("Alexa, add to cart")

**Roku**
- Shoppable overlays with product info
- Direct links to brand websites
- In-ad purchase capabilities (select partners)

**Implementation**
- Work with platform reps for setup
- Provide product feeds and images
- Set up conversion tracking
- Test purchase flow thoroughly

### Voice-Activated Ads

**Amazon Fire TV**
- "Alexa, learn more about [brand]"
- "Alexa, add [product] to cart"
- "Alexa, remind me about [offer]"

**Roku**
- "Hey Roku, learn more"
- "Hey Roku, save this offer"

**Best Practices**
- Keep commands simple and memorable
- Repeat command in voiceover
- Display command on screen for 5+ seconds
- Test voice recognition accuracy

## Common Creative Mistakes

### Mistake 1: Small Text
**Problem**: Text too small to read from 10 feet away
**Solution**: Use minimum 36pt font; test on actual TV from couch distance

### Mistake 2: Repurposing Digital Ads
**Problem**: Using 9:16 vertical or 1:1 square ads designed for mobile
**Solution**: Create CTV-specific 16:9 creative; don't letterbox or pillarbox

### Mistake 3: Ignoring Audio Loudness Standards
**Problem**: Ads too loud or too quiet compared to content
**Solution**: Mix to -24 LUFS per ATSC A/85 standard; use loudness meter

### Mistake 4: Weak Opening
**Problem**: Slow build-up; key message doesn't appear until 10+ seconds
**Solution**: Front-load brand and message in first 3-5 seconds

### Mistake 5: Unclear CTA
**Problem**: No clear next step for viewers
**Solution**: Include prominent QR code, URL, or promo code with voiceover instruction

### Mistake 6: Low Production Quality
**Problem**: Looks cheap or amateurish compared to premium content
**Solution**: Invest in professional production; CTV viewers expect TV-quality ads

### Mistake 7: Wrong Aspect Ratio
**Problem**: Using 4:3, 1:1, or 9:16 aspect ratios
**Solution**: Always use 16:9 for CTV; shoot and edit in this format

### Mistake 8: Ignoring Safe Zones
**Problem**: Important elements cut off on some TV screens
**Solution**: Keep critical elements within 90% safe zone (10% margin on all sides)
