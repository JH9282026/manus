# TikTok Marketing API Reference

## Base URL
https://business-api.tiktok.com/open_api/v1.3/

## Campaign Objectives
- REACH: Brand awareness
- TRAFFIC: Website clicks
- APP_INSTALL: App downloads
- CONVERSIONS: Pixel events
- VIDEO_VIEWS: Video engagement
- LEAD_GENERATION: Lead forms

## Video Specs
- Vertical: 9:16 (1080x1920)
- Square: 1:1 (1080x1080)
- Horizontal: 16:9 (1920x1080)
- Duration: 5-60 seconds
- Optimal: 9-15 seconds
