# SEO Copywriting & Optimization

Write search-engine-optimized content that ranks in Google, satisfies user intent, and converts readers into customers through strategic copywriting and on-page optimization.

## Overview

SEO copywriting balances two audiences: search engines and humans. Content must be discoverable (ranking for target keywords), useful (satisfying search intent), and persuasive (driving desired actions). In 2026, Google's algorithms are sophisticated enough that "writing for SEO" and "writing for humans" are largely the same thing — quality content with clear structure and genuine expertise. This skill covers keyword strategy, on-page optimization, content structure, and E-E-A-T principles.

## On-Page SEO Checklist

| Element | Optimization | Character Limit | Priority |
|---------|-------------|-----------------|----------|
| Meta title | Primary keyword near front + compelling hook | 50–60 chars (display) | Critical |
| Meta description | Include keyword + CTA + value prop | 150–160 chars (display) | High |
| H1 heading | One per page, includes primary keyword | No hard limit; be concise | Critical |
| H2 headings | Include secondary keywords naturally | — | High |
| URL slug | Short, descriptive, includes keyword | 3–5 words ideal | High |
| First paragraph | Include primary keyword in first 100 words | — | High |
| Image alt text | Descriptive, include keyword where natural | 125 chars recommended | Medium |
| Internal links | 3–5 relevant internal links per article | — | High |
| External links | 2–3 authoritative external sources | — | Medium |
| Schema markup | Article, FAQ, HowTo as appropriate | — | Medium |

## Content Structure Framework

### Optimal Article Structure

| Section | Purpose | SEO Function | Word Count |
|---------|---------|-------------|-----------|
| H1 title | Declare topic and keyword | Primary keyword targeting | — |
| Introduction | Hook reader, establish relevance | Include keyword, set expectations | 100–150 words |
| Table of contents | Enable navigation | Generates sitelinks | Automatic or manual |
| H2 main sections | Cover primary subtopics | Target secondary keywords | 200–500 words each |
| H3 subsections | Provide detail under H2s | Long-tail keyword opportunities | 100–300 words |
| Key takeaway/summary | Summarize main points | Featured snippet potential | 50–100 words |
| CTA | Drive conversion action | Not SEO-focused, conversion-focused | 50–100 words |

### Featured Snippet Optimization

| Snippet Type | How to Target | Formatting |
|-------------|---------------|-----------|
| Paragraph | Answer question directly in 40–60 words | Place answer immediately after the H2 question |
| List (ordered) | Use numbered steps or ranked items | H2 question + ordered list |
| List (unordered) | Use bullet points for features/items | H2 question + unordered list |
| Table | Provide structured comparison data | H2 question + HTML table |
| Definition | Define the term concisely | "X is [definition]" in first paragraph |

## E-E-A-T Signals (Experience, Expertise, Authoritativeness, Trustworthiness)

| Signal | Implementation | Impact |
|--------|---------------|--------|
| Author byline | Real name, headshot, bio with credentials | Establishes expertise |
| Author page | Dedicated page with publications, credentials, social | Builds authority |
| First-person experience | "In my 10 years managing..." or "When we tested..." | Demonstrates experience |
| Citations and sources | Link to primary research, studies, data | Builds trustworthiness |
| Updated date | Show "Last updated: [date]" prominently | Signals freshness |
| Expert review | "Reviewed by [Expert Name], [Credential]" | Authority signal for YMYL |
| Original research | Include proprietary data, surveys, case studies | Unique expertise |
| About page | Clear company information, team, mission | Trust signal |

## Readability Optimization

| Factor | Target | Tool | Impact |
|--------|--------|------|--------|
| Reading level | Grade 8–10 (Flesch-Kincaid) | Hemingway App, Grammarly | Accessible to wider audience |
| Sentence length | Average 15–20 words | Grammar tools | Easier comprehension |
| Paragraph length | 2–4 sentences max | Manual check | Better scannability |
| Subheadings | Every 200–300 words | Content structure | Improved scanning and SEO |
| Bullet points | Use for lists of 3+ items | Manual formatting | Faster information processing |
| Bold/emphasis | Key phrases and takeaways | Manual formatting | Guides scanning |
| Transition words | 30%+ of sentences | Yoast SEO, manual | Improved flow |

## Content Length Guidelines

| Content Type | Recommended Length | Why | Exception |
|-------------|-------------------|-----|-----------|
| Blog post (informational) | 1,500–2,500 words | Covers topic thoroughly for ranking | Short-form news (500–800 words) |
| Pillar/guide | 3,000–5,000 words | Comprehensive authority piece | None — go deep |
| Product page | 500–1,000 words | Conversion-focused, not info-heavy | Complex products may need more |
| Category page | 300–500 words | Context for product listings | Add FAQ section for SEO |
| Landing page | 500–1,500 words | Enough to persuade and convert | Test short vs. long |

## Best Practices

- **Write for humans, optimize for search** — if content doesn't serve the reader, rankings won't last
- **Match search intent exactly** — check what ranks on page 1 and create something 10× better
- **Use the keyword naturally** — if it reads awkwardly, you've over-optimized
- **Update content regularly** — refresh top-performing content every 6–12 months
- **Build internal links proactively** — every new article should link to and receive links from 3+ existing pages
- **Don't keyword-stuff** — Google penalizes unnatural keyword density; aim for 1–2% naturally
