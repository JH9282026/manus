# Real Estate Investment Metrics and Financial Analysis

## Core Investment Metrics

### Cash Flow Analysis
**Formula:** Net Operating Income - Debt Service
**Importance:** Measures actual cash generated monthly/annually
**Target:** Positive cash flow for sustainability

### Capitalization Rate (Cap Rate)
**Formula:** NOI / Property Value
**Typical Range:** 4-10% depending on property type and location
**Use:** Compare properties, assess risk/return

### Cash-on-Cash Return
**Formula:** Annual Pre-Tax Cash Flow / Total Cash Invested
**Target:** 8-12% for most markets
**Advantage:** Measures actual cash return on investment

### Return on Investment (ROI)
**Formula:** (Gain - Cost) / Cost × 100%
**Includes:** Cash flow, appreciation, principal paydown
**Timeframe:** Typically calculated over 5-10 years

### Internal Rate of Return (IRR)
**Definition:** Annualized rate of return accounting for timing of cash flows
**Advantage:** Considers time value of money
**Use:** Compare investments with different holding periods

## Conclusion
Understanding and calculating these metrics enables data-driven investment decisions and portfolio optimization.
