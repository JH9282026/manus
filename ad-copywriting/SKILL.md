# Ad Copywriting (PPC, Social, Display)

Create compelling, concise advertising copy that captures attention, communicates value, and drives action within strict character limits and platform constraints.

## Overview

This skill combines direct response copywriting principles with platform-specific best practices to maximize return on advertising spend. Use it to create Google Search Ads, Meta (Facebook/Instagram) ads, LinkedIn Sponsored Content, YouTube video ad scripts, display network copy, and programmatic advertising variations.

## Core Copywriting Principles

### Character Efficiency

Maximize impact within character limits:
- Lead with the strongest benefit
- Use power words (Free, New, Proven, Instant)
- Eliminate filler words
- Test abbreviations vs. full words

### Headline Formulas

1. **Benefit + Timeframe**: "Lose 10 Pounds in 30 Days"
2. **Problem-Agitate-Solve**: "Tired of [Problem]? Try [Solution]"
3. **Social Proof**: "Join 50,000+ Happy Customers"
4. **Curiosity Gap**: "The Secret to [Desired Outcome]"
5. **Direct Offer**: "50% Off Today Only"

### CTA Best Practices

Use action verbs matched to funnel stage:
- **Awareness**: Learn More, Discover, See How
- **Consideration**: Get Free Guide, Compare Plans, Watch Demo
- **Conversion**: Buy Now, Start Free Trial, Get Quote

## Campaign Objective Alignment

| Objective | Copy Focus | CTA Type |
|-----------|------------|----------|
| Awareness | Brand story, problem education | Learn More |
| Traffic | Value proposition, curiosity | Visit Site |
| Engagement | Emotional triggers, relatability | Like/Comment |
| Leads | Offer value, reduce friction | Sign Up Free |
| Conversions | Urgency, social proof, benefits | Buy Now |

## A/B Testing Framework

Test one element at a time:
1. **Headlines**: Test benefit angles, emotional appeals, offers
2. **Descriptions**: Test features vs. benefits, long vs. short
3. **CTAs**: Test verbs, urgency, personalization
4. **Offers**: Test discounts, free trials, guarantees

## Platform-Specific Quick Reference

### Google Search Ads (RSA)

Create 15 headlines (30 chars) and 4 descriptions (90 chars):
- Include target keywords naturally
- Vary headline angles (benefit, feature, offer, social proof)
- Use keyword insertion where appropriate
- Pin critical messages to specific positions

### Meta Ads

Primary text (125 chars optimal, 500 max):
- Hook in first line (visible before "See More")
- Use emojis strategically
- Include social proof
- Clear value proposition

### LinkedIn Ads

Professional tone with B2B focus:
- Lead with business outcomes
- Use industry terminology appropriately
- Reference authority and credibility
- Speak to decision-maker pain points

## Quality Checklist

- [ ] Within platform character limits
- [ ] Clear value proposition in every variant
- [ ] Strong, specific CTA
- [ ] Keywords incorporated naturally (for search)
- [ ] Emotional triggers balanced with benefits
- [ ] Compliant with platform policies
- [ ] Consistent with landing page messaging
- [ ] Sufficient variety for algorithm optimization
