# Working Capital Management

Comprehensive guide to managing short-term assets and liabilities to ensure liquidity, operational efficiency, and profitability.

---

## Working Capital Fundamentals

### Definitions

**Gross Working Capital**: Total current assets

**Net Working Capital (NWC)**:
```
NWC = Current Assets - Current Liabilities
    = (Cash + Marketable Securities + Accounts Receivable + Inventory)
      - (Accounts Payable + Short-term Debt + Accrued Expenses)
```

**Operating Working Capital**:
```
Operating WC = (Accounts Receivable + Inventory) - Accounts Payable
```

Excludes cash and short-term debt (focuses on operating cycle).

### Working Capital Trade-offs

**High Working Capital**:
- *Advantages*: Lower liquidity risk, operational flexibility, customer satisfaction
- *Disadvantages*: Higher carrying costs, lower returns, capital tied up

**Low Working Capital**:
- *Advantages*: Higher returns, efficient capital use, lower carrying costs
- *Disadvantages*: Higher liquidity risk, potential stockouts, strained supplier relationships

**Optimal Level**: Balance liquidity needs with profitability goals.

## Cash Conversion Cycle (CCC)

### CCC Formula

```
CCC = DIO + DSO - DPO

Where:
- DIO = Days Inventory Outstanding
- DSO = Days Sales Outstanding
- DPO = Days Payable Outstanding
```

### Component Calculations

**Days Inventory Outstanding (DIO)**:
```
DIO = (Average Inventory / Cost of Goods Sold) × 365

Or:
DIO = 365 / Inventory Turnover

Where:
Inventory Turnover = COGS / Average Inventory
```

**Days Sales Outstanding (DSO)**:
```
DSO = (Average Accounts Receivable / Revenue) × 365

Or:
DSO = 365 / Receivables Turnover

Where:
Receivables Turnover = Revenue / Average Accounts Receivable
```

**Days Payable Outstanding (DPO)**:
```
DPO = (Average Accounts Payable / Cost of Goods Sold) × 365

Or:
DPO = 365 / Payables Turnover

Where:
Payables Turnover = COGS / Average Accounts Payable
```

### CCC Interpretation

**Positive CCC**: Company pays suppliers before collecting from customers (requires financing)
**Negative CCC**: Company collects from customers before paying suppliers (generates financing)

**Example**:
- DIO = 45 days (inventory sits 45 days)
- DSO = 30 days (customers pay in 30 days)
- DPO = 60 days (company pays suppliers in 60 days)
- CCC = 45 + 30 - 60 = 15 days

**Interpretation**: Cash is tied up for 15 days in operating cycle.

### CCC Optimization Strategies

**Reduce DIO**:
- Improve demand forecasting
- Implement just-in-time (JIT) inventory
- Reduce product variety (SKU rationalization)
- Improve production efficiency
- Liquidate slow-moving inventory

**Reduce DSO**:
- Tighten credit standards
- Offer early payment discounts
- Improve collection processes
- Use electronic invoicing
- Factor receivables (sell to third party)

**Increase DPO**:
- Negotiate longer payment terms
- Take advantage of payment terms (pay on due date, not early)
- Consolidate suppliers for better terms
- Use purchasing cards for small purchases

**Caution**: Don't optimize one component at expense of others
- Extending DPO too much can damage supplier relationships
- Reducing inventory too much can cause stockouts
- Tightening credit can reduce sales

## Cash Management

### Cash Management Objectives

1. **Maintain Liquidity**: Meet obligations as they come due
2. **Minimize Idle Cash**: Reduce opportunity cost
3. **Optimize Float**: Accelerate collections, delay disbursements
4. **Invest Excess Cash**: Earn returns on temporary surplus

### Cash Budget

Project cash inflows and outflows to identify surpluses and shortfalls.

**Format**:

| Item | Jan | Feb | Mar | Q1 Total |
|------|-----|-----|-----|----------|
| **Cash Inflows** | | | | |
| Collections from sales | | | | |
| Other income | | | | |
| **Total Inflows** | | | | |
| **Cash Outflows** | | | | |
| Payments to suppliers | | | | |
| Payroll | | | | |
| Rent | | | | |
| Other expenses | | | | |
| Capital expenditures | | | | |
| **Total Outflows** | | | | |
| **Net Cash Flow** | | | | |
| Beginning Cash | | | | |
| **Ending Cash** | | | | |
| Minimum Cash Required | | | | |
| **Surplus (Shortfall)** | | | | |

**Uses**:
- Identify financing needs in advance
- Plan short-term investments
- Negotiate credit lines
- Time major expenditures

### Float Management

**Types of Float**:

*Collection Float* (delay in receiving cash):
- **Mail Float**: Time for payment to reach company
- **Processing Float**: Time to process and deposit payment
- **Availability Float**: Time for bank to clear payment

*Disbursement Float* (delay in cash leaving account):
- Time between writing check and funds leaving account

**Float Reduction Techniques**:

| Technique | Description | Impact |
|-----------|-------------|--------|
| Lockbox System | Customers send payments to PO box; bank processes | Reduces mail and processing float |
| Electronic Payments | ACH, wire transfers, credit cards | Eliminates mail float |
| Concentration Banking | Multiple collection points, funds swept to central account | Faster access to funds |
| Zero-Balance Accounts | Subsidiary accounts with zero balance; funded as needed | Centralizes cash management |

### Optimal Cash Balance Models

**Baumol Model** (for predictable cash flows):
```
Optimal Cash Balance = √[(2 × F × T) / i]

Where:
- F = fixed cost per transaction (selling securities)
- T = total cash needed per period
- i = interest rate on marketable securities
```

**Miller-Orr Model** (for uncertain cash flows):
```
Return Point = Lower Limit + (1/3 × Spread)
Upper Limit = Lower Limit + Spread

Where:
Spread = 3 × [(3/4 × Transaction Cost × Variance of Cash Flows) / Interest Rate]^(1/3)
```

**Policy**:
- When cash reaches upper limit: Buy securities to return to return point
- When cash reaches lower limit: Sell securities to return to return point

### Short-Term Investment Options

| Investment | Maturity | Risk | Yield | Liquidity |
|------------|----------|------|-------|----------|
| Treasury Bills | 4 weeks to 1 year | Lowest | Lowest | Highest |
| Commercial Paper | 1 to 270 days | Low | Low-Medium | High |
| Certificates of Deposit | 1 month to 5 years | Low (if FDIC insured) | Low-Medium | Medium |
| Money Market Funds | Overnight | Low | Low | Highest |
| Repurchase Agreements | Overnight to 30 days | Low | Low | High |

**Selection Criteria**:
- Safety (preservation of principal)
- Liquidity (quick conversion to cash)
- Yield (return on investment)

## Accounts Receivable Management

### Credit Policy Components

**1. Credit Standards**: Criteria for extending credit
- Financial ratios (current ratio, debt-to-equity)
- Credit scores
- Payment history
- Industry and business risk

**2. Credit Terms**: Payment conditions
- Format: "2/10, net 30" means 2% discount if paid within 10 days, full payment due in 30 days
- Industry norms vary

**3. Collection Policy**: Procedures for collecting overdue accounts
- Reminder letters
- Phone calls
- Collection agencies
- Legal action

### Credit Analysis: The 5 Cs

| Factor | Assessment |
|--------|------------|
| **Character** | Willingness to pay (credit history, reputation) |
| **Capacity** | Ability to pay (cash flow, income) |
| **Capital** | Financial resources (net worth, equity) |
| **Collateral** | Assets to secure credit |
| **Conditions** | Economic environment, industry trends |

### Credit Decision Framework

**Incremental Analysis**:

Extend credit if:
```
Incremental Profit > Incremental Cost

Where:
Incremental Profit = (Additional Sales × Profit Margin) × (1 - Default Rate)
Incremental Cost = (Additional Receivables × Cost of Capital) + (Default Rate × Sales)
```

**Example**:
- Proposed credit policy will increase sales by $1M
- Profit margin: 20%
- Expected default rate: 5%
- Cost of capital: 10%
- Average collection period: 45 days

**Analysis**:
```
Incremental Profit = ($1M × 0.20) × (1 - 0.05) = $190,000

Incremental Cost:
- Carrying cost = ($1M × 45/365) × 0.10 = $12,329
- Default cost = $1M × 0.05 = $50,000
- Total cost = $62,329

Net Benefit = $190,000 - $62,329 = $127,671
```

**Decision**: Extend credit (positive net benefit)

### Early Payment Discounts

**Cost of Forgoing Discount**:
```
Cost = [Discount % / (1 - Discount %)] × [365 / (Credit Period - Discount Period)]
```

**Example**: Terms "2/10, net 30"
```
Cost = [0.02 / 0.98] × [365 / 20]
     = 0.0204 × 18.25
     = 37.2% annual rate
```

**Interpretation**: Customers forgoing the discount are effectively paying 37.2% annual interest.

**Decision for Company**:
- Offer discount if cost < cost of capital
- Example: If cost of capital is 10%, offering 2/10 net 30 is expensive (37.2% > 10%)
- Consider smaller discount (e.g., 1/10 net 30 = 18.4% cost)

### Receivables Monitoring

**Aging Schedule**:

| Age Category | Amount | % of Total | Action |
|--------------|--------|------------|--------|
| 0-30 days | $500,000 | 50% | Normal |
| 31-60 days | $300,000 | 30% | Reminder |
| 61-90 days | $150,000 | 15% | Phone call |
| Over 90 days | $50,000 | 5% | Collection agency |
| **Total** | **$1,000,000** | **100%** | |

**Red Flags**:
- Increasing proportion in older categories
- Rising DSO
- Concentration in few customers

### Receivables Financing

**Pledging**:
- Use receivables as collateral for loan
- Company retains ownership and collection responsibility
- Typical advance: 70-80% of receivables

**Factoring**:
- Sell receivables to factor (third party)
- Factor assumes collection responsibility
- Two types:
  - *With recourse*: Company liable if customer doesn't pay
  - *Without recourse*: Factor bears default risk
- Typical advance: 80-90% of receivables
- Cost: 1-3% of receivables plus interest

**When to Use**:
- Need immediate cash
- Lack credit analysis expertise
- Want to outsource collections
- Seasonal businesses

## Inventory Management

### Inventory Types

**Raw Materials**: Inputs to production
**Work-in-Process (WIP)**: Partially completed goods
**Finished Goods**: Completed products ready for sale

### Inventory Costs

**Carrying Costs** (holding inventory):
- Storage costs (warehouse, utilities)
- Insurance
- Obsolescence and spoilage
- Opportunity cost of capital
- Typical range: 20-30% of inventory value annually

**Ordering Costs** (placing orders):
- Administrative costs
- Shipping and handling
- Receiving and inspection

**Shortage Costs** (running out of stock):
- Lost sales
- Customer dissatisfaction
- Production disruptions
- Emergency orders (higher cost)

### Economic Order Quantity (EOQ)

Optimal order size that minimizes total inventory costs.

**EOQ Formula**:
```
EOQ = √[(2 × D × S) / H]

Where:
- D = annual demand (units)
- S = ordering cost per order
- H = holding cost per unit per year
```

**Total Cost**:
```
Total Cost = (D/Q) × S + (Q/2) × H + D × P

Where:
- Q = order quantity
- P = purchase price per unit
```

**Reorder Point**:
```
Reorder Point = Lead Time Demand + Safety Stock
              = (Daily Demand × Lead Time in Days) + Safety Stock
```

**Example**:
- Annual demand: 10,000 units
- Ordering cost: $100 per order
- Holding cost: $5 per unit per year
- Lead time: 10 days
- Daily demand: 10,000 / 365 = 27.4 units

**Calculation**:
```
EOQ = √[(2 × 10,000 × $100) / $5]
    = √[400,000]
    = 632 units

Number of Orders = 10,000 / 632 = 15.8 orders per year

Reorder Point = 27.4 × 10 = 274 units (assuming no safety stock)
```

### Just-in-Time (JIT) Inventory

**Concept**: Receive inventory exactly when needed, minimizing holding costs.

**Requirements**:
- Reliable suppliers
- Short lead times
- High-quality inputs (no defects)
- Stable demand
- Efficient logistics

**Benefits**:
- Lower carrying costs
- Reduced warehouse space
- Less obsolescence
- Improved quality (defects detected quickly)

**Risks**:
- Vulnerable to supply disruptions
- Less flexibility for demand spikes
- Requires strong supplier relationships

### ABC Inventory Analysis

Classify inventory by importance to focus management attention.

**Classification**:
- **A Items**: 10-20% of items, 70-80% of value → Tight control, frequent review
- **B Items**: 30-40% of items, 15-25% of value → Moderate control
- **C Items**: 40-50% of items, 5-10% of value → Simple controls, bulk orders

**Management Approach**:

| Category | Control Level | Order Frequency | Safety Stock |
|----------|---------------|-----------------|-------------|
| A | Tight | Frequent, small orders | Low |
| B | Moderate | Moderate | Moderate |
| C | Loose | Infrequent, large orders | High |

## Accounts Payable Management

### Payment Timing Strategy

**Factors to Consider**:
- Early payment discounts
- Supplier relationships
- Credit rating impact
- Cash availability

**Decision Framework**:

**If Discount Offered**:
- Calculate cost of forgoing discount
- Take discount if cost > cost of capital
- Example: 2/10 net 30 has 37.2% cost → almost always take discount

**If No Discount**:
- Pay on due date (not early, not late)
- Maximize use of supplier financing
- Maintain good credit standing

### Supplier Relationship Management

**Benefits of Good Relationships**:
- Better payment terms
- Priority during shortages
- Flexibility in emergencies
- Quality improvements
- Cost reductions

**Best Practices**:
- Pay on time (or early if discount)
- Communicate proactively about issues
- Consolidate purchases for volume discounts
- Long-term contracts for key suppliers
- Collaborative planning

### Stretching Payables

**Stretching**: Paying later than terms specify

**Pros**:
- Increases DPO
- Provides free financing
- Improves cash flow

**Cons**:
- Damages supplier relationships
- May lose discounts
- Can hurt credit rating
- Risk of supply disruptions

**When Acceptable**:
- Industry norm
- Supplier has excess capacity
- Strong negotiating position
- Temporary cash crunch

**When to Avoid**:
- Key strategic suppliers
- Sole-source suppliers
- When seeking credit line increases
- If already have poor payment reputation

## Short-Term Financing

### Types of Short-Term Financing

**Spontaneous Financing** (arises naturally from operations):
- Accounts payable
- Accrued expenses (wages, taxes)

**Negotiated Financing** (requires explicit arrangement):
- Bank loans
- Commercial paper
- Secured loans

### Bank Loans

**Line of Credit**:
- Pre-approved borrowing limit
- Draw and repay as needed
- Commitment fee on unused portion (0.25-0.5%)
- Annual cleanup clause (pay off for period)

**Revolving Credit Agreement**:
- Formal, legally binding line of credit
- Higher commitment fee
- More reliable access

**Transaction Loan**:
- Single-purpose loan
- Specific amount and term
- Repayment schedule

### Loan Interest Calculations

**Simple Interest**:
```
Interest = Principal × Rate × Time
Effective Rate = Stated Rate
```

**Discount Interest**:
```
Interest Paid Upfront = Principal × Rate × Time
Proceeds = Principal - Interest
Effective Rate = Interest / Proceeds
```

**Example**: $100,000 loan, 10% rate, 1 year
```
Interest = $10,000
Proceeds = $90,000
Effective Rate = $10,000 / $90,000 = 11.1%
```

**Compensating Balance**:
```
Effective Rate = Interest / (Principal - Compensating Balance)
```

**Example**: $100,000 loan, 10% rate, 20% compensating balance
```
Interest = $10,000
Usable Funds = $80,000
Effective Rate = $10,000 / $80,000 = 12.5%
```

### Commercial Paper

**Characteristics**:
- Unsecured short-term debt
- Issued by large, creditworthy companies
- Maturity: 1-270 days (typically 30-90 days)
- Sold at discount (like T-bills)
- Lower cost than bank loans

**Requirements**:
- Strong credit rating (A2/P2 or better)
- Backup line of credit
- Minimum issue size (~$100,000)

**Advantages**:
- Lower interest cost
- No compensating balance
- Prestige (signals financial strength)

**Disadvantages**:
- Only for large, strong companies
- Impersonal (no relationship)
- Refinancing risk

### Secured Short-Term Loans

**Accounts Receivable Financing**:
- Pledging or factoring (see Receivables section)

**Inventory Financing**:
- **Blanket Lien**: All inventory as collateral
- **Trust Receipt**: Specific items as collateral; borrower holds in trust
- **Warehouse Financing**: Inventory stored in warehouse; lender controls

**Typical Advance Rates**:
- Finished goods: 60-80%
- Raw materials: 50-70%
- Work-in-process: 0-50%

## Liquidity Ratios

### Current Ratio

```
Current Ratio = Current Assets / Current Liabilities
```

**Interpretation**:
- > 2.0: Strong liquidity (may indicate excess working capital)
- 1.5-2.0: Adequate liquidity
- < 1.0: Liquidity concerns

**Limitations**:
- Doesn't consider quality of current assets
- Can be manipulated (window dressing)
- Industry-specific norms vary

### Quick Ratio (Acid-Test Ratio)

```
Quick Ratio = (Current Assets - Inventory) / Current Liabilities

Or:
Quick Ratio = (Cash + Marketable Securities + Accounts Receivable) / Current Liabilities
```

**Interpretation**:
- > 1.0: Can meet short-term obligations without selling inventory
- < 1.0: May need to sell inventory or obtain financing

**More Conservative**: Excludes inventory (least liquid current asset)

### Cash Ratio

```
Cash Ratio = (Cash + Marketable Securities) / Current Liabilities
```

**Interpretation**:
- Most conservative liquidity measure
- Shows ability to pay obligations immediately
- Typically 0.2-0.5 for healthy companies

### Working Capital Turnover

```
WC Turnover = Revenue / Average Net Working Capital
```

**Interpretation**:
- Higher ratio: More efficient use of working capital
- Lower ratio: Excess working capital or inefficiency

## Working Capital Policies

### Aggressive vs. Conservative Policies

**Aggressive Policy** (low working capital):
- Minimal cash balances
- Tight credit standards
- Low inventory levels
- Maximum use of short-term debt
- **Result**: Higher returns, higher risk

**Conservative Policy** (high working capital):
- Large cash balances
- Lenient credit standards
- High inventory levels
- More long-term financing
- **Result**: Lower returns, lower risk

### Matching Principle

**Maturity Matching**: Match asset life with financing maturity
- Permanent current assets + fixed assets → Long-term financing
- Temporary current assets → Short-term financing

**Benefits**:
- Reduces refinancing risk
- Aligns cash flows
- Balances cost and risk

## International Working Capital Management

### Challenges

- Multiple currencies
- Different payment systems
- Varying credit practices
- Political and economic risk
- Complex regulations

### Strategies

**Centralized Cash Management**:
- Pool cash from subsidiaries
- Invest centrally for better rates
- Reduce total cash holdings

**Netting**:
- Offset payables and receivables between subsidiaries
- Reduce number of transactions
- Lower transaction costs and FX exposure

**Leading and Lagging**:
- **Leading**: Pay early (if currency expected to strengthen)
- **Lagging**: Pay late (if currency expected to weaken)
- Natural hedge against FX risk
