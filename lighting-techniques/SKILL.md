# Lighting Techniques

Design and execute professional lighting setups for video, film, and photography production.

## Overview

This skill provides actionable frameworks for planning, rigging, and refining lighting across production scenarios — from single-subject interviews to multi-location narrative shoots. It covers fixture selection, modifier usage, color temperature management, and exposure-lighting coordination.

## Three-Point Lighting Framework

### Key Light

- Primary illumination source — sets exposure and shadow direction.
- Place 30°–45° off camera axis horizontally, 30°–45° above subject eye line.
- Soft sources (softbox, book light) for flattering look; hard sources (Fresnel, open face) for dramatic contrast.
- Size rule: larger source relative to subject = softer shadows.

### Fill Light

- Reduces shadow density on the opposite side of the key.
- Methods: dedicated fixture at lower intensity, bounce card, negative fill (black flag to deepen shadows).
- Ratio control: measure with a light meter or use false color on monitor.

| Key-to-Fill Ratio | Look | Common Use |
|-------------------|------|------------|
| 1:1 | Flat, even | Beauty, corporate |
| 2:1 | Subtle modeling | Interviews, documentary |
| 4:1 | Dramatic contrast | Narrative, thriller |
| 8:1+ | Deep noir | Horror, stylized |

### Back / Rim / Hair Light

- Separates subject from background with an edge highlight.
- Position directly behind subject aimed at head/shoulders, flag to prevent lens flare.
- Intensity typically 0.5–1 stop above key for visible rim.

## Color Temperature and Gel Management

| Source | Approximate Color Temp | CRI/TLCI Target |
|--------|----------------------|------------------|
| Tungsten | 3200K | CRI 100 (native) |
| Daylight / HMI | 5600K | CRI 90+ |
| Fluorescent (office) | 4100K (green spike) | Varies — use minus-green gel |
| LED bi-color | 2700K–6500K dial | CRI 95+ / TLCI 90+ |
| Practicals (lamps) | 2700K–3000K | Match with 1/4 CTO on key |

### Gel Quick Reference

- **CTO (Color Temperature Orange):** Converts daylight to tungsten. Full CTO = 5600K → 3200K.
- **CTB (Color Temperature Blue):** Converts tungsten to daylight. Full CTB = 3200K → 5600K.
- **Plus/Minus Green:** Corrects fluorescent green spike or adds green to match.
- **ND Gel:** Reduces intensity without color shift on windows or hot practicals.

## Modifier Selection Guide

| Modifier | Effect | Best For |
|----------|--------|----------|
| Softbox (rectangular) | Soft wrap, rectangular catchlight | Interviews, beauty |
| Lantern / China ball | 360° omnidirectional soft | Overhead ambient, walk-and-talk |
| Fresnel with barn doors | Controllable hard beam | Dramatic key, spot accents |
| Bounce board (white) | Passive soft fill | Daylight exteriors, budget fill |
| Book light (bounce through diffusion) | Ultra-soft large source | High-end beauty, fashion |
| Grid / egg crate | Narrows spill angle | Controlling light in tight spaces |
| Flags / cutters | Negative fill, light shaping | Creating shadow, contrast |
| Scrim (single/double) | Reduces intensity 1/2 or 1 stop | Dimming without changing fixture |

## Lighting for Specific Scenarios

### Interview Lighting Checklist

1. Set key light 30°–45° to one side, feathered slightly past the subject.
2. Add fill or bounce on opposite side — target 2:1 ratio for natural corporate look.
3. Place hair/rim light behind subject, opposite side from key.
4. Light background separately — 1–2 stops below key for depth.
5. Check for eyeglass reflections (raise key, angle glasses down 5°).
6. Use false color or waveform monitor to verify no clipped highlights on skin.

### Narrative Scene Approach

1. Identify motivated source in the scene (window, lamp, screen).
2. Place key to simulate motivated direction and quality.
3. Build fill naturally from ambient bounce in the environment.
4. Add practicals for visual interest and depth (set dressing lights).
5. Use negative fill to control shadows where needed.
6. Color-gel rim lights to match story tone (warm for comfort, cool for tension).

## Exposure-Lighting Coordination

- **Inverse Square Law:** Doubling distance = 1/4 intensity. Use this to control falloff.
- **Waveform target:** Skin tones at 55–70 IRE for proper exposure.
- **False color zones:** Blue = underexposed, green = midtones, red = overexposed.
- Always set camera exposure first (ISO, shutter, aperture), then adjust lighting to match — not the reverse.

## Common Lighting Problems and Fixes

| Problem | Cause | Fix |
|---------|-------|-----|
| Hot spots on forehead/nose | Key too close or too hard | Move key back, add diffusion |
| Double shadows on background | Multiple uncontrolled sources | Flag spill, use one dominant key |
| Raccoon eyes | Key too high | Lower key to 30° above eye line |
| Color mismatch (orange + blue) | Mixed sources unbalanced | Gel all sources to match, or white-balance to dominant |
| Flat / boring look | Key on axis, no contrast | Move key off-axis, add negative fill |
| Flickering on camera | LED/fluorescent PWM issue | Use flicker-free fixtures or match shutter to frequency |

## Best Practices

1. **Meter and monitor** — always use a light meter for ratio control and waveform for exposure verification.
2. **Light the scene, not the subject** — motivated lighting looks natural.
3. **Control spill** — flags, grids, and barn doors prevent light from going where it shouldn't.
4. **Shape before intensity** — get the quality and direction right, then adjust power.
5. **Safety first** — sandbag all stands, run cables with tape, check heat on fixtures.
6. **Continuity** — document lighting setups with diagrams and photos for matching across shoot days.
