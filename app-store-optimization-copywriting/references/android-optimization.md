# Google Play Store Optimization

Android-specific ASO strategies.

---

## Character Limits

| Element | Limit | Indexed |
|---------|-------|---------|
| App Title | 30 chars | Yes |
| Short Description | 80 chars | Yes |
| Full Description | 4,000 chars | Yes |
| Developer Name | - | Yes |

---

## Key Difference: Description Indexed

Unlike iOS, Google Play indexes your full description:
- Keywords in description affect ranking
- Must balance natural language with keyword inclusion
- Keyword density matters (but avoid stuffing)

---

## Short Description Strategy

### 80 Characters to Work With

Front-load most important keywords and benefits:

**Structure**:
```
[Key benefit] + [Unique differentiator] + [CTA or social proof]
```

**Examples**:
- "Track habits, achieve goals, build a better life. Start free today!"
- "Join 10M+ users meditating daily. Reduce stress in just 10 minutes."

---

## Full Description Optimization

### Keyword Integration

- Use primary keywords 3-5 times naturally
- Include variations and synonyms
- Place keywords in first 1-2 paragraphs
- Use keywords in feature bullets

### Natural vs Keyword-Heavy

**Too keyword-heavy** (avoid):
```
Our meditation app offers meditation for beginners, 
guided meditation, sleep meditation, anxiety meditation...
```

**Natural integration** (good):
```
Transform your daily routine with guided meditations 
designed for beginners and experienced practitioners alike. 
Whether you're seeking better sleep or relief from anxiety...
```

---

## Description Structure

### First 1-2 Paragraphs

- Hook with primary benefit
- Include main keywords
- Social proof if available
- Value proposition

### Feature Section

Use formatting for scannability:

```
KEY FEATURES:
✓ 500+ guided meditations
✓ Sleep stories and sounds
✓ Personalized recommendations
✓ Progress tracking
```

### Final Section

- Secondary benefits
- Social proof
- Clear CTA
- Subscription info (if applicable)

---

## User Reviews

### Reviews Are Indexed

Google indexes keywords from user reviews:
- Monitor review language
- Common phrases can boost rankings
- Encourage detailed reviews

### Response Strategy

- Respond to negative reviews constructively
- Address specific concerns
- Never be defensive
- Invite direct contact for complex issues
