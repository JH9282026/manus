# CRO Frameworks and Methodologies

Comprehensive guide to conversion rate optimization frameworks and systematic approaches.

## ResearchXL Framework

Developed by CXL, this framework provides a structured approach to CRO research through six research categories: Heuristic Analysis, Technical Analysis, Web Analytics Analysis, Mouse Tracking Analysis, Qualitative Surveys, and User Testing.

## PIE Prioritization Framework

Prioritize tests based on Potential, Importance, and Ease (1-10 scale each).

**PIE Score = (Potential + Importance + Ease) / 3**

Higher scores indicate higher priority tests.

## ICE Prioritization Framework

Alternative prioritization using Impact, Confidence, and Ease.

**ICE Score = Impact × Confidence × Ease**

## LIFT Model

Framework for analyzing conversion factors developed by WiderFunnel.

**Six Conversion Factors:**
1. Value Proposition
2. Relevance
3. Clarity
4. Anxiety (reduce)
5. Distraction (minimize)
6. Urgency

**LIFT = (Value Proposition + Relevance + Clarity) - (Anxiety + Distraction) + Urgency**

## Conversion Funnel Optimization Framework

Systematic approach to optimizing each funnel stage: Awareness, Interest, Desire, Action, Retention.
