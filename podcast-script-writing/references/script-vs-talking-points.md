### Script vs. Talking Points

Choose the appropriate scripting level for each sponsor.

**Fully Scripted (Required by Sponsor):**
```
[AD READ - SPONSOR X]
This episode is brought to you by Sponsor X. You know, I've been using Sponsor X for three months now, and honestly? It's changed how I work. [Continue with required copy...]
```

**Talking Points (Flexible):**
```
[AD READ - SPONSOR X]
Key points:
- Product name and what it does
- Personal usage story
- One key benefit
- Offer code: PODCAST20
- URL: sponsorx.com/podcast
```
