### Outline Structure

Solo episodes need more structure to maintain engagement without a conversation partner.

**Solo Episode Outline:**
```
1. Hook (30-60 seconds)
2. Introduction and episode preview (1-2 minutes)
3. Why this topic matters (2-3 minutes)
4. Main point #1 with example (5-7 minutes)
5. Main point #2 with example (5-7 minutes)
6. Main point #3 with example (5-7 minutes)
7. Practical application (3-4 minutes)
8. Conclusion and takeaways (2-3 minutes)
9. Call-to-action (30-60 seconds)
```
