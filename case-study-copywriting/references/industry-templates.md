# Industry Templates

Industry-specific case study guidance and templates.

---

## SaaS Case Studies

### Key Focus Areas

| Area | Emphasis |
|------|----------|
| Implementation | Speed, ease of adoption |
| Integration | Existing tech stack fit |
| Scalability | Growth accommodation |
| User Adoption | Engagement metrics |
| Time-to-Value | ROI timeline |

### SaaS-Specific Metrics

| Metric | What to Measure |
|--------|----------------|
| Implementation Time | Days to go-live |
| User Adoption | % of users actively using |
| Feature Utilization | Which features used most |
| Support Reduction | Decrease in support tickets |
| Cost Comparison | vs. previous solution |
| Productivity Gains | Hours saved per user |

### SaaS Template Structure

```
## [Company] Achieves [Key Result] with [Solution]

**Quick Facts**
- Industry: [Industry]
- Company Size: [Employees]
- Product: [Your product tier/version]
- Implementation Time: [X days/weeks]

**Challenge**: [2-3 sentences on the problem]

**Solution**: [2-3 sentences on implementation]

**Results**:
• [Metric 1]
• [Metric 2]
• [Metric 3]

**Quote**: "[Customer quote]"
```

## Healthcare Case Studies

### Special Considerations

| Consideration | Requirement |
|---------------|-------------|
| HIPAA Compliance | No PHI (protected health information) |
| Patient Privacy | Anonymize all patient references |
| Clinical Claims | May require IRB approval |
| Medical Accuracy | Professional review required |
| Disclaimers | Required for clinical claims |

### Healthcare Metrics

| Metric | Compliance Note |
|--------|----------------|
| Patient Satisfaction | Use aggregate scores only |
| Efficiency | Staff time, not patient-specific |
| Error Reduction | System-level, not case-specific |
| Cost per Patient | Aggregate, de-identified |
| Compliance Rates | Audit results |

### Required Disclaimers

> "Results may vary. This case study does not constitute medical advice. [Solution] is intended for [specific use case] only."

### Healthcare Template Notes

- Focus on operational improvements over clinical outcomes
- Emphasize compliance and security features
- Include administrator and staff perspectives
- Avoid any patient-identifiable information
- Get legal review before publication

## Financial Services Case Studies

### Special Considerations

| Consideration | Requirement |
|---------------|-------------|
| SEC Compliance | No misleading performance claims |
| Client Confidentiality | Often can't name clients |
| Performance Data | Past performance disclaimers |
| Regulatory Review | Compliance team sign-off |
| Forward-Looking Statements | Must be clearly labeled |

### Financial Services Metrics

| Metric | Presentation Approach |
|--------|----------------------|
| Risk Reduction | Percentage improvement |
| Processing Time | Before/after comparison |
| Compliance | Audit results, findings reduction |
| Operational Efficiency | Cost or time savings |
| Customer Satisfaction | Survey scores |

### Required Elements

- Compliance review and sign-off
- Legal disclaimer language
- Date-stamped data
- Performance disclosure statements
- Avoid forward-looking projections

### Anonymization Approach

When client can't be named:
> "A leading regional bank with $XX billion in assets..."
> "A top-10 US asset manager..."
> "A Fortune 500 insurance company..."

## Manufacturing Case Studies

### Key Focus Areas

| Area | Emphasis |
|------|----------|
| Productivity | OEE improvements |
| Quality | Defect reduction, first-pass yield |
| Safety | Incident reduction |
| Downtime | Reduction in unplanned stops |
| Cost | Per-unit cost reduction |

### Manufacturing Metrics

| Metric | Definition |
|--------|------------|
| OEE | Overall Equipment Effectiveness |
| FPY | First Pass Yield |
| MTBF | Mean Time Between Failures |
| Cycle Time | Time per unit |
| Scrap Rate | Waste percentage |
| Labor Productivity | Output per worker |

### Manufacturing Template Focus

- Emphasize quantifiable operational improvements
- Include before/after production data
- Feature maintenance and reliability gains
- Show ROI and payback period
- Include floor-level testimonials

## B2C/Consumer Case Studies

### Key Differences from B2B

| Element | B2C Approach |
|---------|-------------|
| Length | Shorter (500-1,000 words) |
| Tone | Personal, emotional |
| Metrics | Satisfaction, lifestyle impact |
| Visuals | Lifestyle photos, before/after |
| Distribution | Social media, video |

### B2C Focus Areas

- Personal transformation stories
- User-generated content integration
- Social proof and community
- Emotional journey
- Visual storytelling

### B2C Template

```
## [Customer Name]'s Story

**The Challenge**: [Personal situation in relatable terms]

**The Solution**: [How they discovered and used product]

**The Transformation**: [Personal outcome with emotional resonance]

**In Their Words**: "[Authentic, emotional quote]"

[Visual: Before/After or lifestyle image]
```
