# Meta Creative Specifications

Complete technical specifications for all Meta ad formats.

---

## Image Ads

### Feed Placements

**Facebook Feed**
- Recommended resolution: 1200 x 628 px
- Aspect ratio: 1.91:1 to 1:1
- File type: JPG or PNG
- Max file size: 30 MB

**Instagram Feed**
- Recommended resolution: 1080 x 1080 px
- Aspect ratio: 1:1 (square) or 4:5 (portrait)
- File type: JPG or PNG
- Max file size: 30 MB

**Facebook Right Column**
- Recommended resolution: 1200 x 628 px
- Aspect ratio: 1.91:1
- File type: JPG or PNG
- Max file size: 30 MB

### Stories Placements

**Facebook Stories / Instagram Stories**
- Recommended resolution: 1080 x 1920 px
- Aspect ratio: 9:16
- File type: JPG or PNG
- Max file size: 30 MB
- Safe zone: Keep important content within center 1080 x 1420 px

### Reels

**Instagram Reels / Facebook Reels**
- Recommended resolution: 1080 x 1920 px
- Aspect ratio: 9:16
- File type: JPG or PNG
- Max file size: 30 MB

### Marketplace

**Facebook Marketplace**
- Recommended resolution: 1200 x 1200 px
- Aspect ratio: 1:1
- File type: JPG or PNG
- Max file size: 30 MB

## Video Ads

### Feed Video

**Facebook Feed / Instagram Feed**
- Recommended resolution: 1080 x 1080 px minimum
- Aspect ratio: 4:5 (portrait), 1:1 (square), 16:9 (landscape)
- File type: MP4, MOV, or GIF
- Max file size: 4 GB
- Video length: 1 second to 241 minutes
- Max frame rate: 30 fps
- Audio: Stereo AAC, 128 kbps+

### Stories Video

**Facebook Stories / Instagram Stories**
- Recommended resolution: 1080 x 1920 px
- Aspect ratio: 9:16
- File type: MP4 or MOV
- Max file size: 4 GB
- Video length: 1 second to 2 minutes
- Max frame rate: 30 fps

### Reels Video

**Instagram Reels / Facebook Reels**
- Recommended resolution: 1080 x 1920 px
- Aspect ratio: 9:16
- File type: MP4 or MOV
- Max file size: 4 GB
- Video length: 3 seconds to 90 seconds
- Max frame rate: 30 fps
- Audio required

### In-Stream Video

**Facebook In-Stream**
- Recommended resolution: 1080 x 1080 px minimum
- Aspect ratio: 16:9 or 1:1
- File type: MP4 or MOV
- Max file size: 4 GB
- Video length: 5 seconds to 10 minutes
- Max frame rate: 30 fps

## Carousel Ads

### Carousel Specifications

**All Placements**
- Number of cards: 2 to 10
- Image resolution: 1080 x 1080 px
- Aspect ratio: 1:1
- File type: JPG or PNG
- Max file size per image: 30 MB
- Video aspect ratio: 1:1
- Video length: 1 second to 240 minutes

**Text Limits**
- Primary text: 125 characters
- Headline per card: 40 characters
- Description per card: 20 characters
- Link description: 30 characters

---

## Collection Ads

### Collection Specifications

**Cover Media**
- Image: 1200 x 628 px (1.91:1) or 1080 x 1080 px (1:1)
- Video: 1080 x 1080 px minimum, 1:1 or 16:9
- Video length: 1 second to 2 minutes

**Product Items**
- Minimum: 4 products from catalog
- Image: 1080 x 1080 px (1:1)

**Text Limits**
- Primary text: 125 characters
- Headline: 40 characters

---

## Instant Experience (Canvas)

### Component Specifications

**Header**
- Image: 1200 x 628 px minimum
- Video: 1080 x 1080 px minimum, max 2 minutes

**Photo**
- Resolution: 1080 x 1080 px minimum
- Aspect ratio: Any
- Max file size: 30 MB

**Video**
- Resolution: 1080 x 1080 px minimum
- Max file size: 4 GB
- Length: Up to 240 minutes

**Carousel**
- Images: 1080 x 1080 px
- 2-10 cards

**Product Set**
- Pulls from product catalog
- Displays 4-50 products

## Lead Ads

### Lead Form Specifications

**Image**
- Same as single image ad specs
- 1200 x 628 px (1.91:1) or 1080 x 1080 px (1:1)

**Video**
- Same as single video ad specs

**Form Fields**
- Standard: Full name, email, phone, company, job title
- Custom: Up to 15 custom questions
- Question types: Short answer, multiple choice, conditional

**Text Limits**
- Form title: 60 characters
- Form description: 300 characters
- Thank you screen title: 60 characters
- Thank you screen body: 300 characters

## Dynamic Ads

### Product Catalog Requirements

**Product Feed**
- Format: CSV, TSV, XML, or API
- Required fields: id, title, description, availability, condition, price, link, image_link
- Image: 1024 x 1024 px minimum, 1:1 aspect ratio

**Template Formats**
- Single image: 1:1 aspect ratio
- Carousel: 2-10 products, 1:1 aspect ratio per card
- Collection: Cover image + 4+ products

### Stories Specifications

**Image**
- Resolution: 1080 x 1920 px
- Aspect ratio: 9:16
- File type: JPG or PNG
- Max file size: 30 MB

**Video**
- Resolution: 1080 x 1920 px
- Aspect ratio: 9:16
- File type: MP4 or MOV

- Top 250 px: Avoid important content (profile icon area)

**Text**
- Primary text: 125 characters (appears above Stories)
- Headline: 40 characters (appears in CTA area)

## Text Overlay Guidelines

### Text in Image Policy

**Recommendation**: Keep text under 20% of image area

**Impact**:
- 0-10% text: No impact
- 10-20% text: Minimal impact
- 20-30% text: Reduced reach
- 30%+ text: Significantly reduced reach

**Exceptions**:
- Product images with text on packaging
- Book/album covers
- Infographics
- Legal text

## File Format Support

### Images
- JPG (recommended)
- PNG
- GIF (static only for images)

### Videos
- MP4 (recommended)
- MOV
- GIF (animated)

### Audio
- AAC audio codec
- Stereo audio
- 128 kbps or higher

---

## Design Best Practices

### Color
- Use high contrast
- Avoid pure white backgrounds (use off-white)
- Brand colors for consistency

### Composition
- Follow rule of thirds
- Clear focal point
- Minimal clutter

### Mobile Optimization
- Large, readable text
- Simple layouts
- Vertical or square formats preferred
