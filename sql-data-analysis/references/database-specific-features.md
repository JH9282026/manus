# Database-Specific SQL Features

Platform-specific functionality for PostgreSQL, MySQL, SQL Server, and BigQuery.

## PostgreSQL

Array operations, JSON/JSONB, full-text search, window functions, CTEs, and advanced indexing.

## MySQL

GROUP_CONCAT, JSON functions, full-text search, and MySQL-specific optimizations.

## SQL Server

T-SQL features, PIVOT/UNPIVOT, MERGE statements, and SQL Server-specific functions.

## BigQuery

Standard SQL, array and struct operations, wildcard tables, partitioning, and clustering.
