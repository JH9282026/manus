Detailed reference content for the Video Script Writing skill.

---

## Video Script Templates


### 60-Second Explainer Video Template

```
[0:00-0:05] HOOK
Visual: Attention-grabbing opening
Audio: "What if you could [desirable outcome] in half the time?"

[0:05-0:15] PROBLEM
Visual: Relatable struggle depicted
Audio: "Every day, [target audience] faces [specific problem]. 
It's frustrating, time-consuming, and costly."

[0:15-0:25] SOLUTION INTRO
Visual: Product/service reveal
Audio: "Introducing [Product Name]—the [category] that 
finally [solves problem]."

[0:25-0:45] HOW IT WORKS
Visual: Demo/walkthrough (3 key features)
Audio: "Simply [step 1], [step 2], and [step 3]. That's it."

[0:45-0:55] RESULTS/BENEFITS
Visual: Transformation/success
Audio: "The result? [Specific benefit]. [Statistic or proof point]."

[0:55-1:00] CTA
Visual: Logo, CTA button
Audio: "[CTA]. [Website/action]. Try it free today."
```


### YouTube Video Script Template

```
[HOOK - 0:00-0:10]
Open with: Bold statement, question, or teaser
"[Unexpected claim or question that creates curiosity]"

[INTRO - 0:10-0:30]
Establish topic and credibility
"I'm [Name], and today we're diving into [topic]. 
I've [credentials/experience], and by the end of this video, 
you'll know exactly [promise]."

[PREVIEW - 0:30-0:45]
Tell viewers what they'll learn
"We'll cover: [Point 1], [Point 2], and [Point 3]. 
Stick around for [Point 3]—it's the game-changer."

[CONTENT SECTION 1 - 0:45-3:00]
Deliver first main point with examples
Include B-roll, graphics, demonstrations

[RETENTION HOOK - 3:00]
"Now that you understand [topic 1], let's talk about 
[topic 2]—and this is where it gets interesting."

[CONTENT SECTION 2 - 3:00-6:00]
Second main point with deeper detail

[ENGAGEMENT PROMPT - 6:00]
"If you're finding this helpful, hit that like button—
it really helps the channel."

[CONTENT SECTION 3 - 6:00-9:00]
Final main point (the strongest)

[CONCLUSION - 9:00-9:45]
Summarize and deliver on promise
"So now you know [key takeaways]. Remember: [most important point]."

[CTA - 9:45-10:00]
"Subscribe for more [content type]. Check out [related video] 
next—I'll see you there."
```


### Social Media Video Ad Template (15 seconds)

```
[0:00-0:02] HOOK
Visual: Movement/face/text overlay
Audio: "[One-line hook]"

[0:02-0:08] VALUE PROPOSITION
Visual: Product in action
Audio: "[What it does] + [Key benefit]"

[0:08-0:13] PROOF/RESULTS
Visual: Results, testimonial snippet, or transformation
Audio: "[Social proof or result]"

[0:13-0:15] CTA
Visual: Logo + CTA text
Audio: "[Action]. [Now]."
```
