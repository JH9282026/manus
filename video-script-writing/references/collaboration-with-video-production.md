# Collaboration With Video Production

Detailed reference content for the Video Script Writing skill.

---

## Collaboration with Video Production


### Working with Directors
Effective script-director collaboration:

- Present scripts as starting points, not final word
- Discuss intent behind each section
- Be open to visual improvements
- Attend pre-production meetings
- Create shared vision before production


### Working with Editors
Support post-production:

- Provide clear script versions
- Note essential vs. flexible elements
- Be available for questions
- Review rough cuts against script intent
- Accept that editing may improve on script


### Revision Process
Scripts evolve through:

1. **First draft:** Complete vision on paper
2. **Internal review:** Stakeholder feedback
3. **Pre-production revisions:** Director/producer input
4. **Production adjustments:** On-set realities
5. **Post-production refinement:** Edit-driven changes
6. **Final approval:** Stakeholder sign-off


### On-Set Adjustments
Be prepared for changes:

- Have script versions ready (full, shortened, alternate takes)
- Stay available during production
- Make quick decisions when needed
- Document changes for post-production
