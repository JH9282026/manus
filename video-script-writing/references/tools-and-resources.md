# Tools And Resources

Detailed reference content for the Video Script Writing skill.

---

## Tools and Resources


### Scriptwriting Software
- **Final Draft:** Industry-standard for film/TV scripts
- **Celtx:** Free/affordable option with collaboration features
- **WriterSolo:** Web-based scriptwriting tool
- **Google Docs:** Flexible for video scripts with commenting
- **Notion:** Good for script planning and collaboration


### Teleprompter Tools
- **PromptSmart:** Smart scrolling based on voice
- **Teleprompter Premium:** iOS teleprompter app
- **BigVu:** Teleprompter with basic editing


### Timing Tools
- **Word Counter:** Most word processors include this
- **Speechify:** Tests reading pace
- **Manual timing:** Record yourself reading


### Research Resources
- **YouTube Analytics:** Study retention patterns
- **TubeBuddy/VidIQ:** YouTube research tools
- **Ad libraries:** Facebook, TikTok, YouTube ad examples
- **Competitor analysis:** Study what works in your niche


### Learning Resources
- YouTube Creator Academy
- Skillshare video production courses
- Masterclass (documentary, screenwriting)
- Industry blogs (Wistia, Vidyard)


### Template Resources
- Script templates (two-column format)
- Storyboard templates
- Shot list templates
- Brief templates for client communication


### AI and Automation Tools
- **Descript:** Transcription, editing, and script-to-video
- **Synthesia:** AI avatar video generation from scripts
- **Pictory:** Script-to-video automation
- **InVideo:** Template-based video creation
- **Runway:** AI-powered video editing
