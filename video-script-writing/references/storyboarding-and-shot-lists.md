## Storyboarding and Shot Lists


### Translating Script to Visuals
Scripts should facilitate visual planning:

- Include shot descriptions
- Note critical visual moments
- Indicate pacing through language
- Specify must-have shots
- Allow creative interpretation


### Shot Planning
Create shot list from script:

| Shot # | Description | Duration | Audio | Notes |
|--------|-------------|----------|-------|-------|
| 1 | WS office, subject enters | 3 sec | Ambient | Opening shot |
| 2 | MS subject at desk | 4 sec | VO begins | Match action from 1 |
| 3 | CU frustrated expression | 2 sec | VO continues | Emotional beat |


### Storyboard Collaboration
Work with production team:

- Provide detailed scripts but remain open to input
- Discuss key visual moments
- Review storyboards for script alignment
- Adjust dialogue based on visual possibilities
- Iterate collaboratively

**Storyboard feedback questions:**
- Does each frame advance the story?
- Is the pacing visualized correctly?
- Are key emotional moments highlighted?
- Do transitions feel smooth?
- Is the visual variety sufficient?
