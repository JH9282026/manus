# Logo Design Techniques

## Typography in Logo Design

### Font Selection Criteria
- **Serif fonts**: Authority, tradition, trust (law firms, publishing, finance)
- **Sans-serif fonts**: Modern, clean, approachable (tech, startups, healthcare)
- **Script fonts**: Elegance, creativity, personal (fashion, beauty, food)
- **Display fonts**: Bold, unique, memorable (entertainment, sports, youth)

### Custom Lettering
- Modify existing typefaces rather than using stock fonts
- Adjust kerning, weight, and proportions for uniqueness
- Consider ligatures and custom character connections
- Test readability at small sizes after modifications

## Color Theory for Logos

### Color Psychology
| Color | Associations | Industries |
|-------|-------------|-----------|
| Blue | Trust, stability, professionalism | Finance, tech, healthcare |
| Red | Energy, urgency, passion | Food, entertainment, retail |
| Green | Growth, nature, health | Environment, health, finance |
| Yellow | Optimism, warmth, attention | Food, children, energy |
| Purple | Luxury, creativity, wisdom | Beauty, education, spiritual |
| Orange | Friendly, energetic, affordable | Retail, food, sports |
| Black | Sophistication, luxury, power | Fashion, luxury, tech |

### Color System Rules
- Primary palette: 1-2 colors maximum in logo
- Ensure sufficient contrast for legibility
- Test for color blindness accessibility (8% of males affected)
- Define exact color values (Pantone, CMYK, RGB, HEX)

## Logo Variation System

### Required Variations
1. **Primary mark** — Full logo with icon and wordmark
2. **Secondary mark** — Stacked or horizontal alternative
3. **Submark** — Simplified icon or monogram
4. **Favicon** — 16x16 to 32x32 pixel optimized version
5. **Social media** — Circular crop optimized version

### Color Variations
- Full color on light background
- Full color on dark background
- Single color (black)
- Single color (white/reversed)
- Grayscale

### Minimum Size Rules
- Print: Define minimum width in inches/mm
- Digital: Define minimum width in pixels
- Clear space: Minimum padding around logo (typically = height of a character)

## Negative Space Techniques
- Use counter-forms to create hidden symbols
- Examples: FedEx arrow, Spartan Golf logo
- Balance subtlety with discoverability
- Test with unbiased viewers for recognition

## Common Logo Design Mistakes
- Too complex (fails at small sizes)
- Too trendy (dates quickly)
- Too similar to competitors
- Relies on color for meaning
- Poor font choice for industry
- Inconsistent visual weight
