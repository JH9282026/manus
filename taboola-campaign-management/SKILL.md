# Taboola Campaign Management

Manage native advertising campaigns on Taboola's content discovery platform.

## Overview

Taboola is a native advertising platform specializing in content discovery and recommendation widgets. This skill covers campaign structure, ad creation (Items and Motion Ads), targeting, bidding strategies, and performance optimization.

## Campaign Structure

Taboola uses a three-level hierarchy:
- **Campaign**: Budget, dates, bidding strategy
- **Creative**: Items (static) or Motion Ads (video)
- **Targeting**: Geo, platform, audience, publisher

## Ad Types

### Items (Static Ads)
- Image + headline + description
- Recommended size: 1200x800px
- Max 100 character headlines

### Motion Ads (Video)
- 6-60 seconds (15-30 optimal)
- MP4, MOV, AVI formats
- Requires fallback image

## Targeting Options

- **Geographic**: Country, state, city, postal code
- **Platform**: Desktop, mobile, tablet
- **Audience**: Behavioral segments, custom audiences, lookalikes
- **Publisher**: Auto-targeting or manual lists

## Bidding Strategies

| Strategy | Best For | Control |
|----------|----------|---------|
| Fixed CPC | Testing, new campaigns | Full manual |
| Enhanced CPC | Balanced performance | Semi-automated |
| Target CPA | Lead generation | Automated |
| Maximize Conversions | Volume focus | Fully automated |
