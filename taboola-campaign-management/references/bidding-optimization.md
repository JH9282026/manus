# Taboola Bidding Optimization

Bidding strategies and optimization techniques.

## Bidding Strategies

### Fixed CPC
- Full manual control
- Best for: New campaigns, testing
- Set specific CPC bid

### Enhanced CPC (SmartBid)
- Semi-automated
- Algorithm adjusts bids ±30%
- Best for: 10-30 conversions/month

### Target CPA
- Automated bidding
- Set target cost per acquisition
- Best for: 30+ conversions/month
- Requires 2-3 week learning phase

### Maximize Conversions
- Fully automated
- Maximizes conversion volume
- Best for: 100+ conversions/month
- Uses full daily budget

## Optimization Process

**Week 1-2: Launch Phase**
- Monitor approval status
- Check initial CTR
- Identify top performers

**Week 3-4: Optimization Phase**
- Analyze conversion data
- Adjust bids based on CPA
- Refine targeting

**Week 5+: Scaling Phase**
- Increase budgets on winners
- Expand to similar audiences
- Implement automated bidding

## Bid Adjustment Guidelines

- Increase bid: CPA <target, low impression share
- Decrease bid: CPA >target, budget depleting fast
- Make incremental changes (10-30%)
- Wait 3-5 days between adjustments
