# Taboola Creative Best Practices

Guidelines for creating high-performing ads.

## Items (Static Ads)

**Image Specs:**
- Size: 1200x800px
- Format: JPG, PNG
- File size: Under 5MB

**Headline Best Practices:**
- Length: 60-80 characters
- Use curiosity gap
- Include numbers
- Ask questions

## Motion Ads (Video)

**Video Specs:**
- Duration: 15-30 seconds optimal
- Format: MP4, MOV
- Resolution: 1080p
- Include fallback image

**Content Tips:**
- Hook in first 3 seconds
- Use captions (assume muted)
- Clear call-to-action
- Show product/benefit quickly

## Performance Benchmarks

- Average CTR: 0.3-0.5%
- Good CTR: 0.5-1.0%
- Excellent CTR: >1.0%
