# Taboola Campaign Setup Guide

Comprehensive guide for creating Taboola campaigns.

## Campaign Creation Steps

1. Select advertiser account
2. Set campaign name and branding text
3. Configure budget and bidding
4. Set campaign dates
5. Configure targeting
6. Add creatives (Items or Motion Ads)

## Budget Configuration

- Daily budget: Minimum $10
- Total budget: Optional lifetime limit
- Pacing: Balanced or Accelerated

## Targeting Setup

- Geographic: Country, state, city
- Platform: Desktop, mobile, tablet
- Audience: Behavioral segments, custom audiences
- Publisher: Auto-targeting or manual lists

## Best Practices

- Start with broad targeting
- Test multiple creatives
- Monitor performance daily
- Optimize based on data
