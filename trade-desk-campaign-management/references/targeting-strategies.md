# The Trade Desk Targeting Strategies

Advanced targeting options for programmatic campaigns.

## Audience Targeting

### Demographic Targeting
- Age ranges
- Gender
- Income levels
- Education
- Household composition

### Interest & Behavior Targeting
- Interest categories
- Online behaviors
- Purchase intent signals
- Content consumption patterns

### First-Party Data
- CRM data upload
- Website visitor data
- Customer lists
- Transaction history

### Third-Party Data
- Data marketplace segments
- Purchase behavior data
- Intent data providers

### Lookalike Audiences
- AI-driven modeling
- Based on first-party data
- Adjustable similarity levels
- Automatic expansion

## Contextual Targeting

### Content Relevance
- Keyword targeting
- Category targeting
- Page-level context
- Semantic analysis

### Environmental Factors
- Device type
- Connection speed
- Weather conditions
- Time of day

## Geotargeting

### Geographic Levels
- Country
- State/Province
- DMA (Designated Market Area)
- City
- Postal code

### Radius Targeting
- Target around specific coordinates
- Adjustable radius
- Multiple location targeting

### Geo + Demo Combination
- Combine location with demographics
- Example: High-income households in specific cities

## Cross-Device Targeting

- Track users across devices
- Consistent messaging
- Frequency capping across devices
- Cross-device attribution

## Retail & Commerce Data

### Retail Media Networks
- Access to transaction data
- Purchase-based targeting
- In-store and online behavior

### Commerce Audiences
- Recent purchasers
- Category shoppers
- Brand loyalists

## Private Marketplace (PMP)

### PMP Deals
- Exclusive inventory access
- Pre-negotiated rates
- Premium publishers
- Brand-safe environments

### Programmatic Guaranteed (PG)
- Direct media buying
- Fixed CPM rates
- Guaranteed impressions
- Premium placements

## Targeting Best Practices

- Start broad, refine based on performance
- Layer targeting for precision
- Use exclusions to improve efficiency
- Test different audience combinations
- Monitor frequency across targeting layers
- Leverage first-party data when available
- Combine contextual and audience targeting
