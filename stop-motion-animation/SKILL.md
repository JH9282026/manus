# Stop-Motion Animation

Create frame-by-frame animation by physically manipulating objects and capturing individual photographs to produce the illusion of movement.

## Overview

This skill covers the complete stop-motion animation pipeline from concept to final export: pre-production planning, character and set fabrication, equipment setup, shooting techniques, frame-by-frame capture workflows, and post-production editing. Use it when producing claymation, puppet animation, object motion, cutout animation, or any project requiring physical manipulation of objects captured frame-by-frame.

## Equipment Setup by Budget Level

### Beginner Setup ($100-$500)
- **Camera**: Smartphone with Stop Motion Studio app
- **Stabilization**: Tabletop tripod or DIY solution (stacked books)
- **Lighting**: Desk lamps diffused with white paper
- **Software**: Stop Motion Studio (mobile/desktop)
- **Posing Tools**: Blu-Tack, sticky tack, small magnets
- **Stage**: Foam board or cardboard backdrop

### Intermediate Setup ($500-$3,000)
- **Camera**: Entry-level DSLR (Canon EOS Rebel, Sony Alpha series)
- **Stabilization**: Flexible tripod (Joby GorillaPod) or standard tripod
- **Lighting**: Softbox lights or LED panels (Neewer kits)
- **Software**: DaVinci Resolve (free) or Stop Motion Studio Pro
- **Posing Tools**: Wire armatures (aluminum wire), weights
- **Stage**: Portable stage with built-in frames
- **Monitor**: Laptop or tablet for tethered viewing

### Professional Setup ($3,000+)
- **Camera**: High-end DSLR/mirrorless (Canon R5, Canon R6) with AC adapter
- **Stabilization**: Manfrotto tripod, C-stands for overhead shots
- **Lighting**: Flat panel LED lights (Godox, Aputure, Flashpoint FV150)
- **Software**: Dragonframe with wireless controller
- **Posing Tools**: Ball-and-socket armatures, rigging systems, epoxy putty
- **Stage**: Custom-built stages, green screens
- **Monitor**: 24" external monitor with tethering setup
- **Computer**: Mac or PC with sufficient processing power

See `/references/equipment-setup.md` for detailed equipment specifications, purchasing recommendations, and setup configurations.

## Production Workflow

### Pre-Production
1. **Story Development**: Create narrative, write script
2. **Storyboarding**: Visualize shots frame-by-frame, plan sequences
3. **Character Design**: Sketch characters, plan proportions and movement capabilities
4. **Fabrication**: Build characters, props, and sets (see `/references/character-design.md`)
5. **Production Design**: Establish visual style, color palette, aesthetic
6. **Shot Planning**: Determine camera angles, shot sizes, movement

### Production
1. **Scene Setup**: Position camera on stable tripod, connect to computer
2. **Lighting Setup**: Establish consistent lighting (avoid natural light)
3. **Camera Configuration**: Set manual focus, exposure, white balance
4. **Software Setup**: Configure frame rate (typically 12-24 fps), enable onion skinning
5. **Test Shots**: Capture test frames to verify lighting, focus, exposure
6. **Frame-by-Frame Capture**: Make minute adjustments, capture frame, repeat
7. **Remote Triggering**: Use software or remote to prevent camera shake

### Post-Production
1. **Editing**: Arrange frames, adjust timing, trim sequences
2. **Sound Design**: Record voice-overs, add sound effects
3. **Music**: Choose or create background music
4. **Color Correction**: Adjust colors for consistency
5. **Effects**: Add titles, transitions, visual effects if needed
6. **Export**: Render final video in desired format

See `/references/post-production.md` for detailed editing workflows, sound design techniques, and export settings.

## Frame Rate and Timing

| Frame Rate | Frames per Second | Use Case | Shooting Method |
|------------|-------------------|----------|------------------|
| Standard Film | 24 fps | Professional, cinematic look | Shoot on "twos" (12 unique frames/sec) |
| Smooth Motion | 24 fps | Fast action, smooth movement | Shoot on "ones" (24 unique frames/sec) |
| Web/Social | 12-15 fps | Quick projects, stylized look | Shoot on "twos" or "threes" |
| Experimental | Variable | Artistic effects, time manipulation | Mix of ones, twos, threes |

**Calculation**: 1 minute at 24 fps = 1,440 frames. Shooting on "twos" = 720 unique frames needed.

## Shot Composition

### Shot Sizes
- **Wide Shot**: Establishes environment, shows full character and setting
- **Medium Shot**: Character from waist up, balances character and environment
- **Close-Up**: Face or specific object detail, conveys emotion
- **Extreme Close-Up**: Eyes, hands, tiny details for dramatic emphasis

### Camera Angles
- **Straight-On**: Neutral perspective, eye-level with subject
- **Profile**: Side view, shows character relationships
- **Low-Angle**: Camera below subject, makes subject appear powerful
- **High-Angle**: Camera above subject, makes subject appear vulnerable
- **Overhead**: Directly above, useful for cutout animation

See `/references/animation-techniques.md` for advanced cinematography techniques and movement strategies.

## Key Principles for Success

### Consistency is Critical
- **Lighting**: Lock exposure, use artificial light only, prevent flickering
- **Focus**: Set manual focus, never use autofocus
- **Camera Position**: Secure tripod, avoid any movement between frames
- **White Balance**: Set manually to prevent color shifts

### Patience and Planning
- Stop-motion is time-intensive: expect 4-8 hours per finished minute
- Plan every shot before starting capture
- Take breaks to maintain focus and prevent errors
- Review footage regularly using playback feature

### Movement Quality
- **Spacing**: Vary distance between positions for acceleration/deceleration
- **Arcs**: Natural movement follows curved paths, not straight lines
- **Anticipation**: Wind up before major actions
- **Follow-Through**: Parts continue moving after main action stops
- **Squash and Stretch**: Exaggerate for impact and life

## Common Challenges and Solutions

| Challenge | Solution |
|-----------|----------|
| Flickering light | Use only artificial light, lock camera settings, avoid windows |
| Camera shake | Use remote trigger or software control, never touch camera |
| Character falling | Use Blu-Tack, wire armatures, rigging, or tie-downs |
| Inconsistent movement | Enable onion skinning, review previous frames |
| Lost focus | Set manual focus, mark focus point, check regularly |
| Shadows moving | Lock lighting position, use diffused light sources |
| Set disturbance | Work in isolated area, mark positions with tape |
