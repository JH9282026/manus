### Four-Point and Multi-Light Setups

**Four-Point Lighting:**
Adding a background light to three-point creates depth:

- **Key Light:** Primary illumination on subject
- **Fill Light:** Reduces shadow darkness
- **Back Light:** Separates subject from background
- **Background Light:** Illuminates backdrop independently

**Multi-Light Complexity:**
Professional productions often use many lights:

- Each light serves specific purpose
- Layers build complexity
- More control but more variables
- Requires systematic approach

**Building Complex Setups:**
1. Start with key light only
2. Evaluate before adding fill
3. Add back light for separation
4. Light background separately
5. Add accent lights as needed
6. Fine-tune balance
