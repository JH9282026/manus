# Financial Modeling & Valuation

Build sophisticated financial models including DCF, trading comparables, precedent transactions, LBO, and merger models to support M&A, investment, and corporate finance decisions.

## Overview

Provide frameworks for investment banking-grade financial modeling covering three-statement models, valuation methodologies, transaction analysis, and model presentation. Follow Wall Street best practices for model structure, formatting, and auditability.

## Valuation Methodology Overview

| Method | Type | Best For | Output |
|---|---|---|---|
| DCF | Intrinsic | Stable cash flows, going concern | Enterprise value |
| Trading Comps | Relative | Public company benchmarking | Valuation range |
| Precedent Transactions | Relative | M&A pricing reference | Transaction value |
| LBO | Returns-based | PE acquisition, leveraged deals | IRR, MOIC |
| Merger Model | Accretion/dilution | Strategic M&A assessment | EPS impact |
| Sum-of-Parts | Intrinsic/Relative | Diversified companies | Segment values |
| Asset-based | Intrinsic | Liquidation, asset-heavy businesses | NAV |

## Model Formatting Standards

| Element | Convention |
|---|---|
| Hard-coded inputs | Blue font |
| Formulas | Black font |
| Links to other sheets | Green font |
| Assumptions section | Yellow/light background |
| Output/summary | Gray background |
| Units | Clearly labeled ($M, $K, %) |
| Dates | Column headers |
| Signage | Positive = inflow, Negative = outflow |
