## SMS Marketing Checklist


### Campaign Checklist

Before sending any campaign, verify:

**Content:**
- [ ] Message is under 160 characters (or intentionally longer)
- [ ] Brand name is clearly identified
- [ ] Offer is clear and compelling
- [ ] CTA is specific and actionable
- [ ] Link works and loads on mobile
- [ ] Opt-out instructions included

**Compliance:**
- [ ] Recipients have opted in
- [ ] Message complies with TCPA/GDPR
- [ ] Sending during appropriate hours
- [ ] Required disclosures included

**Technical:**
- [ ] Test message sent and reviewed
- [ ] Personalization fields working
- [ ] Segmentation correct
- [ ] Tracking codes in place
- [ ] Fallback text for missing data

**Timing:**
- [ ] Scheduled for optimal time
- [ ] Avoiding conflicts with other sends
- [ ] Timezone considerations addressed
