# Lean Six Sigma Integration

Combining Lean speed with Six Sigma statistical rigor.

---

## When to Use Each Approach

| Aspect | Lean | Six Sigma | Lean Six Sigma |
|--------|------|-----------|----------------|
| **Focus** | Speed, waste elimination | Quality, variation reduction | Both |
| **Approach** | Visual, pragmatic | Statistical, analytical | Integrated |
| **Timeframe** | Rapid improvement | Structured projects | Flexible |
| **Tools** | VSM, 5S, Kanban | DMAIC, SPC, DOE | All tools |

### Use Lean When

- Process has obvious waste
- Speed and flow are primary concerns
- Solutions are relatively straightforward
- Quick wins needed for momentum

### Use Six Sigma When

- Root causes unclear, require data analysis
- Variation and defects are primary concerns
- Process is complex with multiple variables
- Statistical validation required

### Use Lean Six Sigma When

- Problems involve both waste and variation
- Comprehensive transformation needed
- Organization wants unified approach
- Complex processes require multiple toolsets

## Integration Framework

### Phase-Based Integration

| DMAIC Phase | Lean Tools | Six Sigma Tools |
|-------------|------------|-----------------|
| **Define** | SIPOC, VOC | Project Charter, CTQ Tree |
| **Measure** | Process Mapping, Time Studies | Data Collection, MSA |
| **Analyze** | Value Stream Mapping, 5 Whys | Statistical Analysis, DOE |
| **Improve** | 5S, Standard Work, Poka-Yoke | Hypothesis Testing, Piloting |
| **Control** | Visual Management, Kanban | SPC, Control Plans |

## Project Selection

### Selection Criteria

| Criterion | Weight |
|-----------|--------|
| Strategic alignment | 25% |
| Financial impact | 25% |
| Customer impact | 20% |
| Feasibility | 15% |
| Resource availability | 15% |

### Qualification Questions

1. Is the problem measurable with available data?
2. Is root cause unknown (requiring analysis)?
3. Is solution not already known?
4. Is scope manageable?
5. Is there executive sponsorship?
6. Are resources available?

## Integration Benefits

- Speed from Lean combined with rigor from Six Sigma
- Broader toolkit for diverse problems
- Flexibility to apply appropriate tools
- Cultural alignment around quality and speed

---

## Tool Selection Guide

### For Waste Elimination

- Value Stream Mapping
- 5S
- Kanban
- Quick Changeover (SMED)

### For Variation Reduction

- Statistical Process Control
- Design of Experiments
- Measurement System Analysis
- Hypothesis Testing

### For Root Cause Analysis

- 5 Whys (quick)
- Fishbone Diagram (structured)
- Regression Analysis (data-driven)

### For Sustainability

- Control Plans
- Visual Management
- Standard Work
- Audit Programs

---

## Common Integration Mistakes

1. **Tool obsession**: Using tools without purpose
2. **Analysis paralysis**: Over-analyzing simple problems
3. **Skipping Define**: Not properly scoping project
4. **Ignoring culture**: Not addressing people side
5. **No control**: Failing to sustain gains
