# Lean Tools and Techniques

Detailed guidance for implementing Lean tools.

---

## Value Stream Mapping (VSM)

### Process
1. Select product family with similar processing steps
2. Create current state map (all steps, info flows, metrics)
3. Analyze current state (lead time, process time, waste)
4. Design future state map (improved flow)
5. Develop implementation plan

### Key Metrics
- **Process Time (PT)**: Time actually working
- **Lead Time (LT)**: Total time including waiting
- **Value-Added Ratio**: PT ÷ LT × 100%
- **First Pass Yield**: Percentage passing without rework

## 5S Implementation

### Phase 1: Sort
- Red tag items of questionable necessity
- Disposition: keep, relocate, or discard
- Document before state

### Phase 2: Set in Order
- Create designated locations
- Visual controls (labels, floor markings)
- Shadow boards for tools

### Phase 3: Shine
- Establish cleaning schedules
- Identify root causes of dirt/mess
- Make problems visible

### Phase 4: Standardize
- Create 5S audit checklists
- Define responsibilities
- Document standards

### Phase 5: Sustain
- Regular audits
- Recognition programs
- Leadership gemba walks

## Kaizen Events

### Typical Structure
- Duration: 3-5 days
- Team: 6-12 people
- Scope: Specific, bounded problem
- Authority: Team empowered to implement

### Daily Flow
- Day 1: Training, observation, waste identification
- Days 2-3: Solution development and testing
- Day 4: Finalize changes, document
- Day 5: Present results, plan follow-up

## Kanban Systems

### Principles
1. Visualize work
2. Limit work in progress (WIP)
3. Manage flow
4. Make policies explicit
5. Implement feedback loops
6. Improve collaboratively

### Types
- Production Kanban: Authorize production
- Withdrawal Kanban: Authorize movement
- Supplier Kanban: Signal replenishment

---

## Poka-Yoke (Error-Proofing)

### Categories
- **Prevention**: Makes errors impossible
- **Detection**: Identifies errors immediately
- **Warning**: Alerts before error occurs

### Examples
- Form validation
- Physical fixtures preventing wrong orientation
- Sensors detecting missing components
- Checklists preventing skipped steps
