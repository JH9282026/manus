# Multi-Document Synthesis

Combining information from multiple sources.

---

## Synthesis vs. Comparison

### Synthesis
- Combine sources into unified narrative
- Identify common themes
- Note areas of agreement
- Highlight collective insights

### Comparison
- Side-by-side analysis
- Highlight differences
- Identify contradictions
- Present multiple perspectives

---

## Multi-Source Workflow

### Step 1: Individual Analysis
- Summarize each source separately
- Extract key themes
- Note unique claims
- Identify data points

### Step 2: Cross-Reference
- Map overlapping topics
- Identify agreements
- Note contradictions
- Find complementary info

### Step 3: Synthesize
- Create unified structure
- Integrate insights
- Address conflicts
- Build comprehensive view

### Step 4: Attribute
- Note source of each claim
- Indicate consensus vs. single-source
- Preserve important citations

---

## Common Synthesis Patterns

### Thematic Synthesis
Organize by themes across sources.

### Chronological Synthesis
Organize by timeline across sources.

### Comparative Matrix
Table format comparing sources on key dimensions.

---

## Handling Contradictions

### Approach
1. Acknowledge the disagreement
2. Present both perspectives
3. Note methodology differences
4. Indicate which has more support

### Example
"Source A found X, while Source B reported Y. The difference may be due to [methodology/sample/timeframe]. More recent research supports..."

---

## Output Formats

### Unified Summary
Single narrative combining all sources.

### Comparative Analysis
Structured comparison highlighting differences.

### Annotated Bibliography
Separate summaries with synthesis notes.

### Theme Matrix
Table organizing findings by theme and source.
