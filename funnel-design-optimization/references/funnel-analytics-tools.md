# Funnel Analytics & Tools

## Key Funnel Metrics

| Metric | Description | How to Improve |
|--------|-------------|----------------|
| Conversion Rate | % completing desired action per stage | Test copy, CTAs, design, offers |
| Drop-off Rate | Where and why people leave | Fix leaks, reduce friction |
| Average Order Value | Revenue per transaction | Add upsells, bumps, bundles |
| Customer Lifetime Value | Total revenue per customer | Retention, upsells, loyalty |
| Cost Per Acquisition | Cost to acquire one customer | Optimize ads, improve conversion |
| ROAS | Revenue per ad dollar | Better targeting, higher AOV |
| Funnel Velocity | Time from first touch to purchase | Streamline steps, add urgency |

## Funnel Visualization and Reporting
- Build real-time conversion dashboards tracking stage-by-stage performance
- Compare traffic sources, test variants, and revenue/profitability
- Run cohort analysis to track groups of users over time
- Use multi-touch attribution to credit conversions across touchpoints

## Retargeting and Remarketing
- Install Facebook Pixel, Google Tag, and custom pixels
- Show specific ads based on funnel stage:
  - Visited but didn't opt in → Lead magnet ads
  - Opted in but didn't buy → Testimonial/benefit ads
  - Abandoned cart → Discount/urgency ads
  - Past customers → Upsell ads
- Rotate creative to prevent ad fatigue; use frequency caps

## Funnel Automation
- Trigger-based email sending on page visits, link clicks, form submissions, purchases
- If-this-then-that workflows: lead scoring, tag applications, sequence enrollment
- Connect CRM for lead capture, purchase history sync, sales team alerts
- Use Zapier for cross-platform integrations

- **Drip**: E-commerce focused automation
- **Mailchimp**: Email marketing with automation

## Industry-Specific Funnels
- **SaaS**: Free trial → Onboarding → Activation → Conversion → Expansion
- **E-commerce**: Product → Cart → Checkout → Upsell → Thank You
- **Coaching**: Content → Application → Discovery Call → Proposal → Onboarding
- **Local Business**: Lead Capture → Appointment → Service → Follow-up → Referral
- **Real Estate**: Property Search → Inquiry → Showing → Offer → Closing
