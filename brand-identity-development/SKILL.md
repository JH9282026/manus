# Brand Identity Development

Create complete, cohesive brand identity systems from strategy through final deliverables.

## Overview

This skill synthesizes brand strategy, visual design, messaging architecture, and application design into unified brand ecosystems. It covers strategic foundation, logo systems, color architecture, typography, messaging, visual elements, guidelines, and application templates.

## Brand Development Domains

| Domain | Deliverables |
|--------|-------------|
| Strategic Foundation | Purpose, values, positioning, personas |
| Logo System | Primary, secondary, icon, mono versions |
| Color Architecture | Primary, secondary, neutral palettes |
| Typography | Typeface selection, hierarchy, pairing |
| Messaging | Voice, positioning, taglines, narratives |
| Visual Elements | Patterns, icons, photography style |
| Guidelines | Complete style guide (40-80 pages) |
| Applications | Stationery, digital, social templates |

## Logo System Requirements

### Required Versions
- Primary logo (full lockup)
- Secondary variation
- Icon/symbol standalone
- Wordmark only
- Monochrome (black, white)
- Reversed for dark backgrounds

### File Formats
- SVG (scalable, web)
- AI/EPS (print production)
- PNG (transparent digital)
- JPG (general use)
- Favicon set (16px-512px)

## Color Architecture

### Palette Structure

| Tier | Purpose | Quantity |
|------|---------|----------|
| Primary | Core brand colors | 1-3 |
| Secondary | Supporting/accent | 2-4 |
| Neutral | Backgrounds, text | 3-5 |

### Specifications Required

For each color: HEX, RGB, CMYK, Pantone/PMS, accessibility contrast ratios

## Typography System

### Selection Criteria
- Brand personality alignment
- Legibility across sizes
- Web availability/licensing
- Language support
- Weight range available

### Hierarchy Definition
- H1-H6 specifications
- Body copy (long and short form)
- Captions and small text
- Pull quotes/callouts

## Brand Type Adaptations

| Type | Focus |
|------|-------|
| B2B | Credibility, expertise, trust |
| B2C | Emotional connection, memorability |
| Product | Shelf presence, differentiation |
| Service | Experience, relationship |
| Personal | Authenticity, personality |
