### Technical Requirements



#### Presenter Setup
- **Internet**: Minimum 10 Mbps upload, wired connection preferred
- **Computer**: Modern processor, 8GB+ RAM
- **Camera**: 1080p webcam minimum
- **Microphone**: USB or XLR microphone, not laptop mic
- **Lighting**: Ring light or key light minimum
- **Backup**: Phone hotspot, backup presentation access



#### Platform Configuration
- Registration page setup
- Email automation connected
- Branding elements configured
- Polls and engagement features prepared
- Recording settings enabled
- Moderator permissions assigned
