# Email Sequences For Webinars

Detailed reference content for the Webinar Production Strategy skill.

---

## Email Sequences for Webinars


### Pre-Webinar Sequence


#### Registration Confirmation
- Thank them for registering
- Confirm date/time (with timezone)
- Add to calendar link
- Set expectations


#### Value-Building Email
- Preview key topics
- Build anticipation
- Share speaker background
- Encourage sharing


#### Reminder Email (1 day before)
- Time reminder
- Technical requirements
- Preparation suggestions
- One key benefit


#### Final Reminder (1 hour before)
- Join link
- Last chance energy
- Quick benefit reminder


### Post-Webinar Sequence


#### For Attendees
- Thank you message
- Recording link
- Resources mentioned
- Call-to-action


#### For No-Shows
- "Sorry we missed you"
- Recording availability
- Key highlights
- Alternative next steps


#### For All Registrants
- Follow-up value content
- Feedback request
- Next webinar invitation
- Ongoing nurture sequence
