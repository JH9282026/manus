# Repurposing Content Across Platforms

Detailed reference content for the Short Form Video Content skill.

---

## Repurposing Content Across Platforms


### Cross-Platform Strategy

**Simultaneous Posting**:
- Post same content across platforms with minor adjustments
- Efficient use of content
- Maximizes reach
- Tests what works where

**Staggered Posting**:
- Post to primary platform first
- Wait 24-48 hours before posting elsewhere
- Allows platform-specific optimization
- Reduces duplicate content concerns

**Platform-Native Creation**:
- Create unique content for each platform
- Maximum algorithm favor
- Most resource-intensive
- Ideal for major focus platforms


### Adaptation Guidelines

**TikTok → Instagram Reels**:
- Remove TikTok watermark
- Adjust caption for Instagram audience
- Consider more polished aesthetic
- Change hashtag strategy

**TikTok → YouTube Shorts**:
- Remove watermarks
- Add #Shorts hashtag
- Adjust for potentially different audience
- Consider longer captions for SEO

**Instagram Reels → Other Platforms**:
- Remove Instagram-specific branding
- Adjust aspect ratio if needed
- Modify caption for platform norms
- Update hashtags


### Repurposing Long-Form Content

**Extracting Shorts from Long Videos**:
- Identify compelling 15-60 second segments
- Re-edit for vertical format
- Add platform-specific hooks and endings
- Create multiple pieces from single long video

**Turning Podcasts into Short-Form**:
- Pull quotable moments
- Add visual elements
- Create audiogram-style content
- Transcribe and add captions
