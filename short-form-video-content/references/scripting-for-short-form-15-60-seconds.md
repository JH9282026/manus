## Scripting for Short-Form (15-60 Seconds)



### Script Structure

**The 15-Second Script**
```
Hook (2-3 seconds): [Attention grabber]
Setup (3-4 seconds): [Context/problem]
Payoff (5-6 seconds): [Solution/punchline]
CTA (2-3 seconds): [What to do next]
```

**The 30-Second Script**
```
Hook (2-3 seconds): [Strong opening]
Context (5-7 seconds): [Background information]
Main Content (12-15 seconds): [Core value]
Resolution (5-7 seconds): [Conclusion/outcome]
CTA (3-5 seconds): [Clear next step]
```

**The 60-Second Script**
```
Hook (3-5 seconds): [Compelling opening]
Premise (8-10 seconds): [Set the stage]
Build (20-25 seconds): [Develop the content]
Climax (10-15 seconds): [Peak moment]
Resolution (8-10 seconds): [Wrap up]
CTA (5-7 seconds): [Action prompt]
```



### Writing for Brevity

**Eliminate Filler**: Cut every unnecessary word
**One Point Per Video**: Focus on single clear message
**Active Voice**: "Do this" not "This should be done"
**Conversational Tone**: Write how people talk
**Read Aloud**: Time your script by reading at natural pace



### Visual Scripting

Plan the visual alongside the verbal:

```
| Timing | Audio | Visual | Text Overlay |
|--------|-------|--------|--------------|
| 0-3s | "Stop scrolling if..." | Close-up face | "WAIT" |
| 3-8s | "...you want to [benefit]" | Pull back, gesture | "[Benefit]" |
| 8-15s | [Main content] | Demonstration | Key points |
```
