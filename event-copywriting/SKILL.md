# Event Copywriting

Craft persuasive event copy that attracts attendees, builds excitement, and drives registrations across conferences, webinars, workshops, galas, trade shows, and corporate events.

## Overview

Provide frameworks for every stage of event communication — from naming and branding through registration, pre-event engagement, on-site experience, and post-event follow-up. Cover format-specific techniques for virtual, hybrid, and in-person events.

## Event Copy Ecosystem

| Touchpoint | Purpose | Timing |
|---|---|---|
| Event name & tagline | Brand identity | 6-12 months before |
| Save-the-date | Awareness, calendar hold | 3-6 months before |
| Event description | Interest, value proposition | 3-6 months before |
| Speaker bios | Credibility, interest | 2-4 months before |
| Registration page | Conversion | Ongoing |
| Email sequences | Nurture, drive registration | 3-6 months before |
| Social media posts | Awareness, engagement | Ongoing |
| Attendee communications | Logistics, excitement | 1-4 weeks before |
| On-site materials | Guidance, engagement | Day of event |
| Post-event follow-up | Retention, feedback | 1-7 days after |

## Event Description Framework

1. **Hook** — Open with the transformation or outcome attendees will experience
2. **Problem/opportunity** — Why this event matters right now
3. **Value proposition** — What attendees will learn, gain, or experience
4. **Agenda highlights** — Key sessions, speakers, or activities
5. **Social proof** — Past attendee testimonials, speaker credentials
6. **Logistics** — Date, time, location, format (virtual/hybrid/in-person)
7. **CTA** — Register now with urgency element

## Registration Page Conversion Tactics

- Lead with the outcome, not the agenda
- Use social proof prominently (past attendee quotes, company logos)
- Show pricing clearly with early-bird savings
- Minimize form fields (collect essentials only)
- Add countdown timer for urgency
- Include FAQ section to overcome objections
- Display trust indicators (money-back guarantee, privacy policy)
