# Live Stream Production Workflow

## Pre-Production Planning

### 2 Weeks Before
- Define stream goals and success metrics
- Select platform and configure channel
- Create run sheet with timestamps
- Design graphics (overlays, lower thirds, transitions)
- Book guests and send technical requirements
- Promote stream across channels

### 1 Week Before
- Technical rehearsal with all participants
- Test all equipment and connections
- Prepare backup plans for technical failures
- Create chat moderation guidelines
- Set up analytics tracking
- Schedule social media promotion posts

### Day of Stream
- Set up equipment 2+ hours early
- Sound check all audio sources
- Verify internet speed and stability
- Load all scenes and graphics
- Brief all participants on cues and flow
- Start recording backup before going live

## Multi-Camera Production

### Camera Assignments
| Camera | Shot | Purpose |
|--------|------|---------|
| Camera 1 | Wide shot | Establishing, full group |
| Camera 2 | Medium shot | Active speaker |
| Camera 3 | Close-up | Reactions, detail |
| Camera 4 | Screen/slides | Presentation content |

### Switching Best Practices
- Cut on action or speech transitions
- Avoid rapid switching (minimum 3 seconds per shot)
- Use wide shot as safe/default camera
- Cut to speaker before they begin talking
- Use transitions sparingly (dissolve for mood, cut for energy)

## Live Audience Engagement

### Chat Management
- Assign dedicated chat moderator
- Set up chat commands for common questions
- Pin important messages (links, schedule)
- Acknowledge viewers by name periodically
- Run polls and Q&A segments at planned intervals

### Interactive Elements
- Live polls (Slido, StreamElements)
- Q&A queuing systems
- Chat-triggered graphics
- Donation/super chat acknowledgments
- Social media integration (Twitter feed)

---

## Troubleshooting During Stream

| Problem | Quick Fix |
|---------|-----------|
| Frame drops | Lower bitrate by 1000 kbps, close apps |
| Audio echo | Mute unused mics, check monitor routing |
| Camera freeze | Switch to backup scene, restart source |
| Internet drop | Auto-reconnect (OBS setting), switch to backup |
| Guest can't connect | Phone dial-in backup, pre-recorded segment |

---

## Post-Stream Workflow
1. Save recording immediately
2. Export chat log and highlights
3. Review analytics (peak viewers, average watch time, engagement)
4. Edit highlights for VOD and clips
5. Upload to additional platforms
6. Send thank-you to guests and team
7. Document lessons learned
