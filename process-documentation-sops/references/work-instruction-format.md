### Work Instruction Format

**Header:**
- Title
- Reference SOP (if applicable)
- Version and date
- Author

**Body:**
- Prerequisites (what's needed before starting)
- Step-by-step instructions
- Expected outcomes
- Troubleshooting tips

**Example Work Instruction:**

```
WORK INSTRUCTION: Scanning Vendor Invoices
Reference: SOP-FIN-001, Step 3
Version: 1.2 | Date: March 1, 2026

PREREQUISITES:
- Scanner powered on and connected
- Document scanning software open
- Invoice batch ready for processing

PROCEDURE:
1. Place invoice face-down on scanner glass, aligned to corner guides
2. In ScanPro software, select "Invoice" document type
3. Click "Scan" button (or press F5)
4. Verify scan quality in preview window:
   - All text legible
   - No cut-off edges
   - No shadow or skew
5. If quality acceptable, click "Save"
6. If quality poor, click "Rescan" and adjust invoice position
7. Name file using format: INV_[VendorName]_[Date]_[Amount]
   Example: INV_OfficeSupplyCo_20260301_450.00
8. Save to network folder: \\server\AP\Incoming\[Current Month]

TROUBLESHOOTING:
- Scanner not responding: Check USB connection, restart software
- Blurry scans: Clean scanner glass with lens cloth
- File save error: Check network connectivity, verify folder permissions
```
