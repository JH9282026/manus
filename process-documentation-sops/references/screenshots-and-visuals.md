### Screenshots and Visuals

**Screenshot Guidelines:**
1. Capture only relevant portion of screen
2. Use consistent resolution and size
3. Add annotations (circles, arrows, numbers)
4. Highlight interactive elements
5. Update when interfaces change
6. Include alt text for accessibility

**Annotation Tools:**
- Snagit
- Greenshot
- Built-in OS screenshot tools
- Figma/Canva for annotations
