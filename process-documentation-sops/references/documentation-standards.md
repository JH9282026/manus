### Documentation Standards

**File Naming Convention:**
`[Type]-[Dept]-[Number]_[Title]_v[Version].[Extension]`
Example: `SOP-HR-005_Employee_Onboarding_v2.1.pdf`

**Version Numbering:**
- Major changes: 1.0 → 2.0
- Minor changes: 1.0 → 1.1
- Corrections: 1.0 → 1.0.1

**Status Labels:**
- Draft
- Under Review
- Approved
- Superseded
- Archived
