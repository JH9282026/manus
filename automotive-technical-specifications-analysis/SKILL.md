# Automotive Technical Specifications Analysis

Comprehensive technical analysis framework for automotive specifications across luxury and mass-market vehicle segments.

## Overview

This skill provides detailed powertrain analysis (ICE, hybrid, BEV), chassis and dynamics evaluation, safety system assessment, efficiency and emissions analysis, technology feature comparison, and segment-appropriate benchmarking across BMW, Mercedes-Benz, Porsche, Ford, and Chevrolet vehicles.

## Technical Analysis Domains

| Domain | Key Metrics |
|--------|-------------|
| Powertrain | HP, torque, 0-60 mph, top speed, transmission type |
| Efficiency | MPG/MPGe, electric range, charging speed |
| Safety | NHTSA/IIHS ratings, ADAS features |
| Chassis | Suspension type, steering, braking |
| Technology | Infotainment, connectivity, driver assistance |
| Build Quality | Materials, NVH, assembly precision |

## Segment Analysis Frameworks

### Luxury Focus (BMW, Mercedes-Benz, Porsche)
- 0-60 mph acceleration and track performance
- Power-to-weight ratios and chassis balance
- Premium audio and interior materials quality
- Advanced ADAS and autonomous-ready systems
- Customization and personalization options

### Mass-Market Focus (Ford, Chevrolet)
- Total cost of ownership and efficiency
- Payload, towing, and cargo capacity
- Standard equipment and value positioning
- Warranty coverage and reliability metrics
- Feature accessibility across trim levels
