# Crypto-Specific Indicators

Bitcoin and cryptocurrency-native analysis tools.

---

## Bitcoin Heat Map

### Methodology
Price deviation from 200-week moving average.

### Interpretation
| Color | Meaning |
|-------|--------|
| Purple/Blue | Historically undervalued |
| Green | Recovery zone |
| Yellow | Fair value |
| Orange | Getting expensive |
| Red | Historically overvalued |

### Application
- Long-term accumulation guide
- Cycle positioning indicator
- Not for short-term trading

## Bitcoin Rainbow Chart

### Methodology
Logarithmic regression with bands.

### Bands (bottom to top)
1. Fire sale (blue)
2. Buy (green)
3. Accumulate (light green)
4. Still cheap (yellow)
5. HODL (orange)
6. Is this a bubble? (light orange)
7. FOMO intensifies (red)
8. Sell (dark red)
9. Maximum bubble (purple)

### Limitations
- Based on historical patterns
- Diminishing returns each cycle
- Use with other indicators

## Puell Multiple

### Methodology
```
Puell = Daily Mining Revenue / 365-day MA of Mining Revenue
```

### Interpretation
| Level | Signal |
|-------|--------|
| <0.5 | Miner capitulation (buy zone) |
| 0.5-1.0 | Undervalued |
| 1.0-2.0 | Fair value |
| >4.0 | Overheated (sell zone) |

### Contrarian Use

## Stock-to-Flow Model

### Methodology
```
S2F = Stock / Annual Production
```

### Application
- Scarcity-based valuation
- Halving impact modeling
- Controversial predictive value

### Criticisms
- Model has shown significant deviations
- Doesn't account for demand side
- Diminishing accuracy over time
