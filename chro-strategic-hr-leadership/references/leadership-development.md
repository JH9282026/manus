# Leadership Development

## Leadership Competency Framework

### Framework Structure

| Level | Focus | Key Competencies |
|-------|-------|------------------|
| Individual contributor | Self-leadership | Accountability, learning agility |
| First-line manager | Team leadership | Coaching, delegation, feedback |
| Middle manager | Cross-functional | Influence, strategic thinking |
| Senior leader | Enterprise | Vision, transformation, governance |
| Executive | External | Stakeholder management, industry shaping |

### Core Leadership Competencies
- Strategic thinking
- Results orientation
- People development
- Change leadership
- Collaboration and influence
- Decision quality
- Communication effectiveness

## Leadership Assessment

### Assessment Methods

| Method | Best For | Considerations |
|--------|----------|----------------|
| 360 feedback | Development | Self-awareness, blind spots |
| Assessment centers | Selection, potential | Investment, validity |
| Psychometric testing | Personality, capability | Professional interpretation |
| Simulation | Realistic preview | Time, resource intensive |
| Performance review | Track record | Historical, biased |

### High-Potential Identification
- Performance track record
- Leadership competencies
- Learning agility
- Drive and aspiration
- Cultural fit

## Development Approaches

### 70-20-10 Model
- **70% Experience**: Challenging assignments, projects
- **20% Exposure**: Mentoring, coaching, feedback
- **10% Education**: Training, courses, reading

### Development Experiences

| Experience Type | Development Focus |
|-----------------|-------------------|
| Stretch assignment | New capability building |
| Cross-functional move | Breadth, perspective |
| International assignment | Global mindset |
| Turnaround | Performance pressure |
| Start-up/greenfield | Entrepreneurship |
| Large team leadership | Scale management |

## Leadership Programs

### Program Types

| Program | Target | Duration | Focus |
|---------|--------|----------|-------|
| New manager | First-time managers | 3-6 months | Fundamentals |
| High-potential | Emerging leaders | 12-18 months | Acceleration |
| Executive | Senior leaders | 6-12 months | Enterprise leadership |
| Succession | C-suite pipeline | Ongoing | Readiness |

### Program Components
- Assessment and feedback
- Formal learning (workshops, courses)
- Action learning projects
- Coaching and mentoring
- Peer learning cohorts
- Leadership exposure

## Executive Coaching

### When to Use Coaching
- Leadership transition
- Performance enhancement
- Specific skill development
- Executive presence
- Team effectiveness
- Career planning

### Coaching Framework
1. Establish goals and measures
2. Assess current state
3. Create development plan
4. Execute with support
5. Measure progress
6. Sustain and embed
