# Leadership Development

Build leadership pipelines through assessment, development programs, and coaching frameworks.

---

## Leadership Competency Framework

### Universal Leadership Competencies

| Competency | Description | Assessment Methods |
|---|---|---|
| Strategic thinking | Set direction, anticipate trends, make trade-offs | Case studies, 360 feedback |
| People leadership | Inspire, develop, and retain talent | 360 feedback, engagement scores |
| Execution excellence | Drive results, manage performance, remove barriers | Business results, project outcomes |
| Change leadership | Navigate ambiguity, drive transformation, build resilience | Change project outcomes, adaptability assessments |
| Collaboration | Build relationships, influence without authority, resolve conflict | Peer feedback, cross-functional project results |
| Innovation | Challenge status quo, experiment, learn from failure | Innovation portfolio, ideation metrics |

### Level-Specific Competency Expectations

| Level | Key Shift | Development Focus |
|---|---|---|
| Individual contributor → Manager | Managing self → Managing others | Delegation, feedback, coaching basics |
| Manager → Director | Managing others → Managing managers | Strategic alignment, talent development, cross-functional |
| Director → VP | Managing managers → Managing functions | Enterprise thinking, executive presence, P&L accountability |
| VP → C-suite | Managing functions → Managing the enterprise | Board engagement, stakeholder management, organizational transformation |

## High-Potential Programs

### Identification Criteria

Assess potential using validated factors:
- **Aspiration** — Desire for advancement and expanded responsibility
- **Ability** — Cognitive agility, learning speed, and emotional intelligence
- **Engagement** — Commitment to the organization and its mission
- **Track record** — Consistent performance across diverse situations

### Program Design

**12-Month High-Potential Accelerator Structure:**

| Quarter | Focus | Activities |
|---|---|---|
| Q1 | Assessment and awareness | 360 feedback, personality assessments, development planning |
| Q2 | Strategic projects | Cross-functional strategic initiative, executive mentoring |
| Q3 | External exposure | Customer immersion, industry conferences, board observation |
| Q4 | Integration and evaluation | Presentation to executive team, promotion readiness assessment |

## Executive Coaching Framework

### Coaching Engagement Model

1. **Contracting** — Define coaching objectives, success criteria, and confidentiality
2. **Assessment** — 360 feedback, personality instruments, stakeholder interviews
3. **Development planning** — Prioritize 2-3 development areas with specific goals
4. **Coaching sessions** — Bi-weekly 60-minute sessions over 6-12 months
5. **Progress review** — Mid-point and final 360 reassessment
6. **Transition** — Sustain development through self-coaching and peer support

### Common Executive Development Areas

- Executive presence and influence
- Strategic communication and storytelling
- Delegation and empowerment
- Managing enterprise-level stakeholders
- Leading organizational transformation
- Work-life integration and resilience
