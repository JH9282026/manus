# Design Principles & Systems

## Color Theory
- Color wheel: primary, secondary, tertiary relationships
- Harmonies: complementary, analogous, triadic, split-complementary
- Psychology: red (urgency), blue (trust), green (growth), yellow (optimism), purple (luxury)
- Brand palette: 2 primary + 2-3 secondary + 1 accent + 2-3 neutrals
- Specify in HEX, RGB, CMYK, and Pantone for cross-media consistency

## Typography
- Pairing: combine contrast (serif + sans-serif) or complementary styles
- Hierarchy: heading (display/bold), subheading (medium), body (regular), caption (light)
- Readability: line length 50-75 chars, line height 1.4-1.6x, adequate spacing
- Web: Google Fonts, Adobe Fonts; system fonts for performance
- Print: ensure licensing; outline fonts for production files

## Grid Systems
- 8-point grid: base unit for spacing (8, 16, 24, 32, 48, 64px)
- 12-column grid: standard for web and print layouts
- Modular grid: for complex editorial layouts
- Baseline grid: align text across columns for print

## Brand Identity System
1. Logo system (primary, secondary, icon, reversed, monochrome)
2. Color palette with cross-media specifications
3. Typography hierarchy with size scales
4. Imagery guidelines (photography, illustration style)
5. Iconography system
6. Clear space and minimum size rules
7. Usage do's and don'ts

## Design Process
1. Brief and research: objectives, audience, constraints
2. Concept development: sketches, mood boards, style exploration
3. Design execution: digital creation with proper specs
4. Review and iterate: stakeholder feedback, refinement
5. Production: file preparation for final output
6. Delivery: organized asset library with documentation
