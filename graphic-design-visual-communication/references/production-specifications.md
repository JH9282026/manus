# Production Specifications

## Print Production
- Color mode: CMYK (never RGB for print)
- Resolution: 300 DPI minimum at final size
- Bleed: 3mm on all sides beyond trim
- Safe zone: 5mm from trim for critical content
- Outline all fonts or embed them
- Export as PDF/X-1a for compatibility

## Web Optimization
- WebP for modern browsers (30% smaller than JPEG)
- PNG for transparency and crisp graphics
- JPEG for photographs (80-85% quality)
- SVG for logos and icons
- Target: under 200KB per web image

## Accessibility
- All images need descriptive alt text
- Color contrast minimum 4.5:1 for text
- Never rely on color alone for meaning
- Test with colorblindness simulators
