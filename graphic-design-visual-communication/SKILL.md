# Graphic Design & Visual Communication

Create compelling visual content to communicate messages, establish brand identities, and engage audiences across print and digital media.

## Overview

This skill provides frameworks for applying design principles to solve visual communication challenges. It covers brand identity systems, marketing collateral, infographics, social media graphics, and print production with specifications for professional output.

## Core Design Principles

| Principle | Application | Rule of Thumb |
|-----------|-------------|--------------|
| Balance | Symmetrical, asymmetrical, or radial arrangements | Distribute visual weight evenly |
| Contrast | Light/dark, large/small, thick/thin | High contrast = readability + hierarchy |
| Hierarchy | Size, color, position guide attention | 60-30-10 rule for emphasis levels |
| Alignment | Invisible lines connect elements | Left-align for Western reading |
| Proximity | Group related; separate unrelated | Reduces cognitive load |
| Repetition | Recurring colors, shapes, typography | Builds unity and recognition |

## Format Specifications

| Asset Type | Format | Resolution | Color Mode |
|------------|--------|------------|------------|
| Logo | SVG + PNG transparent | Vector | RGB + CMYK |
| Social | PNG / JPEG | 72 DPI | sRGB |
| Print | PDF/X-1a | 300 DPI | CMYK |
| Web | WebP / PNG | 72 DPI | sRGB |

## Quality Standards
- Body text: min 16px (web) / 10pt (print)
- Color contrast: 4.5:1 normal text, 3:1 large text (WCAG AA)
- Spacing: 8-point grid system
- Brand colors: 60-30-10 distribution
