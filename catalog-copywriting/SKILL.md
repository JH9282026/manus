# Catalog Copywriting

Create effective product descriptions for catalogs that efficiently communicate features, benefits, and purchase information within space constraints.

## Overview

This skill enables creation of compelling catalog copy across formats—from print catalogs to e-commerce listings. Use this to write product descriptions that drive purchases while maintaining brand voice consistency across hundreds or thousands of items.

## Catalog Types

| Type | Focus | Copy Style |
|------|-------|------------|
| Consumer Product | Benefits, lifestyle | Engaging, aspirational |
| B2B/Industrial | Specifications, applications | Technical, precise |
| E-commerce | Search, conversion | SEO-optimized, scannable |
| Lookbook | Mood, trends | Evocative, minimal |
| Technical | Specs, compatibility | Detailed, accurate |

## Product Description Structure

### Standard Format
1. **Headline**: Product name + key benefit
2. **Lead**: Opening hook or value statement
3. **Body**: Features, benefits, details
4. **Specs**: Technical specifications
5. **CTA**: Ordering information

### Length Guidelines
| Format | Word Count |
|--------|------------|
| Short-form (e-commerce) | 50-100 words |
| Standard catalog | 75-150 words |
| Featured product | 150-300 words |
| Technical/B2B | 200-400 words |

## Writing Approaches by Catalog Type

### Consumer Products
- Lead with benefits, not features
- Create emotional connection
- Use sensory language
- Include lifestyle context

### B2B/Industrial
- Lead with key specifications
- Include application uses
- Note compatibility/requirements
- Provide ordering information

### E-commerce
- Front-load keywords
- Use scannable formatting
- Include search-friendly attributes
- Optimize for product filters

## Feature-Benefit Conversion

| Feature | Benefit |
|---------|---------|
| "Made from 100% cotton" | "Naturally soft and breathable comfort" |
| "5-year warranty" | "Peace of mind for years to come" |
| "Wireless connectivity" | "Freedom to move without tangled cords" |
| "500GB storage" | "Room for 100,000+ photos" |

### FAB Framework
- **Feature**: What it is/has
- **Advantage**: What the feature does
- **Benefit**: Why it matters to customer

## Essential Copy Elements

### Product Headlines
| Type | Example |
|------|---------|
| Benefit-led | "All-Day Comfort Work Boot" |
| Feature-led | "Waterproof Insulated Parka" |
| Audience-led | "Professional Chef's Knife Set" |

### Specification Format
```
- Material: 18/10 Stainless Steel
- Dimensions: 12" L x 8" W x 4" H
- Weight: 2.5 lbs
- Capacity: 64 oz
```

### Callout Elements
- **NEW** - New products
- **BESTSELLER** - Popular items
- **LIMITED** - Scarcity messaging
- **EXCLUSIVE** - Unique offerings

## SEO Considerations (Digital)

### Keyword Integration
- Include primary keyword in headline
- Use secondary keywords naturally in body
- Include product attributes (color, size, material)
- Add brand name where appropriate

### Structured Data
- Product name
- Description
- Price
- Availability
- Reviews/ratings
- SKU/product ID

## Consistency at Scale

### Style Guide Elements
- Voice and tone guidelines
- Sentence structure templates
- Specification formats
- Capitalization rules
- Abbreviation standards

### Templated Approach
```
[Product Name]: [Key Benefit Statement]

[Opening hook connecting to customer need]. [Primary feature 
and its benefit]. [Secondary features]. [Quality/material 
note]. [Usage suggestion or context].

Specifications:
- [Spec 1]: [Value]
- [Spec 2]: [Value]
```
