# Catalog Copywriting

Write persuasive, informative content for product catalogs that drives purchase decisions within space constraints.

## Overview

Catalog copywriting combines product knowledge with persuasive writing to communicate value within strict space limitations. This skill covers product descriptions, category introductions, technical specifications, and promotional messaging for both print and digital catalog formats.

## Catalog Copy Fundamentals

### Core Principles

| Principle | Application |
|-----------|-------------|
| Every Word Must Work | No filler—each word contributes to understanding or persuasion |
| Consistency | Same voice, format, and quality across all products |
| Scannability | Format for easy scanning, not deep reading |
| Completeness | Include all information needed for purchase decisions |

### Information Hierarchy

| Priority | Content |
|----------|---------|
| 1 | Product Name—clear, descriptive, searchable |
| 2 | Key Benefit/Hook—why buy this product |
| 3 | Features—what it has/does |
| 4 | Benefits—what features mean for user |
| 5 | Specifications—size, materials, dimensions |
| 6 | Variations—colors, sizes, options |
| 7 | Price—clear pricing with variables |
| 8 | Ordering Info—how to buy |

## Product Description Formats

### Short-Form (25-50 words)

**Structure**:
1. Opening hook or key benefit
2. Primary features
3. Differentiator

**Example**:
"Make perfect julienne cuts in seconds with our professional-grade mandoline. Adjustable blade settings from paper-thin to 1/4 inch. Stainless steel construction with non-slip base for safe, effortless slicing."

### Medium-Form (50-100 words)

**Structure**:
1. Compelling opening
2. Primary benefits and features
3. Secondary features
4. Use case or lifestyle context
5. Quality statement

**Example**:
"Transform your patio into an outdoor living room with our weather-resistant wicker sectional. The rust-proof aluminum frame supports plush, quick-dry cushions in your choice of six colors. Modular design lets you configure seating for intimate conversations or entertaining crowds.

UV-resistant resin wicker maintains its rich color season after season, while removable cushion covers make cleaning a breeze."

### Long-Form (100-200+ words)

**Structure**:
1. Headline/hook
2. Opening paragraph with primary value
3. Feature bullets or paragraphs
4. Quality and construction details
5. Use cases and applications
6. Warranty or guarantee

## Feature-Benefit Writing

### Feature to Benefit Translation

| Feature | Benefit |
|---------|---------|
| 300-thread-count Egyptian cotton | Sleep in hotel-quality luxury every night |
| 5-year warranty | Peace of mind knowing your investment is protected |
| Dishwasher safe | Easy cleanup—just load and go |
| Lightweight at 8.5 oz | Comfortable for all-day wear |

### Integration Example

"Crafted from 300-thread-count Egyptian cotton for hotel-quality softness that improves with every wash. Your best night's sleep is guaranteed—backed by our 5-year warranty."

## Category Descriptions

### Category Introduction Purpose

- Set context for product browsing
- Build anticipation and desire
- Guide product selection
- Establish expertise

### Category Structure

```
## [Category Name]

[Opening hook—attention-grabbing statement]

**What You'll Find:**
[Overview of products in this section]

**Why Shop Here:**
[Key benefits of this category]

**Selection Guidance:**
[How to choose the right product]
```

**Example**:
```
## Outdoor Living

Your backyard retreat starts here. From morning coffee on the patio to starlit dinner parties, our outdoor collection brings indoor comfort outside.

**What You'll Find:**
Dining sets for 2 to 12, conversation seating, lounge furniture, and accessories.

**Built to Last:**
Every piece features weather-resistant materials designed for seasons of sun, rain, and everything in between.
```

## Technical Specifications

### Specification Formatting

**Inline Format** (simple specs):
"16" H x 24" W x 12" D. Solid oak with walnut veneer. 18 lbs."

**Bullet Format** (moderate specs):
```
• Dimensions: 16" H x 24" W x 12" D
• Material: Solid oak frame, walnut veneer top
• Weight: 18 lbs
• Assembly: Minimal—attach legs only
```

**Table Format** (complex specs):
| Specification | Details |
|---------------|---------|
| Dimensions | 16" H x 24" W x 12" D |
| Frame | Solid white oak |
| Finish | Hand-rubbed natural oil |
| Weight Capacity | 50 lbs |

## Pricing and Promotional Copy

### Price Presentation

| Format | Example |
|--------|---------|
| Standard | $149 |
| Sale | $149 ~~$199~~ SAVE $50 |
| Tiered | Starting at $149 |
| Per Unit | $24.99/yard |

### Promotional Messaging

| Type | Example |
|------|---------|
| Percentage Off | "20% OFF All Bedding — Use code SLEEP20" |
| Dollar Off | "SAVE $50 on orders over $200" |
| Free Shipping | "FREE SHIPPING on your entire order" |
| Bundle | "Complete Room Package — Save 15%" |

### Value Messaging

- Quality justification: "Handcrafted in America—built to last generations"
- Comparison value: "Comparable to brands costing twice as much"
- Cost per use: "At just $0.50 per use, it pays for itself"

## Cross-Selling Techniques

### Cross-Sell Copy

**"Complete the Look"**:
```
Complete the Look:
• Matching Chair Cushions (p. 34) — $49
• Coordinating Side Table (p. 38) — $129
• Outdoor Rug Collection (p. 42) — from $79
```

**"Pairs Well With"**:
```
Pairs perfectly with:
• Pinot Noir Collection (p. 28)
• Artisan Cheese Board (p. 44)
```

### Upselling Copy

**"Upgrade To"**:
```
Love this chair? Upgrade to our Premium Collection:
• Memory foam cushioning
• Upgraded leather
• Extended warranty
See Premium Series on page 56 →
```

## Print vs Digital Considerations

### Print Catalog Constraints

- Fixed space per product
- No linking or interactivity
- Must be complete—can't link to more info
- Phone/web ordering must be clear

### Digital Catalog Capabilities

- Unlimited space (accordion/expand)
- Clickable links and navigation
- SEO optimization important
- Mobile formatting considerations

### Hybrid Approaches

- Print drives to digital: "See all 12 colors at company.com/colors"
- Digital extends print: "As featured in our Spring Catalog, page 24"

## SEO for Digital Catalogs

### On-Page Elements

| Element | Optimization |
|---------|--------------|
| Product Titles | Include primary keywords naturally |
| Meta Descriptions | Compelling summary with keywords |
| Alt Text | Descriptive image text |
| URL Structure | Clean, keyword-inclusive |

### Keyword Integration

- Product type and variations
- Key features and materials
- Use cases and applications
- Brand and collection names

## Best Practices

1. **Know Your Products** — Research thoroughly before writing
2. **Know Your Audience** — Write for their needs and knowledge
3. **Lead with Benefits** — Put what matters most first
4. **Be Specific** — Concrete details over vague claims
5. **Use Active Voice** — Direct and engaging
6. **Maintain Consistency** — Same format throughout
7. **Edit Ruthlessly** — Every word must earn its space
8. **Include CTAs** — Make it easy to buy

## Common Mistakes

| Mistake | Fix |
|---------|-----|
| Generic descriptions | Customize each with unique details |
| Features without benefits | Always explain what features mean |
| Inconsistent formatting | Follow style guide |
| Missing specifications | Include all purchase-relevant specs |
| Overwriting | Edit ruthlessly for space |
| No mobile testing | Check on mobile devices |
