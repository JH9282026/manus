# Product Templates

Category-specific templates for catalog product descriptions.

---

## Apparel Template

### Structure
```
[Product Name] - [Key Style/Benefit]

[Opening line about style or occasion]. [Material and 
comfort benefits]. [Design details]. [Versatility note].

Fit: [Fit type]
Material: [Fabric composition]
Care: [Care instructions]
Available in: [Colors/sizes]
```

### Example
```
Classic Oxford Button-Down - Timeless Professional Style

Elevate your workweek wardrobe with this versatile oxford 
shirt. Crafted from premium cotton with a soft hand feel 
that gets better with every wash. Features a spread collar, 
adjustable cuffs, and a tailored fit that looks polished 
tucked or untucked.

Fit: Tailored
Material: 100% Premium Cotton
Care: Machine washable
Available in: White, Blue, Pink, Stripe | S-XXL
```

## Electronics Template

```

[Problem it solves or enhancement it provides]. [Key 
technology/feature and what it enables]. [Secondary 
features]. [Compatibility/ecosystem note].

- [Key spec]: [Value]
- [Key spec]: [Value]
- [Key spec]: [Value]

Includes: [What's in box]

Escape into your music with industry-leading noise 
cancellation that blocks out the world. 40mm custom drivers

- Driver: 40mm custom
- Battery: 30 hours
- Connectivity: Bluetooth 5.0, 3.5mm
- Noise Cancellation: Adaptive ANC

Includes: Headphones, carrying case, USB-C cable, 3.5mm cable
```

## Home/Furniture Template

### Structure
```
[Product Name] - [Room/Use + Style]

[Visual/aesthetic description]. [Construction and materials]. 
[Practical features]. [Styling suggestion].

Dimensions: [L x W x H]
Materials: [Primary materials]
Assembly: [Required/Not required]
Weight Capacity: [If applicable]
```

### Example
```
Mid-Century Modern Coffee Table - Living Room Centerpiece

Bring retro sophistication to your living space with clean 
lines and warm walnut tones. Solid hardwood legs with 
smooth oval top create an inviting focal point. Built-in 
shelf provides practical storage for magazines and remotes. 
Pairs beautifully with both modern and traditional seating.

Dimensions: 48" L x 24" W x 18" H
Materials: Walnut veneer top, solid wood legs
Assembly: Some assembly required
Weight Capacity: 50 lbs
```

## B2B/Industrial Template

### Structure
```
[Product Name] | [Model Number]

[Primary application/purpose]. [Key performance feature]. 
[Durability/reliability note]. [Compliance/certifications].

Specifications:
- [Technical spec]: [Value]
- [Technical spec]: [Value]
- [Technical spec]: [Value]

Applications: [List of use cases]
Compliance: [Relevant certifications]
```

### Example
```
Industrial Pump Series 400 | Model 400-HP50

Designed for continuous heavy-duty pumping applications in 
manufacturing and processing environments. Delivers 50 GPM 
flow rate with 99.5% efficiency. Cast iron construction 
withstands demanding conditions. UL Listed and CSA Certified.

Specifications:
- Flow Rate: 50 GPM
- Head Pressure: 120 PSI max
- Motor: 5 HP, 230/460V 3-phase
- Inlet/Outlet: 2" NPT

Applications: Chemical transfer, water treatment, industrial processes
Compliance: UL Listed, CSA Certified, OSHA compliant
```

## Short-Form E-commerce Template

### Structure (50-75 words)
```
[Benefit-led opening sentence]. [Key feature with benefit]. 
[Secondary feature]. [Quality or trust signal].

✓ [Bullet point 1]
✓ [Bullet point 2]
✓ [Bullet point 3]
```

### Example
```
Stay organized on the go with this expandable laptop backpack. 
Waterproof exterior protects your gear in any weather. Padded 
compartment fits laptops up to 15.6". Backed by our 5-year warranty.

✓ Waterproof material
✓ 15.6" laptop compatibility
✓ Multiple organizer pockets
```
