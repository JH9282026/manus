# Product Description Templates

Templates and examples for various product categories.

---

## Apparel

### Template

```
[Lifestyle/benefit opening]. This [garment type] features [key features] for [benefit]. [Material details] ensure [quality benefit]. [Care instruction]. [Fit note if relevant].
```

### Example

"Weekend-ready softness meets polished style. This brushed cotton button-down features our signature relaxed fit, mother-of-pearl buttons, and side vents for easy layering. Premium Pima cotton gets softer with every wash. Machine washable for effortless care. True to size; size up for an oversized look."

### Key Elements

| Element | What to Include |
|---------|----------------|
| Fit | True to size, relaxed, slim, oversized |
| Material | Fabric type, weight, feel |
| Care | Wash instructions |
| Details | Buttons, stitching, hardware |
| Sizing | Size guidance, measurements |
