# Style Consistency

Guidelines for maintaining consistent voice and format across large catalogs.

---

## Voice Guidelines

### Tone Spectrum by Catalog Type

| Type | Tone | Examples |
|------|------|----------|
| Luxury | Sophisticated, exclusive | "Exquisite craftsmanship", "Refined elegance" |
| Casual Consumer | Friendly, approachable | "You'll love", "Perfect for" |
| Technical/B2B | Professional, precise | "Engineered for", "Delivers performance" |
| Lifestyle | Aspirational, emotive | "Transform your", "Experience" |

### Voice Attributes
Define 3-4 attributes for your catalog:
- Example: Knowledgeable, Helpful, Confident, Warm

### Language Register
- **Formal**: "This garment features premium construction"
- **Conversational**: "You'll notice the premium construction"
- **Direct**: "Premium construction throughout"

---

## Structural Consistency

### Standard Paragraph Structure
1. Opening: Hook or benefit statement
2. Body: Features and benefits
3. Close: Usage context or call-to-action

### Sentence Patterns
- Vary sentence length (short-medium-short)
- Lead sentences with benefits, not features
- End with forward momentum

### Word Count Standards
| Element | Count |
|---------|-------|
| Headline | 3-8 words |
| Lead sentence | 15-25 words |
| Body paragraph | 50-100 words |
| Bullet points | 5-15 words each |

---

## Specification Formatting

### Standard Format
```
Category: Value
Category: Value
```

### Units
- Dimensions: L x W x H (inches or cm, be consistent)
- Weight: lbs or kg (consistent throughout)
- Volume: oz, ml, or liters

### Abbreviations
- Define approved abbreviations
- Use consistently
- Expand on first use if needed

---

## Capitalization Rules

### Headlines
- Title Case for product names
- Consistent across all products

### Body Copy
- Sentence case for descriptions
- Capitalize proper nouns and brand names

### Specifications
- Title Case for category labels
- Normal case for values

---

## Punctuation Standards

### Bullets
- No periods for fragments
- Periods for complete sentences
- Consistent within each product

### Dashes
- En dash for ranges (5-10 lbs)
- Em dash for breaks—sparingly

### Serial Comma
- Decide yes or no, apply consistently

---

## Quality Control Checklist

### Per-Product Check
- [ ] Headline follows format
- [ ] Word count within range
- [ ] All specifications complete
- [ ] Voice matches guidelines
- [ ] No spelling/grammar errors
- [ ] SEO keywords included (if digital)

### Catalog-Wide Check
- [ ] Consistent formatting throughout
- [ ] All required sections present
- [ ] Specifications in same order
- [ ] Voice consistency across categories
- [ ] Cross-references accurate

## Template Management

### Master Template Library
- One template per product category
- Version control for changes
- Examples with annotations

### Update Process
1. Propose template change
2. Review with team
3. Update master template
4. Apply to new products
5. Phase update to existing products

### Training Materials
- Template examples
- Before/after comparisons
- Common mistakes to avoid
- Quick reference card
