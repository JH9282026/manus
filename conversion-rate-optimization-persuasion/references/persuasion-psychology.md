# Persuasion Psychology

Psychological principles and cognitive biases for conversion optimization.

---

## Cialdini's Six Principles

### 1. Reciprocity
Give value before asking for action.
- Free content, tools, or trials
- Unexpected bonuses
- Personalized assistance

### 2. Commitment & Consistency
Start with small asks, build commitment.
- Progressive profiling
- Micro-conversions first
- Public commitments

### 3. Social Proof
Show others taking the action.
- Customer counts
- Testimonials and reviews
- Real-time activity notifications
- Case studies

### 4. Authority
Demonstrate expertise and credibility.
- Expert endorsements
- Certifications and awards
- Media mentions
- Professional design

### 5. Liking
Build connection and relatability.
- Team photos and stories
- Brand personality
- Shared values
- Friendly tone

### 6. Scarcity
Create urgency through limitation.
- Limited time offers
- Limited quantity
- Exclusive access
- Countdown timers

---

## Cognitive Biases

### Anchoring
First number influences perception.
- Show original price before discount
- Start with higher tier
- Compare to more expensive alternatives

### Loss Aversion
Losses feel 2x stronger than gains.
- "Don't miss out" framing
- Free trial expiration warnings
- Abandonment recovery emails

### Framing Effect
How information is presented matters.
- "90% fat-free" vs "10% fat"
- Savings vs. costs
- Benefits vs. features

### Decoy Effect
Add option to make target choice appealing.
- Three-tier pricing with middle target
- Clearly inferior options

## Application Guidelines

### Ethical Use
- Be truthful and accurate
- Create genuine value
- Respect user autonomy
- Avoid manipulation
