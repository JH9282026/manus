# Conversion Rate Optimization (CRO) & Persuasion Intelligence

Systematically analyze, diagnose, and optimize digital properties for maximum conversions using psychology principles and rigorous testing.

## Overview

This skill encompasses the complete conversion optimization lifecycle—from research and hypothesis development through testing, implementation, and continuous iteration. Typical improvements range 15-50% on optimized elements.

## CRO Audit Process

### 1. Data Collection
- Funnel analytics (drop-off points)
- Heatmaps (click, scroll, attention)
- Session recordings
- User flow analysis

### 2. Friction Identification
- Navigation confusion
- Form abandonment triggers
- Trust concerns
- Clarity issues

### 3. Hypothesis Development
```
IF we [change],
THEN [metric] will [improve by X%],
BECAUSE [reason based on evidence]
```

### 4. Prioritization (ICE Framework)
- **Impact**: Potential improvement (1-10)
- **Confidence**: Evidence strength (1-10)
- **Ease**: Implementation effort (1-10)
- **Score**: Average of three

### 5. Testing & Implementation
- A/B test design
- Sample size calculation
- Statistical significance (95%)
- Winner deployment

## Persuasion Principles (Cialdini)

1. **Reciprocity**: Give before asking
2. **Commitment**: Start small, build up
3. **Social Proof**: Show others doing it
4. **Authority**: Demonstrate expertise
5. **Liking**: Build connection
6. **Scarcity**: Create urgency
