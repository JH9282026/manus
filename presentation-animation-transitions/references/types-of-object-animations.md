## Types of Object Animations



### Entrance Effects

Control how objects appear on the slide:

| Effect | Description | Best Use |
|--------|-------------|----------|
| Appear | Instant appearance | Quick, no-frills reveal |
| Fade | Gradual opacity increase | Gentle, professional |
| Fly In | Enters from off-screen | Direction/progression |
| Float In | Gentle floating entrance | Elegant, subtle |
| Zoom | Scales from small to full size | Emphasis, focus |
| Bounce | Enters with bouncing effect | Playful, energetic |
| Grow & Turn | Rotates while growing | Creative impact |
| Wipe | Progressively revealed | Sequential, building |

**Best Practices for Entrance Effects**:
- Use consistent entrance effects within a presentation
- Match entrance direction to content flow (left-to-right for Western audiences)
- Reserve dramatic entrances for key content
- Fade is the safest default for professional contexts



### Emphasis Effects

Draw attention to elements already on screen:

| Effect | Description | Best Use |
|--------|-------------|----------|
| Pulse | Brief scaling up and down | Subtle attention |
| Spin | Rotation in place | Dynamic emphasis |
| Grow/Shrink | Size change | Importance indication |
| Transparency | Opacity change | Dimming/highlighting |
| Bold Flash | Brief color/style change | Quick emphasis |
| Teeter | Slight rocking motion | Playful attention |
| Color Change | Gradual color shift | State change |

**Best Practices for Emphasis Effects**:
- Use sparingly—emphasis loses impact with overuse
- Pair with verbal cues from presenter
- Ensure color changes maintain accessibility
- Keep durations brief (0.3-0.5 seconds)



### Exit Effects

Control how objects leave the slide:

| Effect | Description | Best Use |
|--------|-------------|----------|
| Disappear | Instant removal | Quick transitions |
| Fade | Gradual opacity decrease | Gentle exit |
| Fly Out | Exits off-screen | Continuation elsewhere |
| Float Out | Gentle floating exit | Elegant departure |
| Shrink | Scales down to nothing | Diminishing importance |
| Collapse | Vertical compression | Closing/concluding |

**Best Practices for Exit Effects**:
- Exit effects often unnecessary—new content replaces old
- Match exit direction to entrance for consistency
- Use exits to clear space before new content appears
- Fade exits work universally



### Motion Paths

Objects follow custom or preset paths across the slide:

**Preset Paths**:
- Lines (straight movement)
- Arcs (curved movement)
- Turns (multi-directional)
- Shapes (circles, squares, triangles)
- Loops (returning paths)

**Custom Paths**:
- Freeform drawing of exact movement
- Multiple waypoints and curves
- Precise control over trajectory

**Best For**:
- Showing processes and workflows
- Demonstrating movement or travel
- Creating complex animations
- Guiding eye across slide content
