## Animation Timing and Sequencing



### Start Triggers

When animations begin relative to other events:

**On Click**: Animation starts when presenter clicks (maximum control)
**With Previous**: Animation starts simultaneously with the previous animation
**After Previous**: Animation starts immediately after previous animation completes



### Timing Considerations

**Duration**: How long the animation takes
- Short (0.25-0.5s): Quick, efficient
- Medium (0.5-1.0s): Standard, natural
- Long (1.0-2.0s): Dramatic, deliberate

**Delay**: Pause before animation begins after trigger
- Useful for staggered reveals
- Creates anticipation
- Allows reading time

**Speed Consistency**: Maintain similar durations throughout presentation for cohesive feel.



### Creating Animation Sequences

Sequential animations build complex reveals:

1. **Plan the sequence**: Map out what appears when
2. **Set triggers appropriately**: On Click for presenter control, After Previous for automatic sequences
3. **Add delays**: Stagger elements for readability
4. **Preview frequently**: Ensure timing feels natural
5. **Test with narration**: Animations should complement speech timing
