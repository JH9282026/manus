## Performance and File Size Considerations


### Factors Affecting Performance

**File Size Contributors**:
- High-resolution images
- Embedded videos
- Complex 3D models
- Numerous animation effects
- Multiple slides with effects

**Performance Drains**:
- Morph transitions with many objects
- Motion paths with many points
- Simultaneous animations
- Transparency effects
- Large image scaling


### Optimization Strategies

**Image Optimization**:
- Compress images before inserting
- Use appropriate resolution (150 DPI for screen)
- Avoid unnecessary high-resolution assets

**Animation Efficiency**:
- Limit simultaneous animations
- Simplify complex morph transitions
- Reduce motion path complexity
- Use simpler effects when possible

**File Management**:
- Compress media within presentation
- Remove unused slides and assets
- Save in appropriate format for delivery


### Platform Considerations

**Presenting on Different Hardware**:
- Test on actual presentation computer
- Older computers may struggle with complex animations
- Virtual meeting screen share affects performance

**Export Considerations**:
- Video exports may skip complex animations
- PDF exports lose all animation
- Cloud platforms may simplify effects
