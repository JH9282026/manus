## Testing and Previewing Animations


### Testing Workflow

1. **Animation Pane Review**: Check sequence and timing in animation panel
2. **Single Slide Preview**: Test each slide's animations individually
3. **Full Presentation Run**: Preview complete presentation start to finish
4. **Present Mode Test**: Run in actual presentation mode, not edit view
5. **External Display Test**: Test on secondary monitor/projector if applicable
6. **Timing Practice**: Run through while speaking to verify pace


### What to Check

- **Sequence**: Animations fire in intended order
- **Timing**: Duration and delays feel natural
- **Triggers**: Click-based vs. automatic as intended
- **Transitions**: Slides advance smoothly
- **Performance**: No lag or stuttering
- **Consistency**: Similar animations behave similarly throughout
- **Readability**: Content readable during and after animation


### Common Issues to Identify

- Text appearing before it should
- Elements out of position after animation
- Timing too fast or slow for narration
- Missing animations on later slides
- Transitions overriding object animations
- Performance slowdown on complex slides
