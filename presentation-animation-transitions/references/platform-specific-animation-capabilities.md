## Platform-Specific Animation Capabilities


### Microsoft PowerPoint

**Strengths**:
- Morph transition (powerful object transformation)
- Extensive animation presets
- Animation Pane for precise control
- Trigger-based animations
- Motion path customization

**Key Features**:
- 3D model animation support
- Animation Painter for consistency
- Reorder and timing controls
- After Previous timing
- Animation bookmarks

**Limitations**:
- Can become sluggish with complex animations
- Some effects render differently on different systems
- Video export may not capture all effects


### Apple Keynote

**Strengths**:
- Magic Move (intelligent morphing)
- Beautiful, polished presets
- Smooth performance
- Build In/Out/Action organization
- Object effects (reflection, shadow animation)

**Key Features**:
- Action builds for emphasis
- Drawing animation (hand-drawn effect)
- Photo animation (Ken Burns effect)
- Chart build animations
- Inter-slide linking

**Limitations**:
- Fewer animation options than PowerPoint
- Mac/iOS only (playback limitations)
- Limited export options


### Google Slides

**Strengths**:
- Cloud-native collaboration
- Simple, clean interface
- Cross-platform compatibility
- Basic but reliable animations

**Key Features**:
- Fade, fly, zoom effects
- On click/after previous timing
- Object-level animations
- Slide transitions

**Limitations**:
- Fewer animation options
- No morph/magic move equivalent
- Limited motion path support
- Basic timing controls
