## When to Use Animations



### Appropriate Contexts

**Building Complex Diagrams**: Step-by-step reveal of system components
**Process Explanations**: Sequential workflow demonstration
**Data Visualization**: Progressive chart building for clarity
**Storytelling**: Dramatic reveals enhancing narrative
**Before/After Comparisons**: Morphing between states
**Call-to-Action Moments**: Drawing attention to key actions
**Section Transitions**: Signaling topic changes
**Interactive Presentations**: Responding to audience choices
**Training Content**: Step-by-step instruction delivery
**Product Demonstrations**: Feature showcases



### Signals That Animation Would Help

- Content is complex and needs progressive disclosure
- Audience attention needs directing
- Process has clear sequential steps
- Comparison or transformation is being shown
- Key message deserves special emphasis
- Presentation feels static and disengaging
- Timing needs coordination with speech
