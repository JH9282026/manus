# Tools And Resources

Detailed reference content for the Presentation Animation Transitions skill.

---

## Tools and Resources


### Animation Design Tools

**Presentation Software**:
- Microsoft PowerPoint (Windows/Mac/Web)
- Apple Keynote (Mac/iOS/Web)
- Google Slides (Web)
- Prezi (Web - canvas-based)
- Canva Presentations (Web)

**Motion Design Software** (for custom animations):
- Adobe After Effects
- Apple Motion
- Blender (free)
- Cavalry


### Learning Resources

**Courses and Tutorials**:
- LinkedIn Learning: PowerPoint/Keynote animation courses
- YouTube: Presentation design channels
- Envato Tuts+: Motion design tutorials
- Skillshare: Animation fundamentals

**Books**:
- "The Animator's Survival Kit" by Richard Williams (principles)
- "Presentation Zen Design" by Garr Reynolds (design philosophy)
- "Slide:ology" by Nancy Duarte (visual communication)


### Inspiration Sources

- **Dribbble**: Motion design examples
- **Behance**: Presentation design portfolios
- **MotionDesign.school**: Motion principles
- **Apple Keynote Gallery**: Polished examples
- **SlidesCarnival**: Template inspiration


### Templates and Assets

**Template Sources**:
- Envato Elements (subscription library)
- SlideModel (presentation templates)
- GraphicRiver (individual templates)
- 24Slides (professional templates)

**Animation Assets**:
- Lottie Files (JSON animations)
