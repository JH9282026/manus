# Grant Writing & Fundraising

Strategically acquire financial resources through grants, donor cultivation, campaigns, and stewardship for nonprofit sustainability.

## Overview

This skill covers the full development function: identifying funding opportunities, writing compelling proposals, cultivating donor relationships, executing fundraising campaigns, and optimizing development program performance. It integrates persuasive writing, financial planning, and relationship management into cohesive fundraising strategy.

## Development Planning Framework

1. Assess current fundraising performance (retention rates, cost-to-raise, sources)
2. Set goals by source: grants, individuals, corporate, events, earned revenue
3. Build prospect pipeline: research, qualify, cultivate, solicit, steward
4. Create calendar: grant deadlines, campaign timelines, event dates
5. Define metrics: dollars raised, donor counts, retention, average gift, ROI

## Donor Engagement Cycle

```
Identify → Research → Cultivate → Solicit → Thank → Report → Renew/Upgrade
```

- **Identify**: Prospect research using wealth screening and affinity indicators
- **Research**: Capacity, philanthropic history, connection points
- **Cultivate**: Relationship-building touchpoints aligned with donor interests
- **Solicit**: Personalized ask at appropriate level with compelling case
- **Thank**: Immediate acknowledgment within 48 hours
- **Report**: Impact reports showing how gifts created change
- **Renew/Upgrade**: Retention strategy with upgrade pathways
