# Offers, Response Devices, and Testing

## Offer Construction

### Offer Elements

| Element | Options | Impact |
|---|---|---|
| Price | Full, discount, installments | High |
| Premium/bonus | Free gift, add-on, upgrade | High |
| Guarantee | Money-back, satisfaction, lifetime | High |
| Trial | Free trial, sample, demo | High |
| Urgency | Deadline, limited quantity | Medium |
| Payment terms | Pay later, installments, subscription | Medium |
| Shipping | Free, flat rate, expedited | Medium |

### Proven Offer Types
- **Free trial**: "Try it for 30 days, risk-free"
- **Discount**: "Save 25% — this week only"
- **BOGO**: "Buy one, get one free"
- **Premium**: "Order today and receive [bonus] free"
- **Charter/founder**: "Be among the first 500 members"
- **Payment plan**: "Just 3 easy payments of $29.95"

## Response Device Design

### Order Form Best Practices
- Restate the offer on the order form (it's often separated from the letter)
- Use a "YES!" acceptance statement
- Include all product options with clear pricing
- Provide multiple response channels (mail, phone, URL, QR code)
- Pre-fill customer information when possible
- Include guarantee restatement
- Make checkboxes and writing areas large enough for elderly readers

### Business Reply Envelope (BRE)
- Always include a BRE for mail-in responses
- "No postage necessary" reduces friction
- Can print order form on BRE flap for dual function

## Testing Methodology

### What to Test (Priority Order)

1. **List/audience** — Different segments or list sources
2. **Offer** — Price, premium, guarantee, terms
3. **Format** — Letter vs postcard vs self-mailer
4. **Headline** — Benefit vs curiosity vs news
5. **Creative** — Design, imagery, layout

### A/B Test Design
- Test ONE variable per test
- Minimum 5,000 pieces per cell for statistical significance
- Use A/B split from same list for fair comparison
- Track results by unique codes per test cell
- Allow 3-4 weeks for response window before judging results

### Key Metrics

| Metric | Formula | Benchmark |
|---|---|---|
| Response rate | Responses ÷ pieces mailed | 0.5-5% |
| Cost per response | Total cost ÷ responses | Varies by industry |
| Cost per order | Total cost ÷ orders | Target < customer LTV |
| Average order value | Revenue ÷ orders | — |
| ROI | (Revenue - cost) ÷ cost | Positive = viable |

### Roll-Out Decision Framework
- Winning test cell beats control by 15%+ with statistical significance
- If winning by 5-15%, test again with larger sample
- If no clear winner, test something else — the variable doesn't matter enough
- Roll out winning approach to full list within 90 days (lists decay)
