# Common Resume Mistakes

Detailed reference content for the Resume Cover Letter Writing skill.

## Common Resume Mistakes


### Content Mistakes

1. **Duties instead of achievements:** Listing job responsibilities rather than accomplishments.

2. **Missing metrics:** Failing to quantify impact when possible.

3. **Irrelevant information:** Including outdated, off-topic, or personal information.

4. **Objective statements:** Using outdated objective statements instead of summaries.

5. **Unexplained gaps:** Leaving employment gaps without context.

6. **Inconsistent formatting:** Mixing formats, fonts, or styles.

7. **Excessive length:** Including unnecessary detail that dilutes impact.

8. **Underselling:** Being too modest about accomplishments.

9. **Not tailoring:** Sending generic resumes to every application.

10. **Weak verbs:** Using passive or weak language.


### Technical Mistakes

1. **Typos and grammatical errors:** Nothing kills credibility faster.

2. **Poor ATS formatting:** Graphics, tables, or formats that don't parse.

3. **Unprofessional email:** cute.bunny.123@email.com

4. **Missing contact information:** Making it hard to reach you.

5. **Wrong file format:** Not following application instructions.
