# Fundraising & Program Management

## Individual Donor Fundraising

### Donor Cultivation Cycle
1. **Identify** — Research potential donors, capacity assessment
2. **Cultivate** — Build relationship, share impact stories
3. **Solicit** — Make the ask (in person for major gifts)
4. **Steward** — Thank promptly, report on impact, retain

### Donor Pyramid
| Level | Gift Range | Strategy |
|-------|-----------|----------|
| Major donors | $10,000+ | Personal relationship, planned giving |
| Mid-level | $1,000-9,999 | Personalized communication, events |
| Annual fund | $100-999 | Direct mail, email, peer-to-peer |
| Small donors | Under $100 | Online campaigns, social media |

### Donor Retention
- Thank within 48 hours of gift
- Impact report within 90 days
- Personalized year-end summary
- Birthday/anniversary recognition
- Exclusive updates and access
- Target: 60%+ annual retention rate

## Grant Writing

### Grant Proposal Structure
1. **Cover letter** — Brief introduction and ask
2. **Executive summary** — Problem, solution, impact, ask
3. **Organizational background** — Mission, history, capacity
4. **Statement of need** — Data-supported problem description
5. **Program description** — Goals, activities, timeline
6. **Evaluation plan** — Metrics, data collection, reporting
7. **Budget** — Line-item budget with narrative justification
8. **Sustainability plan** — Funding beyond grant period
9. **Attachments** — 990, audit, board list, letters of support

### Grant Budget Best Practices
- Include reasonable indirect costs (10-15% typical)
- Match budget to program description
- Be specific about personnel (FTE, salary)
- Include in-kind contributions
- Show other funding sources

## Program Design & Evaluation

### Logic Model Framework
| Element | Question | Example |
|---------|----------|---------|
| Inputs | What resources? | Staff, funding, facilities |
| Activities | What do we do? | Workshops, counseling, referrals |
| Outputs | How much? | Number served, sessions delivered |
| Outcomes | What changed? | Skills gained, behavior change |
| Impact | Long-term change? | Community health improvement |

### Evaluation Methods
| Method | Use Case | Data Type |
|--------|----------|-----------|
| Pre/post surveys | Knowledge, attitude change | Quantitative |
| Participant interviews | Deep understanding of experience | Qualitative |
| Focus groups | Group perspectives, themes | Qualitative |
| Administrative data | Service utilization, completion | Quantitative |
| Case studies | Individual transformation stories | Mixed |

### Program Improvement Cycle
1. Collect data throughout program delivery
2. Analyze at defined intervals (quarterly minimum)
3. Share findings with staff and stakeholders
4. Identify improvement opportunities
5. Implement changes
6. Measure impact of changes
