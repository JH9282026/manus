# The Trade Desk Reporting & Analytics

API endpoints for performance data and cross-channel reporting.

## Performance Data

The Trade Desk provides comprehensive reporting across all channels.

### Report Types

- Campaign performance
- Ad group performance
- Creative performance
- Cross-channel attribution
- Conversion tracking
- Audience insights

### Key Metrics

- **Impressions**: Ad views
- **Clicks**: User clicks
- **CTR**: Click-through rate
- **Spend**: Total spend
- **CPM**: Cost per thousand impressions
- **CPC**: Cost per click
- **Conversions**: Conversion events
- **CPA**: Cost per acquisition
- **ROAS**: Return on ad spend
- **Viewability**: Viewable impression rate
- **Video Completion Rate**: VCR for video ads
