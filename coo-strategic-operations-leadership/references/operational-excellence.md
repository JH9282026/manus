# Operational Excellence Frameworks

Methodologies for systematic operational improvement.

---

## Lean Methodology

### 8 Wastes (DOWNTIME)
- **D**efects: Errors requiring rework
- **O**verproduction: Making more than needed
- **W**aiting: Idle time between steps
- **N**on-utilized talent: Underused skills
- **T**ransportation: Unnecessary movement of materials
- **I**nventory: Excess stock
- **M**otion: Unnecessary movement of people
- **E**xtra processing: Doing more than required

### Key Tools
- Value stream mapping
- 5S (Sort, Set, Shine, Standardize, Sustain)
- Kanban boards
- Poka-yoke (error-proofing)
- Kaizen events

## Six Sigma

### DMAIC Cycle
1. **Define**: Problem statement, scope, goals
2. **Measure**: Current state data collection
3. **Analyze**: Root cause identification
4. **Improve**: Solution development and testing
5. **Control**: Sustaining improvements

## Theory of Constraints (TOC)

### Five Focusing Steps
1. **Identify**: Find the system constraint
2. **Exploit**: Maximize constraint output
3. **Subordinate**: Align everything else
4. **Elevate**: Increase constraint capacity
5. **Repeat**: Find next constraint

### Application
- Manufacturing bottlenecks
- Service delivery constraints
- Resource limitations
- Policy constraints
