### User Freedom Levels

**High Control Elements:**
- Logo (position, size locked)
- Brand colors (limited palette)
- Footer format (consistent)
- Margins (standardized)

**Moderate Control:**
- Layout selection (from approved options)
- Image placement (within guidelines)
- Font sizes (within range)
- Content arrangement

**User Freedom:**
- Content creation
- Layout selection
- Image choices (within style guides)
- Emphasis decisions
