### PowerPoint Template (.potx) Creation

**Creating .potx Files:**
1. Design complete presentation with master slides
2. Remove all sample content slides
3. Save As → PowerPoint Template (.potx)
4. Place in Templates folder for easy access

**PowerPoint-Specific Features:**
- Theme colors and fonts
- Custom slide layouts
- Animation and transition defaults
- Custom Quick Access Toolbar setups

**Distribution:**
- Personal templates folder
- Network template location
- SharePoint library
- Custom templates in New dialog
