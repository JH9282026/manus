### Slide Master vs. Layout Masters

**Slide Master (Parent Master):**
- Controls elements appearing on ALL layouts
- Sets default fonts, colors, and background
- Contains company logo, footer information
- Establishes base formatting rules
- Changes cascade to all child layouts

**Layout Masters (Child Layouts):**
- Define specific slide arrangements
- Inherit properties from parent slide master
- Can override parent settings locally
- Contain unique placeholder configurations
- Serve specific content purposes
