### Color Schemes

**Building Brand Color Palettes:**

1. **Primary Colors (2-3):**
   - Main brand colors
   - Used for key elements
   - Headers, accents, CTAs

2. **Secondary Colors (2-4):**
   - Supporting palette
   - Charts, icons, backgrounds
   - Complementary to primary

3. **Neutral Colors (3-5):**
   - Text colors (dark gray, not pure black)
   - Background variants
   - Subtle dividers and borders

4. **Accent Colors (1-2):**
   - Highlight important elements
   - Call attention to key data
   - Use sparingly for impact

**Color Application Rules:**
- Define which colors for which purposes
- Establish color hierarchy
- Specify acceptable combinations
- Document contrast requirements
