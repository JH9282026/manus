### Keynote Theme (.kth) Creation

**Creating Keynote Themes:**
1. Design presentation with master slides
2. File → Save Theme
3. Choose location (Theme Chooser or custom)
4. Set as default if desired

**Keynote-Specific Features:**
- Master slide layouts
- Object styles
- Character and paragraph styles
- Background images and fills

**Distribution:**
- Theme Chooser (Add to Theme Chooser)
- iCloud sharing
- Direct file sharing (.kth files)
- Team folder distribution
