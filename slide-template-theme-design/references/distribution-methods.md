### Distribution Methods

**File Sharing:**
- Email attachment (small templates)
- Cloud storage links (Google Drive, OneDrive, Dropbox)
- Intranet downloads
- Version-controlled repositories

**Template Libraries:**
- PowerPoint: File → New → Custom templates
- SharePoint template libraries
- Google Workspace template galleries
- Keynote template chooser

**Enterprise Solutions:**
- Template management platforms
- Digital asset management (DAM) systems
- Brand management portals
- Custom distribution tools
