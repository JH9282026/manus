### Logo Placement

**Strategic Logo Positioning:**

| Position | Use Case | Considerations |
|----------|----------|----------------|
| Top-left corner | Traditional, formal | Most common placement |
| Top-right corner | Modern, clean | Less conventional |
| Bottom-left corner | Subtle presence | Doesn't compete with content |
| Bottom-right corner | Minimal branding | Paired with page numbers |
| Center (title slides only) | High impact opening | Only for opening/closing |

**Logo Sizing Guidelines:**
- Small enough not to dominate
- Large enough to be recognizable
- Consistent size across all slides
- Scale proportionally (never distort)
- Minimum clear space maintained

**Logo Variations:**
- Full color for white/light backgrounds
- Reversed (white) for dark backgrounds
- Single color for busy backgrounds
- Icon-only for very small applications
