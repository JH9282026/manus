### How Inheritance Works

Layout masters inherit from the slide master parent:

1. **Background**: Layouts use parent background unless overridden
2. **Fonts**: Text styles cascade from parent definitions
3. **Colors**: Theme colors apply from parent settings
4. **Logos/Elements**: Objects on parent appear on all layouts
5. **Footers**: Date, slide number, and footer text inherit

**Override Capability:**
- Layouts can hide parent elements
- Local formatting overrides inherited styles
- Specific layouts can use different backgrounds
- Placeholders can have custom formatting
