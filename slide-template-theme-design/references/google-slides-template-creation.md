### Google Slides Template Creation

**Creating Google Slides Templates:**
1. Design presentation in Google Slides
2. Edit master slides (Slide → Edit master)
3. Create all necessary layouts
4. Share via Google Workspace template gallery

**Google Slides-Specific:**
- Theme builder
- Custom layouts in master editor
- Google Fonts integration
- Collaborative editing capability

**Distribution Options:**
- Google Workspace template gallery (organization)
- Shared Drive templates
- Link sharing (make a copy)
- Slides API for programmatic access

**Submitting to Organization Gallery:**
1. Complete template design
2. Contact Workspace administrator
3. Submit for template gallery approval
4. Template appears in organization templates
