### Color Theme Best Practices

1. **Map brand colors strategically**:
   - Primary brand color → Accent 1
   - Secondary colors → Accents 2-4
   - Neutrals → Text/Background colors

2. **Ensure sufficient contrast**:
   - Dark text on light backgrounds
   - Light text on dark backgrounds
   - Minimum 4.5:1 contrast ratio

3. **Test in context**:
   - Charts and graphs
   - Tables
   - SmartArt diagrams
   - Various layouts

4. **Document color usage**:
   - Which accent for what purpose
   - Approved color combinations
   - Accessibility considerations
