### Image-Focused Layouts

**Full-Bleed Image:**
- Edge-to-edge image coverage
- Title overlay with contrast treatment
- Text legibility solutions (gradients, boxes)
- High-impact visual moments

**Image Gallery:**
- Multiple image placeholders
- Consistent sizing grid
- Captions or labels
- Clean arrangement

**Image with Caption:**
- Primary image area
- Caption/description zone
- Attribution space
- Consistent padding
