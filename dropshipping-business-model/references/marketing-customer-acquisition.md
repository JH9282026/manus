## Facebook and Instagram Advertising

### Campaign Structure Framework

**Three-Tier Campaign Structure**:

```
Campaign Level: Objective (Conversions - Purchase)
├── Ad Set 1: Audience Segment A (Interest targeting)
│   ├── Ad 1: Image creative variant A
│   ├── Ad 2: Image creative variant B
│   └── Ad 3: Video creative
├── Ad Set 2: Audience Segment B (Lookalike 1%)
│   ├── Ad 1: Image creative variant A
│   └── Ad 2: Video creative
└── Ad Set 3: Retargeting (Website visitors)
    ├── Ad 1: Carousel ad
    └── Ad 2: Dynamic product ad
```

### Testing Phase Strategy (Budget: $500-$1,000)

**Week 1-2: Product Testing**

**Setup**:
- Test 3-5 products simultaneously
- Budget: $100-$200 per product
- Duration: 3-7 days per product
- Objective: Conversions (Purchase)

**Ad Set Configuration**:
```
Audience:
- Location: United States (or target country)
- Age: 25-55 (adjust based on product)
- Gender: All (or specific if product-specific)
- Interests: 3-5 relevant interests (not too narrow)
- Audience size: 1-5 million

Placements:
- Automatic placements (let Facebook optimize)
- Or manual: Facebook Feed, Instagram Feed, Stories

Budget:
- $20-$30/day per ad set
- Start low, scale winners

Optimization:
- Conversion event: Purchase
- Bid strategy: Lowest cost
```

**Ad Creative Testing**:

Test 3 creative variations per product:
1. **Image ad** — High-quality product photo with benefit overlay
2. **Video ad** — 15-30 second product demo or unboxing
3. **Carousel ad** — 3-5 images showing features/benefits

**Kill Criteria** (after 3-7 days):
- No sales after spending 3x product price
- Cost per click (CPC) > $2
- Click-through rate (CTR) < 1%
- Add-to-cart rate < 5%

**Scale Criteria**:
- 3+ sales with ROAS > 2x
- CTR > 2%
- Add-to-cart rate > 10%
- Positive customer feedback

### Scaling Phase Strategy (Budget: $1,000-$5,000+)

**Horizontal Scaling** (Expand reach):

1. **Duplicate winning ad sets**:
   - Create 3-5 copies of winning ad set
   - Test different audiences (interests, lookalikes)
   - Keep budget same initially ($20-$30/day each)

2. **Lookalike audiences**:
   - Create from purchasers (1%, 2%, 3%, 5%)
   - Create from add-to-carts
   - Create from page engagers
   - Test each as separate ad set

3. **Interest expansion**:
   - Research related interests using Facebook Audience Insights
   - Test broader interest categories
   - Combine multiple interests

**Vertical Scaling** (Increase budget):

```
Day 1: $30/day → 3 sales (ROAS 3x)
Day 2: $40/day (+33%)
Day 3: $50/day (+25%)
Day 4: $65/day (+30%)
Day 5: $85/day (+30%)

Rules:
- Increase 20-30% every 2-3 days if ROAS maintained
- Never double budget in one day (causes algorithm reset)
- Monitor ROAS closely; pause if drops below 2x
- Don't touch winning ads (no edits reset learning)
```

**Campaign Budget Optimization (CBO)**:

```
Campaign Level:
- Enable CBO
- Set total daily budget ($100-$500)
- Let Facebook distribute budget to best-performing ad sets

Ad Set Level:
- Set minimum spend ($10/day) to ensure testing
- No maximum (let Facebook optimize)
```

### Ad Creative Best Practices

**Image Ads**:

**Specifications**:
- Size: 1080x1080px (square) or 1200x628px (landscape)
- Format: JPG or PNG
- Text overlay: Under 20% of image (use Facebook Text Overlay Tool)
- File size: Under 30MB

**Design Elements**:
- High-quality product image (white background or lifestyle)
- Benefit-focused headline overlay
- Contrasting colors to stand out in feed
- Before/after comparisons (if applicable)
- Urgency elements ("Limited Stock", "50% Off Today")

**Example Structure**:
```
Image: Product in use (lifestyle shot)
Text Overlay: "Say Goodbye to Back Pain"
Background: Subtle gradient or solid color
Branding: Small logo in corner
```

**Video Ads**:

**Specifications**:
- Length: 15-30 seconds (shorter is better)
- Format: MP4 or MOV
- Aspect ratio: 1:1 (square) or 4:5 (vertical)
- Resolution: 1080x1080px minimum
- Captions: Required (85% watch without sound)

**Video Structure** (30-second example):
```
0-3 sec: Hook (problem or eye-catching visual)
3-10 sec: Product introduction and key benefit
10-20 sec: Features demonstration
20-25 sec: Social proof (reviews, testimonials)
25-30 sec: Call-to-action ("Shop Now", "Limited Time")
```

**Video Creation Tools**:
- **InVideo** — Templates, stock footage, easy editing
- **Canva Video** — Simple animations and text overlays
- **Animoto** — Slideshow-style videos from images
- **Fiverr** — Hire video editors ($20-$100)

**Carousel Ads**:

**Best Use Cases**:
- Showcasing multiple product features
- Before/after transformations
- Product variations (colors, sizes)
- Step-by-step how-to-use
- Customer testimonials

**Structure** (5 cards example):
```
Card 1: Problem statement + product image
Card 2: Feature 1 with benefit
Card 3: Feature 2 with benefit
Card 4: Social proof (review or testimonial)
Card 5: Call-to-action ("Shop Now" + discount)
```

### Audience Targeting Strategies

**Interest Targeting** (Cold audiences):

**Research Process**:
1. Open Facebook Ads Manager → Audiences → Create Audience
2. Enter broad interest related to product
3. Review "Suggestions" for related interests
4. Use Facebook Audience Insights for demographics

**Targeting Framework**:
```
Product: Fitness resistance bands

Audience 1: Fitness Enthusiasts
- Interests: Fitness and wellness, Gym, Physical fitness
- Age: 25-45
- Gender: All

Audience 2: Home Workout
- Interests: Home fitness, Workout videos, Yoga
- Age: 30-55
- Gender: All

Audience 3: Competitor Targeting
- Interests: [Competitor brand 1], [Competitor brand 2]
- Age: 25-50
- Gender: All
```

**Lookalike Audiences** (Warm audiences):

**Source Audiences** (need 100+ people minimum):
1. **Purchasers** — Highest quality (1%, 2%, 3%)
2. **Add-to-Carts** — High intent (1%, 3%)
3. **Website Visitors** — Engaged traffic (3%, 5%)
4. **Page Engagers** — Social proof (5%)

**Creation Process**:
```
1. Audiences → Create Audience → Lookalike Audience
2. Source: Select custom audience (e.g., "Purchasers - Last 30 Days")
3. Location: United States (or target country)
4. Audience size: 1% (most similar, start here)
5. Create audience
```

**Testing Strategy**:
- Start with 1% lookalike (most similar)
- If successful, test 2%, 3%, 5% (broader reach)
- Create separate ad sets for each percentage
- Compare performance and scale winners

**Retargeting Audiences** (Hot audiences):

**Audience Segments**:

| Audience | Timeframe | Ad Strategy |
|----------|-----------|-------------|
| **Viewed product, no add-to-cart** | Last 7 days | Highlight benefits, reviews |
| **Added to cart, no purchase** | Last 3 days | Urgency, discount code |
| **Initiated checkout, no purchase** | Last 1 day | Strong urgency, limited-time offer |
| **Website visitors** | Last 14 days | Introduce best sellers |
| **Past purchasers** | Last 60 days | Upsell, cross-sell, new products |

**Retargeting Ad Examples**:

```
Abandoned Cart Ad:
Headline: "You Left Something Behind! 🛒"
Text: "Complete your order now and get 15% OFF with code SAVE15. 
Offer expires in 24 hours! ⏰"
CTA: Shop Now

Product Viewer Ad:
Headline: "Still Thinking About It?"
Text: "Join 10,000+ happy customers who love their [Product Name]. 
⭐⭐⭐⭐⭐ 4.8/5 stars | 30-Day Money-Back Guarantee"
CTA: Learn More
```

## TikTok Advertising

### Why TikTok for Dropshipping

**Advantages**:
- Lower CPMs than Facebook ($3-$8 vs. $10-$20)
- High engagement rates (younger demographics)
- Viral potential (organic reach)
- Less ad fatigue (newer platform)
- Native video format (authentic feel)

**Best Product Types**:
- Visually interesting or unique
- Solves relatable problem
- Demonstrates clear before/after
- Appeals to Gen Z/Millennials (18-35)
- Under $50 price point

### TikTok Ad Campaign Setup

**Campaign Structure**:
```
Campaign: Conversions
├── Ad Group 1: Broad targeting (18-35, interests)
│   ├── Video Ad 1: UGC-style demo
│   └── Video Ad 2: Trending sound + product
└── Ad Group 2: Retargeting (website visitors)
    └── Video Ad 3: Testimonial compilation
```

**Ad Group Settings**:
```
Placement: TikTok only (exclude other apps initially)
Targeting:
- Location: United States
- Age: 18-35 (adjust based on product)
- Gender: All or specific
- Interests: Select 3-5 relevant categories
- Behavior: Not required initially

Budget: $20-$50/day
Optimization: Complete Payment
Bid: Lowest cost
```

### TikTok Video Ad Best Practices

**Video Specifications**:
- Length: 9-15 seconds (shorter is better)
- Format: MP4 or MOV
- Aspect ratio: 9:16 (vertical, full screen)
- Resolution: 1080x1920px
- File size: Under 500MB
- Captions: Required (many watch muted)

**Content Formula**:
```
0-2 sec: Hook (pattern interrupt, bold statement)
2-8 sec: Problem + Solution (product demo)
8-12 sec: Social proof or key benefit
12-15 sec: CTA ("Link in bio", "Shop now")
```

**Hook Examples**:
- "I can't believe this actually works! 😱"
- "If you struggle with [problem], you NEED this"
- "This $20 gadget changed my life"
- "POV: You discover the best [product type]"
- "Wait for it... 🤯"

**Video Styles That Work**:

1. **UGC (User-Generated Content) Style**:
   - Authentic, not overly polished
   - Person talking directly to camera
   - Casual setting (home, car, etc.)
   - Relatable language and tone

2. **Before/After Transformation**:
   - Show problem clearly
   - Demonstrate product use
   - Reveal impressive result
   - Add text overlays for emphasis

3. **Trending Sound + Product**:
   - Use popular TikTok sound
   - Integrate product naturally
   - Follow trend format
   - Add unique twist

4. **"Things I Bought That Are Worth It"**:
   - Compilation format
   - Feature product as #1 or finale
   - Quick cuts, high energy
   - Text overlays with benefits

**Video Creation**:
- **Shoot yourself** — Most authentic, use smartphone
- **Hire UGC creators** — Fiverr, Billo, Trend ($50-$200/video)
- **Repurpose customer videos** — Ask permission, offer discount
- **Use stock footage** — InVideo, Canva (less effective)

## Google Shopping Ads

### Why Google Shopping for Dropshipping

**Advantages**:
- High purchase intent (people actively searching)
- Visual product display in search results
- Lower competition than Facebook (in some niches)
- Excellent for branded or specific product searches
- Complements Facebook ads (different stage of funnel)

**Best Product Types**:
- Specific, searchable items ("wireless earbuds", "yoga mat")
- Products with clear search demand
- Competitive pricing vs. Amazon/competitors
- Unique or hard-to-find items

### Google Merchant Center Setup

**Step 1: Create Merchant Center Account**

1. Visit merchants.google.com
2. Sign in with Google account
3. Enter business information:
   - Business name
   - Country
   - Time zone
4. Verify and claim website URL

**Step 2: Website Verification**

**Shopify**:
1. Merchant Center → Settings → Website
2. Copy verification code
3. Shopify → Online Store → Themes → Actions → Edit Code
4. Paste code in `theme.liquid` before `</head>`
5. Return to Merchant Center → Verify URL

**WooCommerce**:
1. Install "Google Listings & Ads" plugin
2. Connect Google account
3. Automatic verification

**Step 3: Product Feed Setup**

**Shopify** (using Google Channel app):
1. Apps → Search "Google Channel" → Install
2. Connect Google Merchant Center account
3. Select products to sync
4. Map product categories to Google taxonomy
5. Sync products (automatic updates)

**WooCommerce** (using plugin):
1. Install "Product Feed Manager for WooCommerce"
2. Create new feed → Google Shopping
3. Map WooCommerce fields to Google attributes
4. Generate feed URL
5. Submit feed URL to Merchant Center

**Required Product Attributes**:
- Title (150 characters max)
- Description (5,000 characters max)
- Image URL (high quality, white background preferred)
- Price
- Availability (in stock, out of stock)
- Brand
- GTIN (if available) or MPN
- Product category (Google taxonomy)
- Condition (new, used, refurbished)

### Google Ads Campaign Setup

**Campaign Structure**:
```
Campaign: Shopping - Standard
├── Ad Group 1: All Products
│   └── Product Group: All products
├── Ad Group 2: Best Sellers
│   └── Product Group: Custom label "bestseller"
└── Ad Group 3: High Margin
    └── Product Group: Custom label "high-margin"
```

**Campaign Settings**:
```
Campaign type: Shopping
Sales country: United States
Merchant Center account: [Your account]
Campaign priority: Low (for testing)
Bidding: Maximize clicks (testing) or Target ROAS (scaling)
Daily budget: $20-$50 (testing)
Networks: Search Network only (exclude partners initially)
```

**Bidding Strategy**:

**Testing Phase**:
- Strategy: Maximize Clicks
- Budget: $20-$50/day
- Duration: 7-14 days
- Goal: Gather data on which products get clicks/sales

**Scaling Phase**:
- Strategy: Target ROAS
- Target: 300-400% (3-4x return)
- Budget: $50-$200+/day
- Adjust target based on performance

### Optimization Strategies

**Product Title Optimization**:
```
Formula: Brand + Product Type + Key Attributes + Size/Color

Bad: "Wireless Earbuds"
Good: "Sony Wireless Earbuds Bluetooth 5.0 Noise Cancelling Black"

Include:
- Brand (if known/trusted)
- Product type (exact match to searches)
- Key features (Bluetooth, wireless, etc.)
- Specifications (size, color, material)
- Benefits (noise cancelling, waterproof)
```

**Negative Keywords**:

Add negative keywords to exclude irrelevant searches:
```
Common negatives:
- "free"
- "cheap"
- "wholesale"
- "bulk"
- "used"
- "repair"
- "DIY"
- "how to make"
- Competitor brand names (if not selling)
```

**Product Segmentation**:

Create product groups for better bid control:
```
1. By Performance:
   - High ROAS products (increase bids 20-50%)
   - Low ROAS products (decrease bids or pause)

2. By Margin:
   - High margin (bid more aggressively)
   - Low margin (bid conservatively)

3. By Category:
   - Category A (e.g., electronics)
   - Category B (e.g., home goods)
```

## Influencer Marketing

### Finding Influencers

**Influencer Tiers**:

| Tier | Followers | Engagement | Cost | Best For |
|------|-----------|------------|------|----------|
| **Nano** | 1K-10K | 5-10% | Free product | Testing, niche audiences |
| **Micro** | 10K-100K | 3-7% | $50-$500 | Best ROI, engaged communities |
| **Mid** | 100K-500K | 2-5% | $500-$5K | Scaling, brand awareness |
| **Macro** | 500K-1M | 1-3% | $5K-$20K | Mass reach, established brands |
| **Mega** | 1M+ | 0.5-2% | $20K+ | Major campaigns only |

**Recommendation**: Start with micro-influencers (10K-100K) for best ROI.

**Finding Influencers**:

1. **Instagram Search**:
   - Search niche hashtags (#fitness, #homegadgets)
   - Filter by "Top" posts
   - Check poster's follower count and engagement
   - Review their content quality and audience fit

2. **TikTok Search**:
   - Search product-related keywords
   - Look for creators with 10K-100K followers
   - Check video views and comment engagement
   - Review their content style and authenticity

3. **Influencer Platforms**:
   - **Upfluence** — Database search, campaign management
   - **AspireIQ** — Influencer marketplace
   - **Grin** — E-commerce focused
   - **Shopify Collabs** — Built-in for Shopify stores

4. **Manual Outreach**:
   - Google: "[niche] influencers"
   - Check competitor's tagged posts
   - Review product review channels on YouTube

### Vetting Influencers

**Engagement Rate Calculation**:
```
Formula: (Likes + Comments) ÷ Followers × 100

Example:
Followers: 50,000
Average likes: 2,000
Average comments: 100
Engagement: (2,000 + 100) ÷ 50,000 × 100 = 4.2%

Benchmarks:
- Excellent: 5%+
- Good: 3-5%
- Average: 1-3%
- Poor: <1% (likely fake followers)
```

**Red Flags**:
- Sudden follower spikes (check Social Blade)
- Low engagement relative to followers
- Generic comments ("Nice!", "Great!")
- Followers from irrelevant countries
- Inconsistent content quality
- Mostly giveaway/sponsored posts

**Audience Analysis**:
- Check follower demographics (age, location, interests)
- Review comment quality (real conversations vs. spam)
- Assess audience alignment with your target market
- Look for previous brand partnerships (relevant niches)

### Outreach and Negotiation

**Outreach Message Template**:

```
Subject: Collaboration Opportunity with [Your Brand]

Hi [Influencer Name],

I'm [Your Name] from [Your Brand], and I love your content on [specific topic/niche]! Your recent post about [specific post] really resonated with our brand values.

We'd love to collaborate with you to promote our [product type]. We think your audience would genuinely benefit from [key product benefit].

We're offering:
✅ Free product (valued at $[X])
✅ [X]% commission on sales through your unique code
✅ [Optional: Flat fee of $[X]]

Would you be interested in discussing a partnership? I'd love to hear your rates and ideas!

Looking forward to connecting!

Best,
[Your Name]
[Your Brand]
[Website]
```

**Compensation Models**:

1. **Free Product Only**:
   - Best for: Nano influencers, product testing
   - Offer: Send free product for honest review
   - No guaranteed post, but often get organic mention

2. **Affiliate Commission**:
   - Best for: Performance-based partnerships
   - Offer: 10-20% commission on sales via unique code
   - Use Shopify discount codes or affiliate apps
   - Low risk, pay only for results

3. **Flat Fee**:
   - Best for: Guaranteed posts, specific deliverables
   - Offer: $50-$500 for micro-influencers
   - Negotiate based on followers and engagement
   - Get usage rights for repurposing content

4. **Hybrid** (Recommended):
   - Offer: Flat fee + commission
   - Example: $200 + 15% commission
   - Motivates both posting and driving sales

**Negotiation Tips**:
- Start with lower offer, leave room to negotiate
- Emphasize long-term partnership potential
- Offer performance bonuses (extra $100 if 50+ sales)
- Request content usage rights for ads
- Ask for Instagram Story + Feed post for better reach

### Campaign Execution

**Deliverables to Request**:
```
Instagram:
- 1 feed post (permanent)
- 2-3 Stories (24 hours)
- Swipe-up link (if available) or link in bio
- Unique discount code mention

TikTok:
- 1 video (15-30 seconds)
- Product link in bio
- Discount code in caption

YouTube:
- Dedicated video or product mention in relevant video
- Affiliate link in description
- Discount code mention
```

**Content Guidelines**:
- Authentic, not overly scripted
- Show product in use (real-life scenario)
- Highlight 2-3 key benefits
- Include clear call-to-action
- Mention discount code prominently
- FTC disclosure (#ad, #sponsored)

**Tracking Performance**:
```
Metrics to monitor:
- Unique discount code uses
- Traffic from influencer's link (UTM parameters)
- Sales attributed to influencer
- Engagement on influencer's post (likes, comments, shares)
- ROI: (Revenue - Cost) ÷ Cost × 100

Tools:
- Shopify discount code analytics
- Google Analytics (UTM tracking)
- Affiliate apps (Refersion, LeadDyno)
```

## Email Marketing

### Email List Building

**Popup Strategies**:

**Welcome Popup** (first-time visitors):
```
Headline: "Get 10% OFF Your First Order! 🎉"
Subtext: "Join 10,000+ happy customers and get exclusive deals."
CTA: "Get My Discount"
Trigger: 15-30 seconds after page load
Frequency: Once per visitor
```

**Exit-Intent Popup** (about to leave):
```
Headline: "Wait! Don't Leave Empty-Handed 😊"
Subtext: "Get 15% OFF if you sign up in the next 60 seconds!"
CTA: "Claim My Discount"
Trigger: Mouse moves toward browser close button
Frequency: Once per session
```

**Spin-to-Win Popup** (gamification):
```
Headline: "Spin to Win Your Discount! 🎰"
Prizes: 10% OFF, 15% OFF, 20% OFF, Free Shipping
CTA: "Spin Now"
Trigger: After 30 seconds or scroll 50%
Frequency: Once per visitor
```

**Tools**:
- **Klaviyo** — Best for e-commerce, advanced automation ($20-$150/mo)
- **Privy** — Popups and email ($15-$45/mo)
- **Omnisend** — Email + SMS ($16-$59/mo)
- **Mailchimp** — Free up to 500 subscribers

### Email Automation Sequences

**1. Welcome Series** (3-5 emails)

**Email 1** (Immediate):
```
Subject: "Your 10% OFF Code Inside! 🎁"

Content:
- Thank you for subscribing
- Discount code (large, prominent)
- Featured best-sellers (3-4 products)
- CTA: Shop Now
```

**Email 2** (Day 2):
```
Subject: "Here's Why 10,000+ Customers Love Us ⭐"

Content:
- Customer testimonials
- Product reviews (5-star)
- Trust badges (money-back guarantee, secure checkout)
- CTA: See Best Sellers
```

**Email 3** (Day 5):
```
Subject: "Your Discount Expires Tomorrow! ⏰"

Content:
- Urgency reminder (discount expiring)
- Product recommendations based on browsing
- Free shipping mention
- CTA: Shop Before It's Gone
```

**2. Abandoned Cart Series** (3 emails)

**Email 1** (1 hour after abandonment):
```
Subject: "You Left Something Behind! 🛒"

Content:
- Cart contents with images
- "Complete your order" CTA
- Trust badges
- No discount yet (test if they'll buy without incentive)
```

**Email 2** (24 hours):
```
Subject: "Still Interested? Here's 10% OFF 🎉"

Content:
- Cart reminder
- 10% discount code
- Customer reviews of abandoned products
- Urgency ("Limited stock")
- CTA: Complete Order Now
```

**Email 3** (48 hours):
```
Subject: "Last Chance: Your Cart Expires Soon! ⏰"

Content:
- Final reminder
- Increased discount (15% OFF)
- Strong urgency ("Cart expires in 24 hours")
- Social proof
- CTA: Claim My Discount
```

**3. Post-Purchase Series** (3-4 emails)

**Email 1** (Immediately after purchase):
```
Subject: "Order Confirmed! Here's What's Next 📦"

Content:
- Order confirmation and details
- Estimated delivery date
- Tracking information (when available)
- Customer support contact
```

**Email 2** (3 days after delivery):
```
Subject: "How's Your [Product Name]? ⭐"

Content:
- Request product review
- Incentive (10% OFF next order for review)
- Easy review link
- Product care tips
```

**Email 3** (14 days after delivery):
```
Subject: "You Might Also Love These... 💡"

Content:
- Cross-sell related products
- "Frequently bought together"
- Exclusive customer discount (15% OFF)
- CTA: Shop Recommendations
```

### Email Design Best Practices

**Structure**:
```
1. Preheader text (50-100 characters, complements subject)
2. Header with logo
3. Hero image or headline
4. Body content (concise, scannable)
5. Clear CTA button (contrasting color)
6. Social proof or trust badges
7. Footer (unsubscribe, contact, social links)
```

**Mobile Optimization**:
- Single column layout
- Large, tappable CTA buttons (44x44px minimum)
- Font size 14px+ for body text
- Compressed images (under 1MB total email size)
- Test on multiple devices before sending

**Subject Line Best Practices**:
```
Formulas that work:
- Urgency: "Last Chance: 50% OFF Ends Tonight! ⏰"
- Curiosity: "You Won't Believe What Just Arrived... 😍"
- Personalization: "[Name], Your Exclusive Offer Inside 🎁"
- Social proof: "Join 10,000+ Happy Customers ⭐"
- Question: "Still Looking for the Perfect [Product]? 🤔"

Tips:
- Keep under 50 characters (mobile truncation)
- Use emojis sparingly (1-2 max)
- A/B test different approaches
- Avoid spam triggers ("FREE!!!", all caps)
- Personalize when possible
```

**Sending Strategy**:
- **Best days**: Tuesday, Wednesday, Thursday
- **Best times**: 10 AM, 2 PM, 8 PM (test for your audience)
- **Frequency**: 2-4 emails per week (promotional)
- **Segmentation**: Send relevant content based on behavior

## Conversion Rate Optimization

### Landing Page Optimization

**Above-the-Fold Elements**:
```
1. Compelling headline (benefit-focused)
2. Subheadline (supporting detail)
3. Hero image/video (product in use)
4. Clear CTA button ("Add to Cart", "Shop Now")
5. Trust badges (secure checkout, guarantee)
6. Social proof ("10,000+ sold", "4.8/5 stars")
```

**Product Page Checklist**:
- [ ] High-quality images (5+ angles, zoom enabled)

### A/B Testing Framework

**Elements to Test** (in priority order):

1. **Product Price**:
   - Test: $39.99 vs. $44.99 vs. $49.99
   - Impact: High (directly affects conversions and profit)
   - Duration: 7-14 days, 100+ visitors per variant

2. **CTA Button**:
   - Test: Color (green vs. orange vs. red)
   - Test: Text ("Add to Cart" vs. "Buy Now" vs. "Get Yours")
   - Impact: Medium-High
   - Duration: 3-7 days, 500+ visitors

3. **Product Images**:
   - Test: Lifestyle vs. white background
   - Test: Image order and primary image
   - Impact: Medium
   - Duration: 7 days, 200+ visitors

4. **Headline**:
   - Test: Benefit-focused vs. feature-focused
   - Test: Question vs. statement
   - Impact: Medium
   - Duration: 7 days, 200+ visitors

5. **Social Proof**:
   - Test: Number of reviews vs. star rating vs. "X sold"
   - Test: Placement (above vs. below fold)
   - Impact: Medium
   - Duration: 7 days, 200+ visitors

**Tools**:
- **Google Optimize** — Free, integrates with Analytics
- **Optimizely** — Advanced, enterprise-level
- **VWO** — Visual editor, easy setup
- **Shopify A/B Testing Apps** — Theme Scientist, Neat A/B Testing

### Trust and Credibility Signals

**Essential Trust Elements**:

1. **Money-Back Guarantee**:
   ```
   "30-Day Money-Back Guarantee
   Not satisfied? Return for a full refund, no questions asked."
   
   Placement: Product page, checkout, footer
   ```

2. **Secure Checkout Badges**:
   - SSL certificate icon
   - Payment method logos (Visa, Mastercard, PayPal)
   - "Secure Checkout" or "256-bit Encryption"
   - Placement: Near CTA button, checkout page

3. **Customer Reviews**:
   - Minimum 10 reviews per product
   - Average 4+ stars
   - Include photos from customers
   - Respond to negative reviews professionally
   - Tools: Loox, Judge.me, Yotpo

4. **Social Proof**:
   - "10,000+ Happy Customers"
   - "Featured in [Media Outlet]"
   - "As Seen On TV" (if applicable)
   - Instagram feed with customer photos

5. **Clear Policies**:
   - Shipping policy (delivery times)
   - Return policy (easy to find and understand)
   - Privacy policy (GDPR compliant)
   - Contact information (email, phone, address)

### Urgency and Scarcity Tactics

**Ethical Urgency** (use truthfully):

1. **Limited Stock**:
   ```
   "Only 3 left in stock - Order soon!"
   
   Implementation: Show actual inventory levels
   Tool: Shopify apps (Stock Countdown, Hurrify)
   ```

2. **Time-Limited Discounts**:
   ```
   "50% OFF ends in: [Countdown Timer]"
   
   Implementation: Real sale with end date
   Tool: Countdown timer apps (Hurrify, Ultimate Sales Boost)
   ```

3. **Seasonal/Holiday Promotions**:
   ```
   "Black Friday Sale - 48 Hours Only!"
   
   Implementation: Actual limited-time event
   Frequency: Major holidays, quarterly sales
   ```

4. **Social Proof Notifications**:
   ```
   "John from New York just purchased this item!"
   
   Implementation: Real purchase notifications
   Tool: Proof, Fomo, Sales Pop
   Frequency: Don't overdo (can seem spammy)
   ```

**Warning**: Avoid fake scarcity ("Only 2 left!" when you have 1,000). It damages trust and can be illegal in some jurisdictions.
