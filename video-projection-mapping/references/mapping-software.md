# Mapping Software

Comprehensive guide to projection mapping software, features, workflows, and selection criteria.

## Software Comparison Matrix

### Beginner-Friendly Software

#### HeavyM

**Platform**: Windows, macOS  
**Price**: €29/month (Starter), €59/month (Pro), €99/month (Studio)  
**Website**: heavym.net

**Key Features**:
- Drag-and-drop interface (no technical expertise required)
- Integrated visual library (ready-to-use effects and animations)
- Music sync with beat-reactive effects (sound analysis)
- 3D Materials (shapes react to light, create depth)
- Edge-blending for multi-projector setups
- Real-time preview
- Export to video or live playback

**Best For**:
- Beginners and non-technical users
- Quick setup and live performances
- Art installations and events
- Users who want pre-made visual content

**Workflow**:
1. Import or create shapes matching projection surface
2. Drag visual effects from library onto shapes
3. Adjust parameters (color, speed, intensity)
4. Sync to music if desired
5. Output to projector in real-time or export video

**Limitations**:
- Less precise control than professional software
- Limited custom content creation (relies on library)
- Not ideal for complex, multi-projector productions

**Learning Curve**: Low (1-2 hours to basic proficiency)

---

#### MapMap

**Platform**: Windows, macOS, Linux  
**Price**: Free (open-source)  
**Website**: mapmapteam.github.io

**Key Features**:
- Free and open-source
- Basic mapping tools (quad warping, mesh warping)
- Video and image playback
- OSC (Open Sound Control) for external control
- Multi-layer support
- Cross-platform (including Linux)

**Best For**:
- Budget-conscious projects
- Learning projection mapping basics
- Simple installations
- Linux users

**Workflow**:
1. Import video or image content
2. Create mapping shapes (quads, meshes)
3. Warp shapes to match projection surface
4. Assign content to shapes
5. Playback and adjust

**Limitations**:
- Basic feature set (no advanced effects or blending)
- Less polished interface than commercial software
- Limited support and documentation
- Not suitable for professional productions

**Learning Curve**: Low-Medium (2-4 hours to basic proficiency)

---

### Professional Software

#### MadMapper

**Platform**: macOS, Windows  
**Price**: €599 (perpetual license)  
**Website**: madmapper.com

**Key Features**:
- 3D spatial scanner (camera-based auto-alignment)
- Bezier masking (precise, curved masks)
- Native timeline (animate parameters over time)
- Multi-output support (multiple projectors)
- DMX, MIDI, OSC control
- Syphon/Spout (share video between applications)
- LED mapping (control LED fixtures)
- Modular system (add-ons for specific features)

**Best For**:
- Professional mapping projects
- Precise control and complex surfaces
- Integration with lighting and audio systems
- Users who need timeline-based animation

**Workflow**:
1. Import 3D scan or manually create mapping mesh
2. Use Bezier tools to create precise masks
3. Import video content or use built-in generators
4. Animate parameters on timeline
5. Set up multi-output for multiple projectors
6. Control live via MIDI/OSC or play timeline

**Limitations**:
- macOS version more mature than Windows
- Steeper learning curve than beginner software
- One-time cost (no subscription, but updates may require upgrade fee)

**Learning Curve**: Medium (5-10 hours to proficiency)

---

#### Resolume Arena

**Platform**: Windows, macOS  
**Price**: €799 (perpetual license)  
**Website**: resolume.com

**Key Features**:
- VJ-style interface (designed for live performance)
- Audio-visual synchronization (BPM sync, audio analysis)
- Extensive effects library (real-time video effects)
- Multi-output support (up to 16 outputs)
- Advanced output mapping (slice, warp, blend)
- DMX, MIDI, OSC control
- Syphon/Spout support
- Plugin support (VST, ISF shaders)

**Best For**:
- Live events and VJ performances
- Music festivals and concerts
- Real-time video manipulation
- Users who need extensive effects and audio sync

**Workflow**:
1. Import video clips into clip slots
2. Set up output mapping (slices for each projector or surface)
3. Warp and blend outputs to match surfaces
4. Trigger clips and apply effects in real-time
5. Sync to audio (BPM, audio analysis)
6. Control via MIDI controller, keyboard, or OSC

**Limitations**:
- Interface can be overwhelming for beginners
- Designed for live performance (less ideal for static installations)
- Requires understanding of VJ workflow

**Learning Curve**: Medium-High (10-15 hours to proficiency)

#### TouchDesigner

**Platform**: Windows, macOS  
**Price**: Free (non-commercial), €600/year (commercial license)  
**Website**: derivative.ca

**Key Features**:
- Node-based visual programming
- Real-time generative graphics (procedural content)
- Interactive capabilities (sensors, cameras, data inputs)
- Python scripting for advanced control
- Multi-output and high-resolution support
- Integration with hardware (Kinect, Arduino, sensors)
- GLSL shader support
- Extensive operator library (3D, compositing, logic, data)

**Best For**:
- Generative and interactive installations
- Custom, unique content creation
- Complex interactive systems
- Users with programming or technical background

**Workflow**:
1. Build node network (operators connected by wires)
2. Create generative content (noise, fractals, 3D, particles)
3. Add interactivity (sensors, cameras, data)
4. Set up output mapping (geometry, warping)
5. Run in real-time, content generates continuously

**Limitations**:
- Steep learning curve (node-based paradigm)
- Requires technical mindset
- Free version has resolution limits (non-commercial use)
- Not ideal for simple, pre-rendered content playback

**Learning Curve**: High (20-40 hours to proficiency, ongoing learning)

**Price**: Custom (hardware + software bundles, rental options)  
**Website**: disguise.one

**Key Features**:

- Timecode and show control
- Redundancy and failover (critical for live events)
- Real-time rendering and playback
- Extensive support and training

**Best For**:
- Mega-events (Olympics, world tours, major festivals)
- Touring productions (concerts, theater)
- Broadcast and film production
- Projects requiring absolute reliability

**Workflow**:
1. Design and previz in cloud platform (Designer)
2. Simulate projection setup virtually
3. Export to Disguise hardware (media server)
4. Install projectors and calibrate
5. Run show with timecode or manual control
6. Monitor and adjust in real-time

**Limitations**:
- Extremely expensive (hardware + software)
- Requires training and expertise
- Overkill for small projects
- Proprietary ecosystem

**Learning Curve**: Very High (professional training recommended)

## Workflow Guides

### Basic Mapping Workflow (MadMapper)

**Step 1: Setup**
1. Connect projector to computer
2. Set computer display to extended (not mirrored)
3. Set projector as secondary display
4. Launch MadMapper

**Step 2: Create Surfaces**
1. Click "Add Surface" > Choose shape (Quad, Triangle, Mesh)
2. Position corners to roughly match projection surface
3. Use Bezier tools for curved edges if needed
4. Repeat for each surface or face of 3D object

**Step 3: Assign Content**
1. Import video files or use built-in generators
2. Drag content onto surface
3. Adjust content properties (scale, position, rotation)

**Step 4: Fine-Tune Alignment**
1. Project test pattern (grid or high-contrast image)
2. Adjust surface corners and edges to match physical surface
3. Use control points for precise warping
4. Enable "Edit Mode" to see both content and wireframe

**Step 5: Playback**
1. Exit edit mode
2. Content plays on mapped surfaces
3. Adjust brightness, opacity, effects as needed
4. Save project for future use

---

### Multi-Projector Blending (Resolume Arena)

**Step 1: Physical Setup**
1. Position projectors with 10-20% overlap
2. Connect all projectors to computer (multiple outputs)
3. Configure display settings (each projector as separate display)

**Step 2: Output Mapping**
1. In Resolume, open Advanced Output
2. Create "Slices" for each projector
3. Assign each slice to corresponding display output
4. Position slices in virtual canvas (overlap where projectors overlap)

**Step 3: Warping**
1. For each slice, enable "Transform" mode
2. Warp corners to match projection surface
3. Use grid overlay to verify alignment

**Step 4: Edge Blending**
1. Enable "Edge Blend" for overlapping slices
2. Adjust blend width (match physical overlap)
3. Adjust gamma curve for smooth blend
4. Test with full-white content to verify seamless blend

**Step 5: Color Matching**
1. Adjust brightness and contrast for each projector
2. Match white balance (color temperature)
3. Use color correction tools if projectors differ
4. Test with various content to ensure consistency

---

### Generative Content (TouchDesigner)

**Step 1: Create Base Geometry**
1. Add "Geometry" COMP (component)
2. Inside, add "Grid" SOP (surface operator) for base mesh
3. Add "Noise" TOP (texture operator) for displacement
4. Connect Noise to Grid via "Displace" SOP

**Step 2: Add Animation**
1. Add "LFO" CHOP (channel operator) for oscillation
2. Connect LFO to Noise parameters (offset, amplitude)
3. Adjust LFO speed and range for desired motion

**Step 3: Render and Output**
1. Add "Render" TOP to render 3D geometry
2. Add "Null" TOP as output
3. Add "Window" COMP for fullscreen output
4. Connect Null to Window, send to projector display

**Step 4: Add Interactivity (Optional)**
1. Add "Kinect" CHOP for motion tracking
2. Use Kinect data to control parameters (noise, color, speed)
3. Map sensor values to visual parameters
4. Test and refine interaction

**Step 5: Optimize Performance**
1. Reduce resolution if frame rate drops
2. Simplify geometry (fewer polygons)
3. Use efficient operators (avoid heavy effects)
4. Monitor performance panel, aim for 60 fps

## Software Selection Guide

### Choose HeavyM if:
- You're new to projection mapping
- You need quick setup for events or performances
- You want pre-made visual content
- Music sync is a priority
- Budget allows for subscription

### Choose MapMap if:
- Budget is very limited (free)
- You're learning projection mapping basics
- Project is simple (single projector, basic shapes)
- You use Linux

### Choose MadMapper if:
- You need precise, professional mapping
- You're working with complex surfaces
- Timeline-based animation is important
- You want one-time purchase (not subscription)
- You need integration with lighting (DMX)

### Choose Resolume Arena if:
- You're doing live VJ performances
- Audio-visual sync is critical
- You need extensive real-time effects
- Multi-output and blending are required
- You're comfortable with VJ-style workflow

### Choose TouchDesigner if:
- You want generative, unique content
- Interactivity is core to the project
- You have programming or technical skills
- You need custom, complex systems
- Budget allows for learning time investment

### Choose Smode if:
- You're working on large-scale productions
- Real-time 3D is required
- High resolution (4K+) is needed
- Budget allows for subscription
- You need broadcast-quality output

### Choose Disguise if:
- You're working on mega-events or tours
- Reliability and redundancy are critical
- Budget is substantial (enterprise-level)
- You need cloud-based pre-visualization
- Professional support and training are required

## Tips for Learning Mapping Software

1. **Start Simple**: Begin with basic shapes and surfaces before tackling complex 3D objects
2. **Use Tutorials**: Most software has official tutorials and community resources
3. **Practice Calibration**: Spend time mastering alignment and warping tools
4. **Experiment**: Try different effects, content, and techniques
5. **Join Communities**: Forums, Discord servers, Facebook groups for support and inspiration
6. **Attend Workshops**: Hands-on learning accelerates proficiency
7. **Build Portfolio**: Document projects to showcase skills and progress

## Common Software Issues and Solutions

| Issue | Cause | Solution |
|-------|-------|----------|
| Software won't detect projector | Display settings, cable | Check display settings (extended, not mirrored), try different cable/port |
| Low frame rate | Underpowered computer, high-res content | Upgrade hardware, reduce content resolution, optimize effects |
| Content doesn't fill surface | Aspect ratio mismatch | Adjust content aspect ratio or surface shape |
| Alignment drifts over time | Projector heat, vibration | Allow projector warm-up, secure mounting, recalibrate |
| Audio sync lag | Buffer settings, processing delay | Adjust audio buffer, reduce effects, use lower latency mode |
| Crashes or freezes | Insufficient RAM, software bug | Close other applications, update software, check system requirements |
