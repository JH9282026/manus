# Outbrain Campaign Setup Guide

Guide for creating Outbrain campaigns.

## Campaign Hierarchy

1. **Marketer** - Customer account
2. **Budget** - Allocated funds
3. **Campaign** - Collection of promoted links
4. **PromotedLink** - Individual content

## Creating a Campaign

### Step 1: Set Campaign Details
- Campaign name
- Campaign objective (traffic, conversions, engagement)
- Start and end dates

### Step 2: Configure Budget
- Daily budget
- Total budget (optional)
- Budget pacing

### Step 3: Select Targeting
- Geographic targeting (country-level)
- Device targeting (desktop, mobile, tablet)
- Publisher selection

### Step 4: Set Bidding
- Choose bidding strategy
- Set CPC or CPA target
- Configure bid adjustments

### Step 5: Add Promoted Links
- Content URL
- Headline (max 100 characters)
- Thumbnail image
- Call-to-action

## Best Practices

- Start with broad targeting
- Test multiple headlines per content
- Monitor performance daily
- Optimize based on engagement metrics
- Use high-quality images
- Write compelling, curiosity-driven headlines
