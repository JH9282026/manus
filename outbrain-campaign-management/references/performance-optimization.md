# Outbrain Performance Optimization

Optimization strategies for Outbrain campaigns.

## Bidding Strategies

### Conversion Bid Strategy
- Optimize for conversions
- Set target CPA
- Requires conversion tracking

### Engagement Bid Strategy
- Optimize for engagement
- Good for content marketing
- Focuses on time on site, pages viewed

### Target CPA
- Automated bidding
- Set specific CPA goal
- Requires conversion data

### Target ROAS
- Return on ad spend optimization
- Best for e-commerce
- Requires revenue tracking

### Max Conversions
- Maximize conversion volume
- Uses full budget
- Best for high-value customers

## Optimization Process

**Week 1-2: Launch**
- Monitor approval status
- Check initial CTR
- Identify top-performing content

**Week 3-4: Optimization**
- Analyze engagement metrics
- Adjust bids based on performance
- Refine targeting
- Pause low performers

**Week 5+: Scaling**
- Increase budgets on winners
- Expand to similar content
- Test new publishers
- Implement automated bidding

## Key Metrics

- **CTR**: Click-through rate
- **Engagement Rate**: Time on site, pages viewed
- **Conversion Rate**: Conversions / clicks
- **CPA**: Cost per acquisition
- **ROAS**: Return on ad spend

## Optimization Tips

- Test multiple headlines per content piece
- Refresh creatives regularly
- Monitor publisher performance
- Use bid adjustments strategically
- Focus on engagement quality, not just clicks
- Optimize landing pages continuously
