# Game Asset Creation

Create optimized 3D assets for game engines and real-time rendering.

## Overview

Game asset creation involves modeling, texturing, and optimizing 3D assets for real-time performance while maintaining visual quality for games and interactive applications.

## Quick Start

| Asset Type | Workflow | Reference |
|------------|----------|-----------|
| Props | Model, UV, bake, texture | `/references/prop-modeling.md` |
| Environment | Modular, tileable | `/references/environment-assets.md` |
| Materials | PBR workflow | `/references/pbr-materials.md` |
| Integration | Engine setup | `/references/engine-integration.md` |

## Core Concepts

### Modeling for Games
- Polygon budget management
- Clean topology
- Modular design
- LOD creation

### UV Mapping
- Efficient UV layout
- Texture atlasing
- Trim sheets
- Texel density

### PBR Texturing
- Albedo, roughness, metallic
- Normal maps
- Ambient occlusion
- Substance Painter workflow
