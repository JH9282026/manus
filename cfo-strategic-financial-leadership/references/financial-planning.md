# Financial Planning

Comprehensive guide to financial planning, budgeting, and analysis.

---

## Long-Range Financial Plan (LRFP)

### Components
1. **Revenue Plan**
   - Revenue by segment/product
   - Growth assumptions
   - Price and volume drivers

2. **Cost Plan**
   - Cost of goods sold
   - Operating expenses
   - Fixed vs. variable breakdown

3. **Investment Plan**
   - Capital expenditures
   - Working capital needs
   - M&A assumptions

4. **Funding Plan**
   - Debt facilities
   - Equity needs
   - Cash flow sufficiency

5. **Returns Plan**
   - Margin progression
   - ROIC trajectory
   - Shareholder returns

### Key Assumptions Document
- Economic assumptions (GDP, inflation)
- Industry assumptions (market growth)
- Company assumptions (win rates, pricing)
- Cost assumptions (labor, materials)
- Investment assumptions (capex, M&A)

## Annual Operating Plan (AOP)

### Budget Process Timeline
| Phase | Timing | Activities |
|-------|--------|------------|
| Planning kickoff | Month 1 | Guidelines, assumptions |
| First draft | Month 2 | Bottom-up submissions |
| Review/iteration | Month 3 | Consolidation, gaps |
| Approval | Month 4 | Board/executive approval |

### Budget Structure
- **Income Statement**: Revenue through net income
- **Balance Sheet**: Assets, liabilities, equity
- **Cash Flow**: Operating, investing, financing
- **Headcount**: By department, role
- **Capital**: By project, category

### Zero-Based vs. Incremental
| Approach | Description | Best For |
|----------|-------------|----------|
| Incremental | Prior year + changes | Stable operations |
| Zero-Based | Build from scratch | Cost optimization |
| Driver-Based | Linked to activities | Accuracy, flexibility |

## Variance Analysis

### Variance Types
| Type | Formula |
|------|----------|
| Absolute | Actual - Plan |
| Percentage | (Actual - Plan) / Plan |
| Volume | (Actual Vol - Plan Vol) × Plan Price |
| Price/Mix | (Actual Price - Plan Price) × Actual Vol |

### Analysis Framework
1. **Identify**: Significant variances
2. **Decompose**: Break into components
3. **Explain**: Root cause analysis
4. **Impact**: Full year implications
5. **Action**: Response if needed

### Reporting Format
```
Metric: [Name]
Actual: [Value]
Plan: [Value]
Variance: [Value] / [Percentage]
Explanation: [Root cause]
Impact: [Full year implication]
Action: [Response if needed]
```

## Financial Modeling

### Three-Statement Model
- Income Statement
- Balance Sheet
- Cash Flow Statement
- All linked and balanced

### Key Modeling Principles
- Separate assumptions from calculations
- Clear documentation
- Error checking built in
- Scenario capability
- Version control
