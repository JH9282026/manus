# Board Governance

Guidance on board management and corporate governance.

---

## Board Structure and Composition

### Board Committees

| Committee | Focus | Key Responsibilities |
|-----------|-------|---------------------|
| Audit | Financial oversight | Financial reporting, internal controls, auditor |
| Compensation | Executive pay | Compensation philosophy, CEO pay, equity plans |
| Nominating/Governance | Board composition | Director recruitment, governance policies |
| Risk | Enterprise risk | Risk framework, major risk oversight |

### Board Composition Considerations

| Factor | Assessment |
|--------|------------|
| Independence | Majority independent directors |
| Diversity | Gender, ethnic, professional diversity |
| Skills | Skills matrix coverage |
| Tenure | Balance of fresh and experienced |
| Availability | Sufficient time commitment |

## Board Meeting Management

### Meeting Cadence

| Meeting Type | Frequency | Duration |
|--------------|-----------|----------|
| Regular Board | Quarterly | 4-6 hours |
| Committee Meetings | Per charter | 1-2 hours |
| Strategy Session | Annual | 1-2 days |
| Special Meetings | As needed | Variable |

### Board Package Best Practices

| Principle | Application |
|-----------|-------------|
| Advance Distribution | 5-7 days before meeting |
| Consistent Format | Standard template and structure |
| Executive Focus | Strategic, not tactical detail |
| Decision Clarity | Clear recommendations and asks |
| Pre-Read Assumption | Don't present what's in materials |

### Board Package Template

```
## Board Meeting [Date]

**SECTION 1: EXECUTIVE SUMMARY**
- Key highlights (wins and challenges)
- Decisions required
- Matters for discussion

**SECTION 2: STRATEGIC UPDATE**
- Progress on strategic initiatives
- Strategic issues and decisions

**SECTION 3: FINANCIAL PERFORMANCE**
- Results vs plan
- Key variances and drivers
- Forecast update

**SECTION 4: OPERATIONAL REVIEW**
- Key operational metrics
- Major projects and initiatives

**SECTION 5: RISK AND COMPLIANCE**
- Enterprise risk update
- Compliance matters

**SECTION 6: LOOKING FORWARD**
- Outlook and guidance
- Key initiatives upcoming

**APPENDIX**
- Detailed financials
- Supporting materials
```

## CEO-Board Relationship

### Effective Board Engagement

| Practice | Application |
|----------|-------------|
| No Surprises | Proactive communication of issues |
| Transparency | Honest about challenges |
| Preparation | Well-prepared presentations |
| Follow-Through | Execute on commitments |
| Accessibility | Available for director questions |

### Managing Board Dynamics

| Situation | Approach |
|-----------|----------|
| Difficult Director | Address privately, involve chair |
| Board Conflict | Facilitate productive discussion |
| Micromanagement | Clarify roles, improve communication |
| Disengagement | Increase involvement, provide information |

## Governance Best Practices

### CEO Performance Evaluation

| Element | Approach |
|---------|----------|
| Annual Review | Formal evaluation by board |
| Criteria | Strategic, financial, leadership metrics |
| Process | Self-assessment, board assessment |
| Feedback | Direct from board chair |
| Development | Ongoing development plan |

### Succession Planning

| Element | Content |
|---------|--------|
| Emergency Succession | Immediate backup if CEO unavailable |
| Short-Term Succession | Interim leadership plan |
| Long-Term Succession | CEO development pipeline |
| Development Plans | Specific development for candidates |
| Board Oversight | Regular board review of plans |
