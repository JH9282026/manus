### Speech Writing Checklist

**Pre-Writing**:
- [ ] Audience clearly defined
- [ ] Purpose identified
- [ ] Core message crystallized
- [ ] Time constraints known
- [ ] Speaker voice researched

**Draft Review**:
- [ ] Strong, attention-grabbing opening
- [ ] Clear three-point structure (or similar)
- [ ] Stories and examples included
- [ ] Transitions smooth
- [ ] Memorable conclusion
- [ ] Call-to-action clear
- [ ] Appropriate for occasion
- [ ] Matches speaker voice

**Final Check**:
- [ ] Read aloud completely
- [ ] Timed accurately
- [ ] Approved by speaker
- [ ] Backup copies ready
- [ ] Delivery format prepared
