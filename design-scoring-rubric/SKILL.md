# Design Scoring Rubric

Apply a standardized 1-10 scoring rubric to objectively evaluate design quality across visual aesthetics, UX, content, technical, and innovation dimensions.

# Design Scoring Rubric

## Overview
This comprehensive rubric provides a standardized 1-10 scoring system for evaluating design quality across all dimensions. Use this rubric throughout the design process to objectively measure progress and identify areas for improvement.

## Scoring Scale
| Score | Level | Description |
|-------|-------|-------------|
| 1-2 | Poor | Fundamental issues; requires complete rework |
| 3-4 | Below Average | Major problems; needs significant revision |
| 5-6 | Average | Functional but unremarkable; needs improvement |
| 7-8 | Good | Quality work with minor issues; polish needed |
| 9-10 | Excellent | Professional quality; ready for production |

**Target: All designs must achieve 9/10 or higher before delivery.**

---

## Category 1: Visual Aesthetics (25%)
### 1.1 Color Usage (5%)

| Score | Criteria |
|-------|----------|
| 1-2 | Clashing colors, no cohesion, poor contrast, random application |
| 3-4 | Inconsistent palette, some contrast issues, unclear hierarchy |
| 5-6 | Basic palette works, adequate contrast, limited sophistication |
| 7-8 | Harmonious palette, good contrast, clear hierarchy, minor refinements needed |
| 9-10 | Sophisticated palette with excellent contrast, supports brand, creates mood, perfect balance |

**Key Questions**:
- Does the color palette create visual harmony?
- Are there at least 3 contrast levels for hierarchy?
- Do colors support the intended mood/brand?
- Is color used consistently throughout?

---

### 1.2 Typography (5%)

| Score | Criteria |
|-------|----------|
| 1-2 | Poor readability, inappropriate fonts, no hierarchy, inconsistent sizing |
| 3-4 | Readable but generic, weak hierarchy, inconsistent line heights |
| 5-6 | Adequate type choices, basic hierarchy, some inconsistencies |
| 7-8 | Good font pairing, clear hierarchy, proper sizing, minor spacing issues |
| 9-10 | Perfect type system with excellent pairing, clear hierarchy, optimal readability, consistent application |

**Key Questions**:
- Is body text easily readable (16px+ equivalent)?
- Is there clear typographic hierarchy (3+ levels)?
- Do font pairings complement each other?
- Are line heights and letter spacing optimized?

---

### 1.3 Layout & Composition (5%)

| Score | Criteria |
|-------|----------|
| 1-2 | Chaotic layout, no grid, poor alignment, crowded or too sparse |
| 3-4 | Basic layout with alignment issues, inconsistent spacing, weak structure |
| 5-6 | Functional layout, basic grid usage, some alignment inconsistencies |
| 7-8 | Well-structured layout, consistent grid, good balance, minor refinements needed |
| 9-10 | Perfect composition with clear visual hierarchy, consistent grid, excellent balance, intentional white space |

**Key Questions**:
- Is there a clear underlying grid system?
- Is alignment consistent throughout?
- Does white space feel intentional, not accidental?
- Is there visual balance across the design?

---

### 1.4 Visual Hierarchy (5%)

| Score | Criteria |
|-------|----------|
| 1-2 | No clear hierarchy, everything competes for attention |
| 3-4 | Weak hierarchy, unclear what's most important |
| 5-6 | Basic hierarchy exists but could be stronger |
| 7-8 | Clear hierarchy guiding attention well, minor improvements possible |
| 9-10 | Masterful hierarchy with clear primary, secondary, tertiary levels; eye naturally follows intended path |

**Key Questions**:
- Can users identify the primary action within 3 seconds?
- Is there clear differentiation between importance levels?
- Does the eye flow naturally through the content?
- Are CTAs prominent without being overwhelming?

---

### 1.5 Polish & Attention to Detail (5%)

| Score | Criteria |
|-------|----------|
| 1-2 | Rough, unfinished appearance, obvious inconsistencies |
| 3-4 | Basic polish lacking, visible rough edges, inconsistent treatments |
| 5-6 | Adequate polish, some inconsistencies in details |
| 7-8 | Good polish with minor inconsistencies, professional appearance |
| 9-10 | Pixel-perfect execution, consistent details, refined edges, premium feel |

**Key Questions**:
- Are all corners, shadows, and borders consistent?
- Is icon style unified throughout?
- Are hover/active states properly designed?
- Do small details enhance the overall experience?

---

## Category 2: User Experience (25%)
### 2.1 Usability (10%)

| Score | Criteria |
|-------|----------|
| 1-2 | Confusing interface, unclear how to accomplish tasks |
| 3-4 | Difficult to use, requires significant learning, unclear labels |
| 5-6 | Usable but not intuitive, some confusion points |
| 7-8 | Intuitive for most tasks, minor usability issues |
| 9-10 | Effortless to use, intuitive navigation, clear affordances, delightful interactions |

**Key Questions**:
- Can users complete primary tasks without confusion?
- Are interactive elements clearly identifiable?
- Is navigation logical and predictable?
- Are error states helpful and recoverable?

---

### 2.2 Information Architecture (5%)

| Score | Criteria |
|-------|----------|
| 1-2 | Illogical structure, impossible to find content |
| 3-4 | Confusing organization, many dead ends |
| 5-6 | Basic structure works, some navigation challenges |
| 7-8 | Logical structure, easy to find most content |
| 9-10 | Perfect organization, intuitive pathways, content always findable |

**Key Questions**:
- Is the site/app structure logical?
- Can users find what they need in 3 clicks or less?
- Is navigation consistent across all pages?
- Are labels clear and descriptive?

---

### 2.3 User Flow (5%)

| Score | Criteria |
|-------|----------|
| 1-2 | Broken flows, users get stuck or lost |
| 3-4 | Flows work but have friction points, unclear next steps |
| 5-6 | Functional flows with some unnecessary steps |
| 7-8 | Smooth flows with clear progression, minor optimizations possible |
| 9-10 | Optimal flows, minimal friction, clear progress indicators, satisfying completion |

**Key Questions**:
- Are key user journeys optimized?
- Is progress clearly communicated?
- Are there unnecessary steps that can be eliminated?
- Do users always know what to do next?

---

### 2.4 Content Clarity (5%)

| Score | Criteria |
|-------|----------|
| 1-2 | Confusing content, jargon-heavy, unclear purpose |
| 3-4 | Wordy content, some unclear messaging |
| 5-6 | Adequate content but could be clearer |
| 7-8 | Clear, concise content with minor improvements possible |
| 9-10 | Crystal clear messaging, scannable content, compelling copy, perfect tone |

**Key Questions**:
- Is content scannable with clear headings?
- Is the language appropriate for the audience?
- Are value propositions clearly communicated?
- Is content length appropriate for context?

---

## Category 3: Consistency & Systems (15%)
### 3.1 Design System Adherence (5%)

| Score | Criteria |
|-------|----------|
| 1-2 | No design system, everything is custom and inconsistent |
| 3-4 | Partial system with many one-off components |
| 5-6 | Basic system exists but inconsistently applied |
| 7-8 | Strong system with consistent application, minor variations |
| 9-10 | Comprehensive system, every element follows established patterns, highly scalable |

---

### 3.2 Component Consistency (5%)

| Score | Criteria |
|-------|----------|
| 1-2 | Same component looks different everywhere |
| 3-4 | Significant variations between similar components |
| 5-6 | Generally consistent with some variations |
| 7-8 | Highly consistent with rare exceptions |
| 9-10 | Perfect consistency, components are reusable and predictable |

---

### 3.3 Spacing & Rhythm (5%)

| Score | Criteria |
|-------|----------|
| 1-2 | Random spacing, no rhythm, chaotic appearance |
| 3-4 | Inconsistent spacing, weak visual rhythm |
| 5-6 | Basic spacing scale, some inconsistencies |
| 7-8 | Consistent spacing system, good rhythm, minor issues |
| 9-10 | Perfect spacing scale, clear rhythm, breathing room, harmonious proportions |

---

## Category 4: Responsiveness & Adaptation (15%)
### 4.1 Mobile Design (5%)

| Score | Criteria |
|-------|----------|
| 1-2 | Unusable on mobile, elements overlap or break |
| 3-4 | Functional but cramped, difficult interactions |
| 5-6 | Adequate mobile experience, some compromises |
| 7-8 | Good mobile experience, properly adapted |
| 9-10 | Excellent mobile-first design, optimized touch interactions, feels native |

---

### 4.2 Tablet/Desktop Adaptation (5%)

| Score | Criteria |
|-------|----------|
| 1-2 | Broken at different sizes, content stretches or breaks |
| 3-4 | Works but with awkward layouts at some sizes |
| 5-6 | Functional at all sizes, not optimized |
| 7-8 | Well-adapted with appropriate layouts per breakpoint |
| 9-10 | Perfect adaptation, each breakpoint feels intentionally designed |

---

### 4.3 Cross-Platform Consistency (5%)

| Score | Criteria |
|-------|----------|
| 1-2 | Completely different experience across platforms |
| 3-4 | Brand consistency lost across platforms |
| 5-6 | Basic consistency maintained |
| 7-8 | Strong consistency while respecting platform conventions |
| 9-10 | Perfect balance of brand consistency and platform appropriateness |

---

## Category 5: Accessibility (10%)
### 5.1 Color Accessibility (4%)

| Score | Criteria |
|-------|----------|
| 1-2 | Fails WCAG AA, low contrast throughout |
| 3-4 | Some contrast issues, color-only indicators |
| 5-6 | Meets WCAG AA for most elements |
| 7-8 | Fully WCAG AA compliant |
| 9-10 | WCAG AAA compliant, excellent for all users including color blind |

---

### 5.2 Interactive Accessibility (3%)

| Score | Criteria |
|-------|----------|
| 1-2 | No focus states, keyboard unusable, tiny targets |
| 3-4 | Basic focus states, some keyboard issues, small targets |
| 5-6 | Adequate focus states, keyboard mostly works, 44px targets |
| 7-8 | Good focus states, full keyboard support, adequate targets |
| 9-10 | Excellent focus states, perfect keyboard navigation, generous targets |

---

### 5.3 Content Accessibility (3%)

| Score | Criteria |
|-------|----------|
| 1-2 | No alt text, poor semantic structure |
| 3-4 | Basic alt text, some semantic issues |
| 5-6 | Adequate alt text and semantics |
| 7-8 | Good accessibility with proper ARIA where needed |
| 9-10 | Fully accessible, screen reader optimized, clear semantics |
