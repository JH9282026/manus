# Job Description Writing

Write job descriptions that attract qualified candidates, set expectations, and ensure compliance.

## Overview

This skill covers title optimization, role summaries, responsibility frameworks, requirement specification, inclusive language, ATS optimization, and legal compliance.

## Structure

| Section | Purpose | Length | Reference |
|---------|---------|--------|-----------|
| Title | Searchability | 2-5 words | `/references/job-description-structure.md` |
| Summary | Hook candidates | 3-5 sentences | `/references/job-description-structure.md` |
| Responsibilities | Set expectations | 6-10 bullets | `/references/job-description-structure.md` |
| Requirements | Define qualifications | 5-8 items | `/references/job-description-structure.md` |
| Compensation | Attract with value | 3-5 items | `/references/legal-compliance-optimization.md` |
| EEO Statement | Compliance | Standard | `/references/legal-compliance-optimization.md` |

## Writing Principles
1. Use standard, searchable titles
2. Lead with impact, not tasks
3. Be specific with scope and scale
4. Separate required from preferred qualifications
5. Use inclusive language
6. Optimize for ATS
7. Include salary range (increases applications 30%+)
