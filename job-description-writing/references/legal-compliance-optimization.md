# Legal Compliance & Optimization

## EEO Compliance
- Include EEO statement: "Equal Opportunity Employer"
- Avoid protected class references in requirements
- Don't require characteristics unrelated to job performance
- Document essential functions for ADA compliance

## ADA Considerations
- List essential functions (core duties that define the role)
- Describe physical requirements only if truly essential
- Use "ability to" rather than physical descriptions
- Include reasonable accommodation statement

## Inclusive Language
- Replace gendered terms: "chairman" -> "chairperson"
- Avoid age-coded: "digital native" -> "proficient with digital tools"
- Remove ability bias: "walk the floor" -> "conduct inspections"
- Use tools: Textio, Gender Decoder to audit language
- Focus on competencies over credentials

## ATS Optimization
- Use standard section headers (not creative names)
- Include relevant keywords from job function
- Avoid tables, images, or complex formatting
- Use consistent formatting and bullet points

## Compensation
- Include salary range (required in many jurisdictions)
- List benefits: health, retirement, PTO, remote work
- Mention equity/bonus if applicable
- Research market rates: Glassdoor, Levels.fyi, Payscale
- State whether range is negotiable
