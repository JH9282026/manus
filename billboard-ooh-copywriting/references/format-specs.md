# OOH Format Specifications

Detailed specifications for out-of-home advertising formats.

---

## Traditional Billboards

### Standard Sizes

| Format | Dimensions | Location | Best Use |
|--------|------------|----------|----------|
| Bulletin | 14' x 48' | Highways, arterials | Maximum impact |
| Poster | 12' x 24' | Secondary roads, urban | Frequency |
| Junior Poster | 6' x 12' | Pedestrian areas | Human scale |
| Wallscape | Variable | Building walls | Landmark |

### Design Specifications

- Resolution: 72-100 DPI at full size
- Color mode: CMYK for print
- Bleed: 2-4 inches all sides
- Safe area: 6-12 inches from edges
- File formats: PDF, AI, TIFF

---

## Digital Billboards (DOOH)

### Common Aspect Ratios

| Ratio | Resolution | Use |
|-------|------------|-----|
| 16:9 | 1920x1080 | Landscape digital |
| 4:3 | 1400x1050 | Traditional digital |
| 9:16 | 1080x1920 | Portrait/kiosk |

### Digital Capabilities

- **Dayparting**: Morning/lunch/evening creative rotation
- **Weather triggers**: Rain, sun, temperature-responsive
- **Live data**: Sports scores, countdowns, social feeds
- **Programmatic**: Real-time bidding and optimization

### Rotation Timing

- Standard: 8-second spot
- Loop: 60-second cycles (7-8 advertisers)
- Premium: 10-15 second spots
- Sequential: Multi-frame storytelling

## Transit Advertising

### Bus Formats

| Format | Size | Notes |
|--------|------|-------|
| King (side) | 30" x 144" | Full side panel |
| Queen (side) | 30" x 88" | Partial side |
| Tail (back) | 21" x 72" | Driver dwell time |
| Interior card | 11" x 28" | Captive audience |

### Subway/Rail

- Platform posters: 46" x 60"
- Station domination: Multiple formats
- Train wraps: Varies by system
- Digital platform: Network dependent

### Airport

- Terminal digital: Various sizes
- Baggage claim: Long dwell time
- Jet bridges: Captive audience
- Backlit displays: 4' x 6' standard

---

## Street Furniture

### Bus Shelter

- Standard size: 48" x 68"
- Backlit: Yes (24/7 visibility)
- Two-sided: Often dual display
- Dwell time: 3-15 minutes

### Urban Panel/Kiosk

- Multiple faces: 2-4 displays
- Digital integration: Common
- Eye level: Direct engagement
- Interactive potential: Touch, NFC
