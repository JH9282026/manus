# Recruitment Metrics and Analytics

Measuring and optimizing hiring effectiveness.

## Core Metrics

### Efficiency Metrics
- **Time to Fill:** Days from requisition to offer acceptance
- **Time to Hire:** Days from application to offer acceptance
- **Cost per Hire:** Total recruitment costs / number of hires

### Quality Metrics
- **Quality of Hire:** Performance ratings + retention + hiring manager satisfaction
- **Offer Acceptance Rate:** Offers accepted / offers extended
- **90-Day Retention:** New hires retained after 90 days

### Source Effectiveness
- **Source of Hire:** Track which channels yield best candidates
- **Source Quality:** Performance by source
- **Source Cost:** Cost per hire by source

### Experience Metrics
- **Candidate Satisfaction:** Survey scores
- **Application Completion Rate:** Started / completed applications
- **Interview Show Rate:** Attended / scheduled interviews

## Analytics Dashboards

### Key Visualizations
- Funnel analysis (application to hire)
- Time-to-fill trends
- Source effectiveness comparison
- Quality of hire by source
- Diversity metrics

### Reporting Frequency
- Real-time: Pipeline status
- Weekly: Activity metrics
- Monthly: Performance review
- Quarterly: Strategic analysis

## Predictive Analytics

### Use Cases
- Candidate success prediction
- Attrition risk modeling
- Hiring demand forecasting
- Source optimization
