# Growth Experimentation

Systematic approach to testing growth hypotheses.

## Experiment Framework

1. Hypothesis generation
2. Prioritization (ICE/PIE)
3. Design and setup
4. Execution
5. Analysis
6. Learning and iteration

## Hypothesis Template

```
We believe that [change]
will result in [impact]
because [reasoning].
We will measure using [metric].
```

## Prioritization

**ICE Score = Impact × Confidence × Ease**

Higher scores = higher priority

## Common Experiment Types

- Onboarding optimization
- Retention improvements
- Monetization tests
- Referral mechanics
- Feature adoption
