### Change Logs

**Change Log Contents:**
- Date of change
- Version number
- Change description
- Reason for change
- Author/approver
- Impact summary

**Change Log Example:**
```
Version 2.1 - 2024-03-15
- Added secondary approval for amounts >$50,000
- Updated routing rules for international payments
- Changed by: J. Smith
- Approved by: M. Johnson
- Reason: Compliance with updated policy FIN-2024-03
```
