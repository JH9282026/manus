### Audit Trail Components

**Essential Elements:**
1. **Who**: User identity and role
2. **What**: Action performed
3. **When**: Timestamp
4. **Where**: System/location
5. **Why**: Business reason (when applicable)
6. **Result**: Outcome of action

**Data Captured:**
- User authentication events
- Data access and modifications
- Approvals and rejections
- System configurations changes
- Error conditions
- Security events
