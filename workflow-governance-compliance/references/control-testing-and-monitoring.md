### Control Testing and Monitoring

**Testing Methods:**
- **Walkthroughs**: Trace transactions through process
- **Inquiry**: Interview control performers
- **Observation**: Watch control execution
- **Re-performance**: Execute control independently
- **Data Analysis**: Test data for anomalies

**Monitoring Approaches:**
- Continuous monitoring (automated)
- Periodic testing (scheduled)
- Exception-based review (triggered)
- Random sampling (statistical)
