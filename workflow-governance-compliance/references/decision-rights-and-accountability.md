### Decision Rights and Accountability

**RACI Matrix for Governance:**
- **Responsible**: Performs the work
- **Accountable**: Ultimate authority and accountability
- **Consulted**: Provides input before decisions
- **Informed**: Notified after decisions

**Decision Categories:**
| Decision Type | Authority Level |
|--------------|-----------------|
| Strategic process changes | Executive sponsor |
| Process standards | Process owner |
| Day-to-day operations | Process steward |
| Compliance interpretation | Compliance officer |
| Technical implementation | IT leadership |
