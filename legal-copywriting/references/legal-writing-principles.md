# Legal Writing Principles

## Plain Language Guidelines

### Readability Standards
- Target 8th-grade reading level (Flesch-Kincaid)
- Average sentence length: 15-20 words
- Avoid Latin phrases — use English equivalents
- Define technical terms on first use
- Use active voice whenever possible

### Substitution Guide
| Legal Jargon | Plain Language |
|-------------|---------------|
| Hereinafter | From now on / Going forward |
| Notwithstanding | Despite / Even if |
| Pursuant to | Under / According to |
| Shall | Must / Will |
| Aforementioned | Previously mentioned / Above |
| In the event that | If |
| Prior to | Before |
| Subsequent to | After |
| Terminate | End / Cancel |
| Indemnify | Protect / Cover losses |

### Formatting for Readability
- Use numbered sections for easy reference
- Bold key terms and obligations
- Use bullet points for lists (not run-on sentences)
- Include a table of contents for documents over 2 pages
- Add summary boxes at section starts

## EULA (End User License Agreement) Writing

### Key Sections
1. **License Grant** — Scope, limitations, restrictions
2. **Restrictions** — Reverse engineering, redistribution, modification
3. **Ownership** — IP remains with licensor
4. **Updates** — How updates are delivered and applied
5. **Support** — What support is provided
6. **Termination** — Auto-termination on breach
7. **Warranty Disclaimer** — AS IS, no guarantees
8. **Liability Limitation** — Cap damages
9. **Export Compliance** — International use restrictions

## Disclaimer Types and Templates

### General Disclaimer
Clarify that information is provided for general purposes and not professional advice. Specify no guarantee of accuracy or completeness.

### Medical/Health Disclaimer
State content is not medical advice, does not replace professional consultation, and users should consult healthcare providers.

### Financial Disclaimer
Clarify that content is not investment advice, past performance doesn't guarantee future results, and users should consult financial advisors.

### Affiliate Disclaimer
Disclose affiliate relationships, explain compensation structure, maintain honest review commitment.

## Refund Policy Best Practices

### Clear Structure
1. Eligibility criteria (timeframe, condition requirements)
2. Non-refundable items/services
3. Refund process (steps to request)
4. Processing timeline
5. Refund method (original payment method)
6. Partial refunds (when applicable)
7. Exchange options

### Regulatory Requirements
- EU: 14-day cooling-off period for online purchases
- Many US states: Require posted refund policies
- Digital goods: Specific exceptions may apply
- Subscriptions: Pro-rata refund considerations
