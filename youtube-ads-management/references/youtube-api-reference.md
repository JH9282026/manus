# YouTube Ads API Reference

## Google Ads API
Base: https://googleads.googleapis.com/

## Campaign Types
- VIDEO: YouTube campaigns

## Ad Formats
- IN_STREAM_SKIPPABLE: TrueView in-stream
- IN_STREAM_NON_SKIPPABLE: Non-skippable
- BUMPER: 6-second bumper
- VIDEO_DISCOVERY: TrueView discovery
- OUT_STREAM: Mobile-only

## Bidding Strategies
- TARGET_CPM: Target cost per 1000 impressions
- TARGET_CPV: Target cost per view
- MAXIMIZE_CONVERSIONS: Conversion focus
