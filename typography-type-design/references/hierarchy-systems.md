## Typographic Hierarchy

Hierarchy organizes information visually, guiding readers through content in order of importance.

### Creating Hierarchy

**Size**: Larger text draws attention first. Headlines are largest, followed by subheadings, then body text, with captions and fine print smallest.

**Weight**: Bold text commands attention within any size category. Use weight variations to emphasize key information without changing size.

**Color**: Darker or higher-contrast text appears more important. Color can also distinguish categories or create visual interest while maintaining hierarchy.

**Spacing**: Increased spacing around elements signals importance. Headlines often have generous surrounding space, while body text flows more densely.

**Position**: Elements positioned prominently (top, center, beginning) receive more attention. Placement reinforces hierarchy established through other means.

**Style**: Italics, all caps, small caps, and other stylistic variations create distinction without changing size or weight dramatically.

### Hierarchy Levels

**Primary (Headlines)**: The most prominent text, designed to capture attention and communicate main ideas. Headlines use the largest sizes, boldest weights, or most distinctive typefaces.

**Secondary (Subheadings)**: Organizational text that structures content into scannable sections. Subheadings are smaller than headlines but larger and more prominent than body text.

**Tertiary (Body Text)**: The main content, optimized for extended reading. Body text prioritizes readability over attention-grabbing, using comfortable sizes and proven text faces.

**Quaternary (Captions, Labels, Meta)**: Supporting text like captions, timestamps, footnotes, and interface labels. This level uses the smallest sizes while maintaining legibility.

### Visual Hierarchy Principles

**Contrast**: Create clear distinctions between hierarchy levels. Subtle differences get lost—ensure each level is obviously different from adjacent levels.

**Scale**: Establish a consistent scale system with proportional relationships between sizes. Modular scales create harmonious proportions throughout.

**Alignment**: Consistent alignment (left, center, or justified) reinforces structure. Alignment changes can signal hierarchy shifts but should be used intentionally.

**Proximity**: Related elements cluster together, separated from unrelated elements. Spacing patterns reinforce content relationships.

**Repetition**: Consistent treatment of equivalent elements (all H2s styled identically) creates recognizable patterns that readers learn quickly.

### Hierarchy in Practice

**Editorial Design**: Magazines use dramatic headline/body contrast, pull quotes, and varied column widths to create engaging layouts that guide reading flow.

**Web Design**: Digital interfaces rely on clear hierarchy for navigation, content consumption, and calls-to-action. Responsive hierarchy must maintain clarity across screen sizes.

**Poster Design**: Posters distill hierarchy to essentials—one dominant element captures attention, supported by minimal secondary information.

**UI Design**: Interface typography creates clear information architecture, distinguishing actions, labels, content, and feedback through consistent hierarchical systems.
