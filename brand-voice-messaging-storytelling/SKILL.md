# Brand Voice, Messaging & Storytelling

Develop distinctive brand voice, strategic messaging frameworks, and compelling brand stories that create emotional connections and drive consistent communication across all touchpoints.

## Overview

This skill provides comprehensive frameworks for defining how a brand speaks (voice), what it communicates (messaging), and the narratives it tells (storytelling). Use this to create unified brand communication that resonates with target audiences while maintaining consistency across channels.

## Voice vs Tone vs Messaging

| Element | Definition | Stability | Example |
|---------|------------|-----------|---------|
| Voice | Brand's consistent personality | Constant | Friendly, professional, bold |
| Tone | Voice adapted to context | Flexible | Celebratory, empathetic, urgent |
| Messaging | Strategic content communicated | Evolving | Value propositions, key messages |

## Voice Attribute Framework

Define 3-5 core attributes for each brand voice:

### For Each Attribute, Specify:
- **What it means**: Clear explanation
- **What it doesn't mean**: Boundaries
- **Do this**: Correct application examples
- **Don't do this**: Examples to avoid

### Voice Dimensions (Map on Spectrums)
- Formal ←→ Casual
- Serious ←→ Humorous
- Respectful ←→ Irreverent
- Enthusiastic ←→ Matter-of-fact
- Technical ←→ Simple

## Messaging Framework Hierarchy

1. **Core Message**: Single overarching brand message
2. **Key Pillars**: 3-5 supporting themes
3. **Proof Points**: Evidence for each pillar
4. **Supporting Messages**: Context-specific adaptations

## Value Proposition Formulas

| Formula | Structure | Best For |
|---------|-----------|----------|
| Helper | "We help [target] [achieve goal] by [approach]" | Service businesses |
| For-Who | "For [target] who [need], [brand] provides [solution]" | Product launches |
| Only | "[Brand] is the only [category] that [unique benefit]" | Differentiation |

## Storytelling Frameworks

### Hero's Journey (Brand Application)
1. Customer's ordinary world (current state)
2. Problem arises (call to adventure)
3. Discovers your brand (meeting the mentor)
4. Tries solution (crossing threshold)
5. Achieves transformation (reward)
6. Becomes advocate (return with elixir)

### Before-After-Bridge (BAB)
- **Before**: Paint current painful state
- **After**: Show desired future state
- **Bridge**: Explain how to get there

### Problem-Agitate-Solve (PAS)
- **Problem**: Identify issue clearly
- **Agitate**: Emphasize pain and consequences
- **Solve**: Present your solution

## Story Types to Develop

| Type | Purpose | Key Elements |
|------|---------|--------------|
| Origin | Build emotional connection | Founding insight, early struggles, breakthrough |
| Customer | Demonstrate transformation | Challenge, solution, measurable results |
| Purpose | Show mission in action | Values, community impact, vision |
| Product | Explain creation and care | Development journey, problem solved |

## Tone Matrix Template

| Context | Formality | Mood | Example |
|---------|-----------|------|---------|
| Customer support | Semi-formal | Empathetic | "We understand—let's fix this." |
| Marketing | Casual | Enthusiastic | "Ready to transform your workflow?" |
| Error messages | Semi-formal | Apologetic | "Something went wrong. Try again?" |
| Success | Casual | Celebratory | "You're all set! High five!" |
| Crisis | Formal | Transparent | "Here's what happened and what we're doing." |

## Tagline Characteristics

- **Length**: 3-7 words ideal
- **Memorable**: Rhythm, alliteration, or rhyme
- **Unique**: Distinctive and ownable
- **Meaningful**: Conveys value or personality
- **Timeless**: Built to last, not trendy

## Content Strategy Pillars

Define 3-5 content themes aligned with messaging:
- **Education**: How-tos, tutorials, industry insights
- **Inspiration**: Success stories, case studies
- **Entertainment**: Behind-the-scenes, culture, humor
- **Community**: User content, conversations
- **Product**: Features, updates, use cases
