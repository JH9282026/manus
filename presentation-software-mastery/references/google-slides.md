## Google Slides



### Essential Features and Tools

**Text and Typography:**
- Google Fonts library access
- Basic text formatting
- Bullet and numbering options
- Text alignment and spacing
- Find and replace

**Shapes and Graphics:**
- Shape library with categories
- Line and connector tools
- Diagrams (Insert > Diagram)
- Word Art for stylized text
- Polyline for custom shapes

**Media:**
- Insert images from multiple sources (upload, web, Drive, Photos, camera)
- YouTube video embedding
- Audio insertion (from Drive)
- Image editing (crop, mask, recolor)

### Master Slides and Themes

**Accessing Theme Editor:**
- Slide > Edit theme
- Or click Theme button in toolbar > Edit theme

**Theme Components:**
- Master slide with global formatting
- Layout templates for different slide types
- Color palette and fonts
- Background options

**Creating Custom Themes:**
1. Edit theme to modify master slides
2. Set colors: Theme > Colors
3. Import themes from other presentations
4. No direct theme export (copy presentation as template)

**Theme Best Practices:**
- Set up layouts for all your content types
- Define consistent placeholder positioning
- Apply custom colors to the theme palette
- Save a "template presentation" in Drive



### Google Workspace Integration

**Google Drive:**
- Organize presentations in folders
- Share with Drive permissions
- Search presentations from Drive
- Access from any device

**Google Docs:**
- Embed presentations in documents
- Link to specific slides
- Export outlines to Docs

**Google Sheets:**
- Link charts from Sheets
- Charts update automatically when data changes
- Insert > Chart > From Sheets

**Google Meet:**
- Present directly to Meet calls
- Screen share with presenter view
- Collaborative viewing in Meet

**Google Sites:**
- Embed presentations in websites
- Auto-updating embedded content
- Responsive display
