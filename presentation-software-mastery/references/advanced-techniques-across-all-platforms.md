## Advanced Techniques Across All Platforms


### Embedding Media Effectively

**Video Best Practices:**
- Compress videos before embedding
- Use MP4 H.264 for compatibility
- Consider YouTube links in Google Slides
- Set poster frames for thumbnails
- Test playback on presentation computer

**Audio Integration:**
- Use background music sparingly
- Set appropriate volume levels
- Consider fade in/out effects
- Provide audio controls for accessibility
- Test audio output in venue

**Web Content:**
- PowerPoint: Use web viewer add-ins
- Keynote: Export to HTML
- Google Slides: Embed iframes via add-ons


### Custom Animations

**Animation Principles:**
- Use animations purposefully (not decoratively)
- Keep timing consistent (0.3-0.5 seconds typical)
- Animate content in logical sequence
- Use entrance animations for new content
- Reserve emphasis for key points

**Building Complex Sequences:**
1. Plan animation flow with storyboard
2. Layer objects in correct order
3. Set trigger points (on click vs. automatic)
4. Test timing with rehearsal
5. Simplify if animations distract


### Presenter View Mastery

**All Platforms Offer:**
- Current slide display
- Next slide preview
- Speaker notes
- Timer/clock
- Navigation tools
- Annotation tools (pen/laser)

**Presenter View Setup:**
- Connect external display before opening software
- Configure display settings (extend desktop)
- Practice with presenter view before presenting
- Know shortcuts to swap displays


### Remote and Virtual Presenting

**Screen Sharing Best Practices:**
- Share specific window (not entire screen)
- Hide desktop notifications
- Use presenter view in a separate window
- Test audio/video before presenting
- Have backup plan for technical issues

**Hybrid Presenting:**
- Test both in-person and virtual displays
- Use larger fonts for remote viewers
- Ensure animations work in screen share
- Consider recording for asynchronous viewing
