## Microsoft PowerPoint



### Essential Features and Tools

**Text and Typography:**
- Format Painter (Ctrl+Shift+C / Ctrl+Shift+V)
- Character and paragraph spacing controls
- Text columns and bullet customization
- WordArt for stylized text effects
- Find and Replace with formatting options

**Shapes and Graphics:**
- Shape gallery with extensive options
- Merge Shapes (Union, Combine, Fragment, Intersect, Subtract)
- Shape formatting (fill, outline, effects)
- Grouping and alignment tools
- Precise positioning with guides and gridlines

**Images and Media:**
- Image compression and cropping tools
- Remove Background feature
- Artistic effects and color corrections
- 3D model insertion and manipulation
- Video trimming and playback options
- Screen recording (Insert > Screen Recording)



### Collaboration and Sharing

**Real-Time Collaboration:**
- Save to OneDrive or SharePoint
- Share via Share button (top right)
- Co-edit with presence indicators
- Track changes with comments
- Use @mentions for notifications

**Export Options:**
- PDF (high-quality for distribution)
- Video (MP4 with timings and narrations)
- GIF (for social media)
- Images (PNG, JPG)
- Handouts (with notes pages)
- PowerPoint Show (.ppsx) for presentation-only

**Presenter View Setup:**
1. Connect to external display
2. Slide Show tab > Use Presenter View
3. Features: Current slide, next slide, notes, timer, tools



### PowerPoint Tips and Tricks

1. **Quick alignment**: Select multiple objects > Format > Align
2. **Duplicate and space**: Ctrl+D creates evenly spaced copies
3. **Embed fonts**: File > Options > Save > Embed fonts
4. **Reduce file size**: File > Compress Pictures
5. **Quick chart update**: Right-click chart > Edit Data
6. **Selection Pane**: Home > Select > Selection Pane for layer management
7. **Eyedropper**: Format Shape > Shape Fill > Eyedropper
8. **Snap to grid**: View > Gridlines and Guides
9. **Rehearse timings**: Slide Show > Rehearse Timings
10. **Package presentation**: File > Export > Package for CD
