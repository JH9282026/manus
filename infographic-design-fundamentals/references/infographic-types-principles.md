# Infographic Types & Principles

## Types
- **Statistical**: lead with impactful stat; charts, percentages, pictograms
- **Timeline**: vertical (digital) or horizontal (presentation); date markers, milestones
- **Process**: numbered steps, arrows, icons, brief text per step
- **Comparison**: side-by-side, consistent categories, visual indicators
- **Geographic**: accurate maps, color/size coding, legends
- **Hierarchical**: tree structures, size = importance, color = category

## Visual Hierarchy
- Size contrast: headlines 3-4x body
- Color: bright draws attention; muted recedes
- Position: top-left highest priority (Western reading)
- Isolation: white space = importance

## Color Theory
- 3-5 colors plus neutrals
- Brand primary + complementary accents
- 4.5:1 contrast for text
- Color-blind safe: avoid red-green only

## Typography
- Max 2-3 typefaces
- Title -> headers -> body -> captions -> source
- Min 14px digital body; 10pt print
- Sans-serif for screen; serif for print elegance
