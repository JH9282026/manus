# Data Visualization & Production

## Chart Best Practices
- Bar charts: start at zero baseline
- Horizontal bars for long labels
- Pie charts: max 5-6 segments
- Data labels directly on charts
- Consistent colors for same categories
- Sort meaningfully

## Platform Dimensions
| Platform | Size |
|----------|------|
| Blog | 800-1200px wide |
| Pinterest | 1000x1500 |
| Instagram | 1080x1080 or 1080x1350 |
| Twitter | 1200x675 |
| LinkedIn | 1200x627 |

## File Formats
- PNG: web, transparency
- PDF: print, downloads
- SVG: scalable, interactive
- JPEG: smaller size, no transparency

## Production Checklist
1. Verify data accuracy; cite sources
2. Spell check
3. Color contrast accessibility
4. Test at display size
5. Export required formats
6. Compress for web
7. Add alt text
