# Infographic Design Fundamentals

Design compelling infographics that transform complex information into clear, engaging visual stories.

## Overview

This skill covers infographic types, design principles, layout patterns, color theory, typography, and data visualization for creating effective visual content.

## Core Principles
1. Visual hierarchy — size, color, position guide the eye
2. Simplicity — one main message; remove non-essentials
3. Flow — logical reading path
4. Consistency — uniform colors, fonts, icons
5. White space — prevent clutter
6. Data integrity — accurate representation
7. Accessibility — contrast; don't rely on color alone
