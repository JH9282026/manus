# Cryptocurrency Fundamental Analysis

Evaluate digital assets beyond price action by examining economic structures, utility propositions, team credentials, technological architecture, and macro-environmental factors.

## Overview

This skill addresses unique aspects of decentralized systems, token economics, and the intersection of technology and monetary policy—areas not covered by traditional equity analysis.

## Tokenomics Framework

### Supply Dynamics
- **Max Supply**: Total tokens that can exist
- **Circulating Supply**: Currently available
- **Emission Schedule**: Rate of new token creation
- **Burn Mechanisms**: Token destruction features

### Distribution Analysis
- Pre-mine vs. fair launch
- Team/advisor allocation (%)
- Vesting schedules and cliffs
- Insider concentration risk

### Utility Assessment
- Governance rights
- Staking rewards
- Fee payment/discounts
- Access/membership
- Collateral function

## Valuation Metrics

| Metric | Calculation | Interpretation |
|--------|-------------|----------------|
| Market Cap | Price × Circulating Supply | Current valuation |
| FDV | Price × Max Supply | Fully diluted value |
| NVT Ratio | Network Value / Transaction Volume | Higher = potentially overvalued |
| MVRV | Market Value / Realized Value | >3.7 historically overbought |

## Red Flags

- Heavy insider allocation (>40%)
- Complex tokenomics obscuring value flows
- Unclear utility or forced token usage
- Short vesting with large unlocks
- FDV >> Market Cap (dilution risk)
