# Landing Page Design

## Lead Generation Pages
- Form as centerpiece; offer value in exchange
- Lead magnets: ebook, whitepaper, consultation, tool
- Fields: name + email minimum; add qualifying fields cautiously
- Immediate delivery after conversion

## Click-Through Pages
- Warm visitors before main conversion page
- Product benefits, social proof, desire building
- Single CTA to checkout or signup

## Sales Pages
- Long-form AIDA: Attention, Interest, Desire, Action
- Headline -> problem -> agitation -> solution -> proof -> offer -> guarantee -> CTA
- Multiple CTAs throughout
- Testimonials, case studies, FAQs

## Squeeze Pages
- Minimalist: headline + value prop + email field
- No navigation, minimal content
- Maximum focus on single conversion

## Hero Section Design
- Headline: clear value in 6-10 words
- Subheadline: expand on promise
- Image/video: reinforce message (product shots, not stock)
- CTA: "Get instant access" not "Submit"
- Trust: ratings, logos, testimonial snippet

## Visual Layout
- F-pattern or Z-pattern reading flow
- Directional cues pointing to CTA
- White space for breathing room
- Contrasting CTA button color
- Above-the-fold critical content
