# Mediation Techniques

Structured approaches for facilitating conflict resolution.

---

## Pre-Mediation Preparation

### Individual Meetings
1. Build rapport with each party
2. Understand their perspective
3. Identify underlying interests
4. Set realistic expectations
5. Explain the process

### Session Planning
- Select neutral location
- Arrange seating thoughtfully
- Prepare ground rules
- Plan opening statement

---

## Mediation Session Structure

### 1. Opening (10-15 min)
```
"Thank you both for being here. My role is to help
you have a productive conversation, not to judge
or decide who's right. I'm neutral. The goal is to
find a solution that works for both of you."
```

### 2. Party Statements (15-20 min each)
- Uninterrupted time for each party
- Active listening by mediator
- Summarize and reflect

### 3. Issue Identification (20-30 min)
- List all issues
- Prioritize together
- Separate positions from interests

### 4. Option Generation (30-45 min)
- Brainstorm without judging
- Focus on interests
- Encourage creativity
- Reality test options

### 5. Agreement Building (20-30 min)
- Negotiate specific terms
- Address concerns
- Document agreements
- Plan implementation

## Interest-Based Problem Solving

### Position vs. Interest
- **Position**: What someone says they want
- **Interest**: Why they want it (underlying need)

### Uncovering Interests
- "What would that give you?"
- "Why is that important?"
- "What are you really concerned about?"

### Example
- Position: "I need a private office"
- Interest: Need for concentration, status, confidentiality
