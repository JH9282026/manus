## Key Performance Metrics

| Metric | Formula | Target |
|--------|---------|--------|
| OEE | Availability × Performance × Quality | >85% world-class |
| First Pass Yield | Good units / Total units | >95% |
| Cycle Time | End time - Start time | Minimize |
| Takt Time | Available time / Customer demand | Match demand |
| Scrap Rate | Scrap units / Total produced | <2% |
| On-Time Delivery | On-time orders / Total orders | >98% |
| Inventory Turns | COGS / Average inventory | Industry benchmark |

## Eight Wastes of Lean (DOWNTIME)

| Waste | Description | Countermeasure |
|-------|-------------|---------------|
| Defects | Rework, scrap, returns | Poka-yoke, SPC |
| Overproduction | Making more than needed | Pull systems, Kanban |
| Waiting | Idle time between steps | Flow optimization |
| Non-utilized talent | Underused employee skills | Cross-training |
| Transportation | Unnecessary material movement | Layout optimization |
| Inventory | Excess WIP and finished goods | JIT, demand leveling |
| Motion | Unnecessary worker movement | Workplace organization (5S) |
| Extra processing | Over-engineering | Value stream mapping |
