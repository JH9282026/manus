# Lean Manufacturing & Six Sigma

## Lean Manufacturing Tools

### 5S Workplace Organization
1. **Sort (Seiri)** — Remove unnecessary items
2. **Set in Order (Seiton)** — Organize remaining items
3. **Shine (Seiso)** — Clean workplace and equipment
4. **Standardize (Seiketsu)** — Create standards for 1-3
5. **Sustain (Shitsuke)** — Maintain discipline with audits

### Value Stream Mapping (VSM)
1. Select product family to map
2. Document current state (material and information flow)
3. Identify value-add vs. non-value-add time
4. Calculate lead time and process cycle efficiency
5. Design future state map with improvements
6. Create implementation plan with priorities

### Kanban System
- **Visual signal** for production/replenishment
- **Pull-based** — only produce when downstream requests
- Calculate Kanban quantity: (Daily demand × Lead time × Safety factor) / Container size
- Start with more Kanbans and gradually reduce
- Track WIP limits at each workstation

### Kaizen Events
- 3-5 day focused improvement workshops
- Cross-functional team of 6-10 members
- Follows Plan-Do-Check-Act cycle
- Target: 30-50% improvement in specific metric
- Immediate implementation, not reports

## Six Sigma DMAIC

### Define Phase
- Project charter with problem statement
- Voice of Customer (VOC) analysis
- SIPOC diagram (Suppliers, Inputs, Process, Outputs, Customers)
- Project scope and goals

### Measure Phase
- Identify key metrics (CTQ — Critical to Quality)
- Establish measurement system (Gage R&R)
- Collect baseline data
- Calculate process capability (Cp, Cpk)
- Create process maps

### Analyze Phase
- Root cause analysis (Fishbone, 5 Whys)
- Statistical analysis (hypothesis testing, regression)
- Failure Mode and Effects Analysis (FMEA)
- Identify vital few variables (Pareto principle)

### Improve Phase
- Generate solution alternatives
- Design of Experiments (DOE)
- Pilot improvements
- Validate results statistically
- Implement controls

### Control Phase
- Statistical Process Control (SPC) charts
- Control plans and response plans
- Standard Operating Procedures (SOPs)
- Training and handoff documentation
- Monitoring dashboard
