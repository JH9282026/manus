# Shot Matching

## Matching Workflow

### Step 1: Establish Reference
1. Choose best-looking shot as reference
2. Or create target look first
3. Apply initial corrections
4. This becomes matching target

### Step 2: Compare Using Scopes
1. Split-screen reference vs. current shot
2. Compare waveforms for exposure
3. Compare vectorscopes for color
4. Compare RGB parade for balance

### Step 3: Match Exposure
1. Align black points (lift)
2. Align white points (gain)
3. Match overall brightness (gamma)
4. Verify on waveform

### Step 4: Match White Balance
1. Identify color cast differences
2. Adjust RGB balance to match
3. Focus on neutrals (whites, grays)
4. Verify on parade scope

### Step 5: Match Contrast
1. Compare tonal spread
2. Adjust contrast to match feel
3. Check shadow and highlight rolloff

### Step 6: Match Saturation
1. Compare on vectorscope
2. Adjust global saturation
3. Fine-tune specific colors

## Common Matching Challenges

### Different Cameras
- Match color science first
- Use camera-specific LUTs
- Adjust for dynamic range differences
- Unify white balance

### Different Lighting
- Match overall color temperature
- Balance contrast levels
- Unify shadow colors
- Consider mixed lighting

### Different Times of Day
- Match exposure carefully
- Unify color temperature
- Consider natural vs. artificial
- Accept some variation for realism

## Group-Based Workflow

### Grouping Strategy
1. Group shots by scene/setup
2. Apply base grade to group
3. Make individual adjustments
4. Maintain consistency efficiently

### Group Types
- Scene groups: Same location/lighting
- Character groups: Consistent skin tones
- Time-of-day groups: Consistent ambiance

---

## Reference Still Workflow

### Using Reference Stills
1. Capture still from target shot
2. Display alongside current shot
3. Match to reference
4. Apply across similar shots

### Reference Best Practices
- Use representative frames
- Include variety of lighting
- Document reference selections
- Update references as grade evolves
