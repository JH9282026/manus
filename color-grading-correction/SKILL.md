# Color Grading & Color Correction

Transform raw footage into polished, professional video through technical correction and creative grading.

## Overview

This skill provides comprehensive color workflows for video post-production. Apply technical corrections for accurate, consistent footage, then creative grades for emotional impact and visual style. Master primary and secondary corrections, shot matching, and delivery for various platforms.

## Workflow Selection

| Footage Type | Primary Focus | Key Steps | Reference |
|--------------|---------------|-----------|----------|
| LOG/RAW | Full pipeline | Normalize, correct, grade | `/references/primary-correction.md` |
| Standard Rec.709 | Enhancement | Balance, match, stylize | `/references/primary-correction.md` |
| Multi-camera | Consistency | Match, unify look | `/references/shot-matching.md` |
| Creative project | Style | Develop look, apply | `/references/creative-grading.md` |
| Broadcast delivery | Compliance | Legalize, QC | `/references/delivery-specs.md` |

## Core Competencies

### Color Correction
- Normalize exposure (lift, gamma, gain)
- Balance white balance and remove color casts
- Adjust contrast and dynamic range
- Match shots for continuity
- Correct skin tones

### Color Grading
- Develop creative looks and styles
- Apply cinematic color palettes
- Create mood and atmosphere
- Build brand-consistent visuals
- Design period and genre looks

### Secondary Corrections
- Isolate and adjust specific colors (HSL)
- Create and track power windows
- Correct skin tones separately
- Enhance skies and environments
- Fix localized exposure issues

### Delivery Optimization
- Ensure broadcast-safe levels
- Output for multiple platforms
- Create and manage LUTs
- Prepare HDR deliverables
- Quality control review

## Color Pipeline

| Phase | Purpose | Tools |
|-------|---------|-------|
| Input transform | Normalize LOG/RAW | Technical LUT, color space |
| Primary correction | Fix exposure, balance | Color wheels, curves |
| Shot matching | Consistency | Scopes, reference stills |
| Secondary correction | Targeted fixes | Qualifiers, masks |
| Creative grade | Style and mood | Looks, creative LUTs |
| Output transform | Delivery format | Color space, levels |

## Reading the Scopes

| Scope | Shows | Use For |
|-------|-------|----------|
| Waveform | Luminance (brightness) | Exposure, contrast |
| RGB Parade | RGB channels separate | Color balance, casts |
| Vectorscope | Hue and saturation | Color accuracy, skin tones |
| Histogram | Brightness distribution | Overall exposure |
