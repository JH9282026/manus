### Keyboard Shortcuts

**Essential Shortcuts**:
- Cmd+Y: Create symbol
- Cmd+G: Group
- Cmd+R: Rename
- Cmd+D: Duplicate
- Cmd+L: Lock layer
- Cmd+Shift+L: Unlock all
- Cmd+E: Export
- Cmd+K: Scale

**Custom Shortcuts**:
- Preferences > Shortcuts
- Assign to frequent actions
- Plugin shortcuts
