# Commercial Real Estate Strategy

Manage commercial real estate portfolios for maximum value creation.

## Overview

This skill provides comprehensive CRE strategy capabilities across portfolio optimization, leasing, tenant management, development feasibility, and asset repositioning. Apply institutional-quality analysis frameworks to maximize property-level NOI and portfolio returns.

## Strategy Type Selection

| Strategic Priority | Focus Area | Key Analysis | Reference |
|-------------------|------------|--------------|----------|
| Portfolio optimization | Hold/sell/buy | Asset assessment matrix | `/references/portfolio-strategy.md` |
| Revenue growth | Leasing strategy | Rent optimization, occupancy | `/references/leasing-strategy.md` |
| Value creation | Asset repositioning | Capital improvements, ROI | `/references/asset-management.md` |
| Development | Feasibility analysis | Market, returns | `/references/development-analysis.md` |
| Exit planning | Disposition | Timing, pricing | `/references/portfolio-strategy.md` |

## Core Competencies

### Portfolio Strategy
- Assess portfolio composition and performance
- Develop hold/sell/acquire recommendations
- Optimize geographic and sector allocation
- Balance risk and return across assets
- Plan capital deployment

### Leasing and Tenant Strategy
- Analyze lease economics and mark-to-market
- Design tenant mix strategies
- Develop leasing guidelines and parameters
- Structure deal terms and concessions
- Manage lease rollover risk

### Asset Management
- Create asset business plans
- Identify value-add opportunities
- Plan capital improvements
- Optimize operating expenses
- Track performance metrics

### Development Analysis
- Conduct market feasibility studies
- Model development returns
- Assess development risk
- Plan entitlement strategies
- Evaluate build vs. buy decisions

## CRE Metrics

| Metric | Calculation | Use |
|--------|-------------|-----|
| NOI | Revenue - Operating Expenses | Property performance |
| Cap Rate | NOI / Property Value | Valuation, comparison |
| Cash-on-Cash | Annual Cash Flow / Equity | Investor returns |
| IRR | Internal Rate of Return | Total return |
| DSCR | NOI / Debt Service | Leverage capacity |
