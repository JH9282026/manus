# Property Type Analysis

## Office

### Key Metrics

| Metric | Description | Benchmark |
|--------|-------------|----------|
| Rent PSF | Annual rent per SF | Market dependent |
| Occupancy | Leased / total SF | 90%+ stabilized |
| Tenant retention | Renewal rate | 70%+ |
| TI/LC | Tenant improvement + commission | $50-100/SF typical |
| Operating expenses | Cost PSF | $15-25/SF typical |

### Value Drivers
- Location and accessibility
- Building class and quality
- Amenities and services
- Tenant credit quality
- Lease term and rollover

### Current Trends
- Flight to quality post-COVID
- Amenity expectations rising
- Flexible space demand
- ESG importance increasing
- Suburban vs. urban shifts

## Retail

### Key Metrics

| Metric | Description | Benchmark |
|--------|-------------|----------|
| Sales PSF | Tenant sales productivity | Category dependent |
| Occupancy cost | Rent / sales | 8-12% typical |
| Traffic | Foot traffic counts | Trending data |
| Tenant sales | Reported sales | Growth trends |

### Tenant Categories

| Category | Role | Lease Terms |
|----------|------|-------------|
| Anchor | Traffic driver | Long-term, lower rent |
| Junior anchor | Secondary draw | Medium term |
| In-line | Fill space, variety | Shorter, higher rent |
| Outparcel/Pad | Convenience | Ground lease or ownership |

### Value Drivers
- Traffic and visibility
- Anchor tenant quality
- Tenant mix synergy
- Trade area demographics
- Competition and accessibility

## Industrial

| Metric | Description | Benchmark |

| Dock doors | Loading capacity | Per 10,000 SF |

| Type | Use | Specifications |
|------|-----|----------------|

### Value Drivers

## Multifamily

### Key Metrics

| Metric | Description | Benchmark |
|--------|-------------|----------|
| Rent per unit | Monthly rent | Market dependent |
| Occupancy | Occupied / total units | 95%+ stabilized |
| Rent growth | YoY rent increase | Market + asset specific |
| Turnover | Annual turn rate | 40-50% typical |
| OpEx per unit | Operating costs | $4-8K/unit/year |

### Subtypes

| Type | Characteristics |
|------|----------------|
| Garden | Low-rise, suburban |
| Mid-rise | 4-6 stories, urban/suburban |
| High-rise | 7+ stories, urban core |
| Student | University proximity |
| Senior | Age-restricted, services |
| Affordable | Income-restricted |

### Value Drivers
- Location and accessibility
- Unit mix and finishes
- Amenity package
- Property management
- Rent growth potential
