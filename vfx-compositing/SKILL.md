# VFX Compositing

Composite 3D renders with live action for film and VFX production.

## Overview

VFX compositing combines 3D renders, live-action footage, and effects into final shots using node-based workflows in software like Nuke, After Effects, and Fusion.

## Core Concepts

### Keying
- Green/blue screen keying
- Despill and edge refinement
- Garbage mattes
- Clean plate integration

### Tracking
- 2D tracking
- 3D camera tracking
- Planar tracking
- Match moving

### Color Management
- Linear workflow
- Color spaces (sRGB, ACEScg)
- LUTs and color grading
- Match lighting and color
