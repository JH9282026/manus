# Campaign Measurement & Optimization

## KPI Framework

### Awareness Metrics
| Metric | Target Range | Measurement |
|--------|-------------|-------------|
| Impressions | Campaign-dependent | Ad platform |
| Reach | X% of target audience | Platform analytics |
| Brand recall | +5-15% lift | Brand lift study |
| Share of voice | Benchmark vs competitors | Media monitoring |

### Engagement Metrics
| Metric | Benchmark | Measurement |
|--------|-----------|-------------|
| CTR | 1-3% (display), 3-5% (search) | Ad platform |
| Video view rate | 15-30% | Platform analytics |
| Social engagement | 1-5% | Social analytics |
| Email open rate | 20-30% | ESP |
| Content time on page | 2-4 minutes | Web analytics |

### Conversion Metrics
| Metric | Definition | Target |
|--------|-----------|--------|
| Conversion rate | Conversions / Clicks | 2-5% (landing page) |
| Cost per lead (CPL) | Spend / Leads | Industry benchmark |
| Cost per acquisition (CPA) | Spend / Customers | Below LTV threshold |
| Return on ad spend (ROAS) | Revenue / Ad spend | 3-5x minimum |
| Customer lifetime value (LTV) | Total revenue per customer | Track trending |

## Attribution Models

| Model | Logic | Best For |
|-------|-------|----------|
| Last click | 100% credit to final touchpoint | Direct response |
| First click | 100% credit to first touchpoint | Awareness campaigns |
| Linear | Equal credit across all touchpoints | Multi-touch evaluation |
| Time decay | More credit to recent touchpoints | Long sales cycles |
| Position-based | 40% first, 40% last, 20% middle | Balanced view |
| Data-driven | Algorithmic based on actual data | Advanced, high volume |

## Campaign Optimization Process

### Weekly Review Checklist
- [ ] Check spend vs. budget pace
- [ ] Review CPL/CPA vs. targets
- [ ] Identify top and bottom performing channels
- [ ] Assess creative fatigue (CTR decline)
- [ ] Review audience segment performance
- [ ] Check landing page conversion rates
- [ ] Adjust bids and budgets based on performance

### Optimization Levers
| Lever | When to Pull |
|-------|-------------|
| Budget reallocation | Channel performance divergence |
| Bid adjustments | CPL/CPA above target |
| Creative refresh | CTR declining >20% |
| Audience refinement | Segment performance variance |
| Landing page changes | Conversion rate below benchmark |
| Dayparting | Time-of-day performance patterns |
| Geographic focus | Regional performance differences |

### A/B Testing Priority
1. Audience segments (highest impact)
2. Messaging and value proposition
3. Creative format (image vs video)
4. Landing page layout
5. CTA wording and placement
6. Send time and frequency
