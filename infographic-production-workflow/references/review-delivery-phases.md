# Review Through Delivery

## Phase 5: Review
- Internal: design quality, data accuracy, messaging, accessibility
- Client: present with rationale; structured feedback form
- 2-3 revision rounds (define in scope)
- Written sign-off before finalization

## Phase 6: Finalization
- Typography check: alignment, spacing
- Color consistency verification
- Data accuracy final check
- Spell check and grammar review
- Export: PNG (web), PDF (print), social crops
- Compress without quality loss
- Generate embed code

## Quality Assurance
- View at actual display size
- Test on mobile devices
- Verify links/interactive elements
- Cross-browser test
- Final stakeholder sign-off

## Phase 7: Delivery
- Organized folder structure with labeled files
- Source/editable files if contracted
- Documentation: file inventory, data sources, embed code
- Distribution strategy recommendations
- Project retrospective and archive
