# Discovery Through Design

## Phase 1: Discovery
- Stakeholder meeting: objectives, audience, context
- Define goal: what action should viewers take?
- Scope: single/series, static/interactive, channels
- Review brand guidelines and competitive examples
- Establish budget, timeline, revision rounds

## Data Collection
- Primary sources: surveys, databases, internal reports
- Secondary: industry reports, government stats, academic studies
- Verify accuracy through cross-referencing
- Organize with source citations; identify gaps

## Phase 2: Content Strategy
- Identify single most important message
- Narrative structure: what story does data tell?
- Information hierarchy: primary, secondary, supporting
- Draft headlines, headers, and copy
- Select type and dimensions for primary channel

## Phase 3: Wireframing
- 2-3 low-fidelity layout options
- Test information flow and reading path
- Verify hierarchy at display size
- Stakeholder feedback before full design

## Phase 4: Visual Design
- Apply brand colors, typography, style
- Create/select iconography and illustrations
- Build data visualizations per chart best practices
- Ensure accessibility: contrast, readability
- Multiple versions if needed (social, print, web)
