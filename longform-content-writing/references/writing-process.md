# Long-Form Writing Process

## Research Phase

### Source Hierarchy
1. **Primary research** — Surveys, interviews, proprietary data (highest value)
2. **Academic research** — Peer-reviewed studies, journals
3. **Industry reports** — Analyst reports, market research
4. **Expert opinions** — Thought leader quotes, interviews
5. **News and trends** — Recent developments, market shifts

### Research Organization
- Use reference management tool (Zotero, Mendeley)
- Categorize sources by section of content
- Rate source credibility (1-5)
- Extract key quotes and statistics
- Track source URLs for citation

---

## Outlining Techniques

### Hierarchical Outline
1. Define 5-8 major sections
2. Add 3-5 subsections per section
3. Note key points for each subsection
4. Identify where data/visuals will appear
5. Plan transitions between sections
6. Mark where CTAs will be placed

### Content Mapping
Map each section to:
- Reader question it answers
- Key data points to include
- Visual element needed
- Internal/external links
- CTA or conversion element

## Writing Techniques

### Opening Hooks
- **Startling statistic**: "87% of companies report..."
- **Bold statement**: "Traditional marketing is dead."
- **Question**: "What if you could reduce costs by 40%?"
- **Story/anecdote**: Brief narrative illustrating the problem
- **Contrarian view**: Challenge conventional wisdom

### Section Writing Framework
1. **Hook** — First sentence grabs attention
2. **Context** — Why this matters
3. **Content** — Core information, data, analysis
4. **Example** — Illustrate with case or scenario
5. **Transition** — Bridge to next section

### Maintaining Reader Engagement
- Vary sentence length (short for emphasis, long for explanation)
- Use second person ("you") to address reader directly
- Include practical takeaways throughout
- Break up text with subheadings, bullets, and visuals
- Use storytelling elements (characters, conflict, resolution)

---

## Editing Process

### Four-Pass Editing
1. **Structural edit** — Flow, organization, completeness, argument strength
2. **Line edit** — Clarity, concision, word choice, voice consistency
3. **Copy edit** — Grammar, spelling, punctuation, style guide compliance
4. **Proofread** — Final typos, formatting, link verification

### Self-Editing Checklist
- [ ] Does every section serve the central thesis?
- [ ] Are claims supported with evidence?
- [ ] Is jargon defined or eliminated?
- [ ] Do transitions flow naturally?
- [ ] Are visuals referenced in text?
- [ ] Do CTAs feel natural, not forced?
- [ ] Is the conclusion actionable?
