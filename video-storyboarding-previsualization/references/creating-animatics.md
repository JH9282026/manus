### Creating Animatics

**Definition:**
An animatic is a filmed storyboard with timing, basic movement, and audio.

**Process:**
1. Import storyboard panels
2. Arrange in timeline
3. Set duration for each panel
4. Add camera moves (zoom, pan)
5. Add temp audio (dialogue, music, SFX)
6. Add temp transitions
7. Export video

**Purpose:**
- Test pacing and timing
- Present to stakeholders
- Guide editing approach
- Music synchronization
