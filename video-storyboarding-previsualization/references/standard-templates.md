### Standard Templates

**Basic Format:**
```
+-------------------+
|                   |
|   Frame Area     |
|   (16:9 ratio)   |
|                   |
+-------------------+
Scene: ___  Shot: ___
Action: ______________
________________________
Camera: _______________
Duration: _____________
Audio: ________________
```

**Multiple Panels Per Page:**
- 3x3 grid (9 panels)
- 2x4 grid (8 panels)
- 3x2 grid (6 panels)
- Single panel (presentation)
