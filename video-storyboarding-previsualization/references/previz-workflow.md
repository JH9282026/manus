### Previz Workflow

**Stage 1: Asset Preparation**
- Import or build environments
- Create character stand-ins
- Add prop placeholders
- Establish scale

**Stage 2: Camera Work**
- Block camera positions
- Set up shots
- Plan movements
- Match real camera specs

**Stage 3: Animation**
- Character blocking
- Action timing
- Interaction choreography
- Performance reference

**Stage 4: Editing**
- Assemble sequence
- Test timing
- Add temp audio
- Refine pacing

**Stage 5: Output**
- Export video
- Generate reports
- Extract camera data
- Create documentation
