- Consistent resolution

- Format compatibility

**Animatic Assembly:**
1. Import frames in sequence
2. Set initial timing
3. Add camera moves
4. Include temp audio
5. Refine pacing
6. Add transitions
7. Export for review

- Track changes systematically
- Update source files

- Relink in animatic
