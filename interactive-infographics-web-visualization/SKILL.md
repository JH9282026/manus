# Interactive Infographics & Web Visualization

Create dynamic, web-based infographics with interactive data exploration through clicks, hovers, scrolls, and filters.

## Overview

This skill covers interactive design patterns, JavaScript visualization libraries, scrollytelling, responsive design, accessibility, and performance for web-based data experiences.

## Interaction Selection

| Need | Type | Implementation | Reference |
|------|------|---------------|-----------|
| Details on demand | Hover/Tooltip | CSS + JS events | `/references/interactivity-design.md` |
| Drill into data | Click/Tap | State management | `/references/interactivity-design.md` |
| Focus subset | Filtering | Dynamic binding | `/references/interactivity-design.md` |

## Library Guide

| Library | Best For | Curve | Size |
|---------|---------|-------|------|
| D3.js | Custom complex | Steep | 80KB |
| Chart.js | Standard charts | Low | 60KB |
| Plotly.js | Scientific | Medium | 3MB |
| Recharts | React apps | Low | 50KB |
| Leaflet | Maps | Low | 40KB |

## Accessibility
- Keyboard navigation for all interactive elements
- Screen reader descriptions
- Respect prefers-reduced-motion
- 44x44px minimum touch targets
