# Interactivity Design

## Hover/Tooltip
- Reveal values, labels, context on cursor position
- Position intelligently; avoid obscuring elements
- Touch alternative for mobile (tap to reveal)

## Click/Tap
- Drill from overview to detail
- Toggle data series visibility
- Select and highlight for comparison
- Clear visual feedback for selected state

## Filtering
- Multi-criteria with dynamic updates
- Show active filters; "reset all" button
- Indicate empty results

## Scrollytelling
- Trigger changes at scroll positions
- Pin visualization while text scrolls
- Progressive reveal building complexity
- Provide "skip to end" option
- Accessible without scroll interaction

## Animation
- 300-500ms transitions for state changes
- Maintain object constancy
- Stagger multiple elements (50-100ms)
- Allow disabling (prefers-reduced-motion)
- Purposeful: clarify, not decorate
