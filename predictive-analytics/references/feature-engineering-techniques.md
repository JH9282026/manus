# Feature Engineering Techniques

Advanced methods for creating powerful features from raw data.

## Numeric Transformations

Scaling, normalization, log transforms, polynomial features, binning strategies.

## Categorical Encoding

One-hot, label, target, frequency encoding, and handling high-cardinality categories.

## Time-Based Features

Date parts, cyclical encoding, lag features, rolling statistics, time since events.

## Text and NLP Features

TF-IDF, word embeddings, sentiment analysis, text statistics.
