# Microsoft Advertising API Integration

Programmatically manage Microsoft Advertising campaigns using Bing Ads API for automation and custom integrations.

## Overview

This skill covers Bing Ads API authentication, campaign management, bulk operations, and reporting automation. Use it for managing large Microsoft Ads accounts, building custom tools, and integrating with external systems.

## API Access

### Requirements

- **Microsoft Advertising Account**
- **Developer Token** (free, instant approval)
- **OAuth 2.0 Credentials**
- **API Version**: v13 (current)

### API Services

| Service | Purpose |
|---------|---------|
| **Campaign Management** | Create/update campaigns, ad groups, ads |
| **Bulk Service** | Large-scale operations (CSV/TSV) |
| **Reporting** | Performance data extraction |
| **Ad Insight** | Keyword research, bid suggestions |

## Common Use Cases

| Use Case | Complexity | Value |
|----------|------------|-------|
| **Automated Reporting** | Low | High |
| **Bulk Campaign Creation** | Medium | High |
| **Bid Automation** | Medium | High |
| **Keyword Management** | Medium | Medium |
