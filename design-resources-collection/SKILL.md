## Inspiration Platforms
### Primary Sources

#### Dribbble
- **URL**: https://dribbble.com
- **Best For**: UI design, visual concepts, micro-interactions, component ideas
- **Search Tips**:
  - Use specific queries: "SaaS dashboard", "e-commerce checkout", "mobile app onboarding"
  - Filter by color for palette inspiration
  - Save shots to collections for moodboards
- **Key Categories**:
  - Web Design: https://dribbble.com/tags/web-design
  - Mobile: https://dribbble.com/tags/mobile
  - Dashboard: https://cdn.dribbble.com/userupload/33966818/file/original-b1ba8d3b8dc266432777e893158a5907.png?format=webp&resize=400x300&vertical=center
  - Landing Page: https://dribbble.com/tags/landing-page

#### Behance
- **URL**: https://behance.net
- **Best For**: Complete case studies, branding projects, detailed process
- **Search Tips**:
  - Look for "Case Study" in titles for in-depth process
  - Filter by "Curated" for quality

#### Awwwards
- **URL**: https://awwwards.com
- **Best For**: Cutting-edge web design, animation inspiration, technical excellence
- **Key Sections**:
  - Sites of the Day: https://awwwards.com/websites/
  - Nominees: https://awwwards.com/websites/nominees/
  - Collections: https://awwwards.com/collections/

#### Land-book
- **URL**: https://land-book.com
- **Best For**: Landing page design, marketing pages, hero sections

#### Mobbin
- **URL**: https://mobbin.com
- **Best For**: Mobile app patterns, real-world app screenshots, UX flows
- **Categories**: iOS, Android, Web

#### Screenlane
- **URL**: https://screenlane.com
- **Best For**: Mobile UI patterns organized by screen type
