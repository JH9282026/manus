# Pinterest Ads Creative Formats

Master all Pinterest ad formats from Standard Pins to AR Try-On experiences for maximum visual impact and engagement.

## Overview

Pinterest offers eight distinct ad formats, each optimized for different marketing objectives and user experiences. This skill covers technical specifications, use cases, creative best practices, and optimization strategies for all Pinterest ad types.

## Ad Format Selection Guide

| Format | Best For | Requirements | Objectives |
|--------|----------|--------------|------------|
| Standard Pins | Product promotion, traffic | Single image | All |
| Video Pins | Storytelling, engagement | Video 4-180s | Awareness, views |
| Idea Ads | Tutorials, multi-step | 2-20 pages | Engagement |
| Carousel Ads | Collections | 2-5 images | Conversions |
| Shopping Ads | E-commerce | Product catalog | Sales |
| Collection Ads | Lifestyle inspiration | Hero + 3-25 products | Conversions |
| AR Try-On | Virtual testing | 3D models | Engagement |
| Quiz Ads | Personalization | 2-3 questions | Engagement |

## Standard Pins (Promoted Pins)

The foundational Pinterest ad format - visually identical to organic Pins but promoted to targeted audiences.

### Specifications

| Element | Requirement |
|---------|-------------|
| File Type | .PNG or .JPEG |
| Aspect Ratio | 2:3 (recommended) |
| Recommended Size | 1000 x 1500 px |
| Max File Size | 20 MB |
| Headline | Up to 100 characters |
| Description | Up to 500 characters |
| Destination URL | Required |

### Use Cases

- **Product Promotion**: Single product showcase
- **Traffic Generation**: Drive clicks to blog posts, articles
- **Brand Awareness**: Introduce brand or campaign
- **Evergreen Content**: Long-lasting promotional content

### Creative Best Practices

**Visual Design**:
- Use vertical format (2:3 ratio) for maximum screen real estate
- High-quality, well-lit product photography
- Minimal text overlay (20% or less of image)
- Brand logo in corner (subtle, not dominant)
- Clear focal point (product, person, key element)

**Color Strategy**:
- Bright, saturated colors perform better
- Contrast with Pinterest's white background
- Consistent brand color palette
- Test warm vs. cool tones for your audience

**Text Elements**:
- Clear, benefit-focused headline
- Keyword-rich description for search visibility
- Strong call-to-action ("Shop Now", "Learn More", "Get Inspired")
- Include price if promoting products

### Optimization Tips

- Test multiple image variations (3-5 per ad group)
- Refresh creative every 4-6 weeks to prevent fatigue
- Use lifestyle imagery alongside product shots
- Include people using products for relatability
- A/B test text overlay vs. clean images

## Video Pins

Autoplay video ads that capture attention in feeds and search results.

### Specifications

| Element | Requirement |
|---------|-------------|
| File Type | .MP4, .MOV, or .M4V |
| Aspect Ratio | 1:1, 2:3, or 9:16 |
| Recommended Size | 1080 x 1920 px (2:3) |
| Duration | 4 seconds to 15 minutes (optimal: 6-15 seconds) |
| Max File Size | 2 GB |
| Min Resolution | 240p |
| Encoding | H.264 or H.265 |
| Audio | Optional (autoplay is muted) |
| Headline | Up to 100 characters |
| Description | Up to 500 characters |

### Video Types

**Short-Form (4-15 seconds)**:
- Quick product demos
- Before/after transformations
- Recipe highlights
- Fashion try-ons
- Attention-grabbing teasers

**Long-Form (30+ seconds)**:
- Detailed tutorials
- Brand stories
- Product reviews
- Educational content
- Behind-the-scenes

### Creative Best Practices

**First 3 Seconds**:
- Hook viewer immediately (visual surprise, question, bold statement)
- Show product or key benefit
- Use motion and dynamic visuals
- No slow intros or logos

**Visual Storytelling**:
- Design for sound-off viewing (captions, text overlays)
- Use bold, readable text (large font, high contrast)
- Show, don't tell (visual demonstrations)
- Maintain consistent branding throughout

**Technical Quality**:
- Shoot in vertical format (9:16 or 2:3)
- Ensure good lighting and focus
- Smooth transitions and editing
- Optimize file size for fast loading
- Include captions for accessibility

### Optimization Tips

- Front-load value (key message in first 3 seconds)
- Test different video lengths
- Use native Pinterest video (not YouTube links)
- Include clear CTA at end
- Monitor completion rate and optimize accordingly

## Idea Ads

Multi-page interactive content combining images, videos, text, and stickers.

### Specifications

| Element | Requirement |
|---------|-------------|
| Pages | 2-20 |
| File Type | Video (.MP4, .MOV, .M4V) or Images (.JPEG, .PNG, .BMP, .TIFF, .WEBP) |
| Aspect Ratio | 9:16 |
| Recommended Size | 1080 x 1920 px |
| Duration per Page | 60 seconds max (ideal: 15-30 seconds) |
| Video File Size | Max 100 MB per video |
| Image File Size | Max 20 MB per image |
| Title | 100 characters max |
| Description | 250 characters max per frame |
| Clickable Links | Supported |

### Use Cases

- **Tutorials**: Step-by-step how-tos
- **Recipes**: Cooking instructions with ingredients
- **Guides**: Travel itineraries, shopping guides
- **Storytelling**: Brand narratives, customer stories
- **Product Showcases**: Multi-angle product views
- **Influencer Collaborations**: Sponsored content

### Creative Best Practices

**Structure**:
- **Page 1**: Hook and introduction
- **Pages 2-N**: Step-by-step content
- **Final Page**: CTA and destination link

**Content Flow**:
- Logical progression (chronological, difficulty-based)
- Consistent visual style across pages
- Mix of images and videos for variety
- Use stickers and text overlays for emphasis

**Engagement Elements**:
- Product tags for shoppable content
- Topic tags for discoverability
- Paid partnership tags for transparency
- Interactive stickers (polls, questions)

### Optimization Tips

- Keep total length under 10 pages for higher completion
- Use video for key steps, images for supporting content
- Include brand logo on each page
- Test different narrative structures
- Monitor page-by-page drop-off rates

## Carousel Ads

Swipeable multi-image ads showcasing multiple products or features.

### Specifications

| Element | Requirement |
|---------|-------------|
| Cards | 2-5 images |
| File Type | .PNG or .JPEG |
| Aspect Ratio | 1:1 or 2:3 |
| Recommended Size | 1000 x 1000 px (1:1) or 1000 x 1500 px (2:3) |
| Max File Size | 20 MB per card (32 MB in some sources) |
| Headline | Up to 100 characters per card |
| Description | Up to 500 characters |
| Destination URL | Optional per card (can vary) |

### Use Cases

- **Product Collections**: Related items (outfit components, room decor)
- **Product Variations**: Different colors, styles, sizes
- **Feature Highlights**: Multiple product benefits
- **Step-by-Step Guides**: Sequential instructions
- **Before/After**: Transformation sequences
- **Storytelling**: Visual narrative across cards

### Creative Best Practices

**Card Design**:
- Consistent visual style across all cards
- Each card should stand alone (users may not see all)
- Clear differentiation between cards
- Logical sequence (left to right)

**Content Strategy**:
- **Card 1**: Hero product or main message
- **Cards 2-4**: Supporting products or details
- **Card 5**: CTA or brand message

**Visual Cohesion**:
- Same background or theme
- Consistent lighting and color grading
- Unified brand elements
- Complementary products or features

### Optimization Tips

- Test 2-card vs. 5-card carousels
- Put best-performing image first
- Use unique URLs per card to track clicks
- Test 1:1 vs. 2:3 aspect ratios
- Monitor swipe-through rate

## Shopping Ads

Dynamic product ads pulled from your catalog with automatic updates.

### Specifications

| Element | Requirement |
|---------|-------------|
| Product Feed | .CSV, .TSV, or .XML |
| File Type | .JPEG or .PNG |
| Aspect Ratio | 2:3 (recommended) |
| Min Image Size | 1000 x 1500 px |
| Product Details | Title, description, price, availability |
| Headline | Up to 100 characters |
| Description | Up to 500 characters |

### Use Cases

- **Dynamic Retargeting**: Show products users viewed
- **Product Catalog Promotion**: Automatically promote inventory
- **Personalized Recommendations**: AI-driven product suggestions
- **Seasonal Campaigns**: Promote relevant products automatically

### Product Feed Requirements

**Required Fields**:
- `id`: Unique product identifier
- `title`: Product name
- `description`: Product details
- `link`: Product page URL
- `image_link`: Product image URL
- `price`: Product price with currency
- `availability`: In stock, out of stock, preorder

**Recommended Fields**:
- `brand`: Brand name
- `condition`: New, used, refurbished
- `product_type`: Category hierarchy
- `google_product_category`: Google taxonomy
- `sale_price`: Discounted price
- `age_group`: Adult, kids, infant
- `gender`: Male, female, unisex

### Creative Best Practices

**Product Images**:
- High-quality, professional photography
- Clean white or neutral backgrounds
- Multiple angles available in feed
- Lifestyle images for context
- Consistent image style across catalog

**Product Data**:
- Accurate, up-to-date pricing
- Detailed, keyword-rich descriptions
- Correct availability status
- Complete product attributes

### Optimization Tips

- Update feed daily for price/availability accuracy
- Use high-quality images (min 1000 x 1500 px)
- Optimize product titles with keywords
- Implement AR features ("Shop the Look")
- Segment campaigns by product category

## Collection Ads

Immersive full-screen shopping experience with hero image/video and product grid.

### Specifications

| Element | Requirement |
|---------|-------------|
| Hero Asset | 1 image or video |
| Product Images | 3-25 products |
| Hero Aspect Ratio | 1:1 or 2:3 |
| Product Aspect Ratio | 1:1 |
| Hero File Type | .JPEG, .PNG (image); .MP4 (video) |
| Hero File Size | Max 10 MB (image); Max 2 GB (video) |
| Product File Size | Standard image requirements |
| Destination URL | Required per product |
| Headline | Up to 100 characters |
| Platform | Mobile only |

### Use Cases

- **Lifestyle Branding**: Products in real-world context
- **Lookbooks**: Fashion collections, seasonal catalogs
- **Room Inspiration**: Home decor in styled spaces
- **Gift Guides**: Curated product selections
- **Themed Collections**: Holiday, occasion-based groupings

### Creative Best Practices

**Hero Asset**:
- Lifestyle imagery showing products in use
- Aspirational, high-quality visuals
- Multiple products visible in context
- Video for added engagement (optional)

**Product Selection**:
- Complementary items (complete the look)
- Varied price points
- Best-sellers and new arrivals
- Thematically cohesive

**Layout Strategy**:
- Hero image sets the mood/theme
- Products below enable immediate shopping
- Seamless transition from inspiration to purchase

### Optimization Tips

- Test different hero assets (lifestyle vs. product-focused)
- Optimize product order (best-sellers first)
- Use video hero for higher engagement
- Ensure fast-loading product images
- Monitor click-through by product position

## AR Try-On Pins

Augmented reality ads allowing virtual product testing via camera.

### Specifications

| Element | Requirement |
|---------|-------------|
| Supported Products | Lipstick, eyewear, select beauty/home items |
| File Type | 3D models (.USDZ for iOS, .GLB for Android) |
| Preview Image | Min 1000 x 1000 px |
| Product Tags | Required |
| Compatibility | Mobile devices with AR support |
| Creation Tool | Lens Studio or AR partner platforms |

### Use Cases

- **Beauty Products**: Lipstick, eyeshadow, foundation
- **Eyewear**: Glasses, sunglasses
- **Home Decor**: Furniture placement, wall art
- **Fashion Accessories**: Jewelry, watches

### Creative Best Practices

**3D Model Quality**:
- Accurate color representation
- Realistic textures and materials
- Optimized file size for fast loading
- Multiple product variations available

**User Experience**:
- Clear instructions for AR activation
- Intuitive controls (switch products, capture photo)
- Seamless transition to purchase
- Social sharing enabled

### Optimization Tips

- Offer multiple shades/styles for try-on
- Include product details and pricing in AR view
- Enable direct purchase from AR experience
- Promote AR feature in ad copy
- Track AR engagement and conversion rates

## Quiz Ads

Interactive multiple-choice quizzes leading to personalized recommendations.

### Specifications

| Element | Requirement |
|---------|-------------|
| Questions | 2-3 |
| File Type | Video (.MP4, .MOV, .M4V) or Image (.BMP, .PNG, .JPG, .TIFF, .WEBP) |
| File Size | Max 32 MB (video), 20 MB (image) |
| Duration | Min 3 sec, max 60 sec (video) |
| Aspect Ratio | 2:3 (title and results pins) |
| Title | 100 characters max |
| Text Overlay | 10 words max |
| Questions | 96 characters max |
| Answers | 48 characters max |
| Results Title | 100 characters max |
| Results Description | 500 characters max |
| Platform | Mobile only |

### Use Cases

- **Product Recommendations**: "Find your perfect shade"
- **Style Quizzes**: "What's your design aesthetic?"
- **Personalization**: "Build your ideal routine"
- **Engagement**: "Which product matches your personality?"

## Creative Best Practices (All Formats)

### Universal Design Principles
- High-resolution images (minimum 1000 px width)
- Consistent brand aesthetic, mobile-optimized
- Minimal text overlay (20% or less), large readable fonts
- Clear CTAs aligned with campaign objective

### Platform-Specific Considerations
- Pinterest users are planning-focused (45-day horizon)
- Inspirational over promotional approach
- Helpful, educational content performs best

### Testing Framework
Test one variable at a time: image style, color schemes, text overlay, aspect ratios, headlines, CTAs, video vs. static. Run for 7-14 days minimum with 100+ conversions for statistical significance.
