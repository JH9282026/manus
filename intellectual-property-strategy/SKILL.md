# Intellectual Property Strategy

Develop comprehensive IP strategies that protect innovations, brands, and creative assets while enabling monetization and competitive advantage.

## Overview

Intellectual property is often a company's most valuable asset class — yet many organizations underinvest in IP strategy. A proactive IP approach protects market position, generates licensing revenue, strengthens M&A valuation, and deters competitors. This skill covers the four pillars of IP (patents, trademarks, copyrights, trade secrets), portfolio strategy, monetization frameworks, and competitive landscape analysis.

## Patent Portfolio Strategy

### Patent Decision Framework

| Factor | File Patent | Keep as Trade Secret |
|--------|------------|---------------------|
| Can competitors reverse-engineer? | Yes → Patent | No → Trade Secret |
| Is it independently discoverable? | Yes → Patent | No → Trade Secret |
| Will it be visible in the product? | Yes → Patent | No → Trade Secret |
| Do you need defensive publication? | Yes → Patent or publish | — |
| Is enforcement feasible? | Yes → Patent | No → Consider trade secret |
| Cost-benefit positive? | Revenue or defense justifies cost | Cost exceeds benefit |

### Patent Filing Strategy

| Strategy | Description | When to Use |
|----------|-------------|-------------|
| Offensive filing | File to assert against competitors | Core differentiating technology |
| Defensive filing | File to prevent competitors from patenting | Key operational methods |
| Defensive publication | Publish without patenting to create prior art | Block competitors without patent cost |
| Continuation strategy | File continuations for broader/narrower claims | Evolving technology areas |
| International (PCT) | File internationally via Patent Cooperation Treaty | Products sold in multiple countries |
| Design patents | Protect distinctive product appearance | Consumer products, UI design |

### Patent Application Process

| Stage | Timeline | Cost | Activity |
|-------|----------|------|----------|
| Prior art search | 2–4 weeks | $2K–$5K | Search existing patents and publications |
| Provisional filing | 1–2 months | $3K–$8K | File provisional for 12-month priority date |
| Non-provisional filing | Within 12 months of provisional | $10K–$25K | Full application with claims |
| Examination | 18–36 months | $5K–$15K (office actions) | USPTO examination and responses |
| Grant | After examination | $2K–$5K | Issue fee and maintenance fees |
| Maintenance | Years 3.5, 7.5, 11.5 post-grant | $1.6K–$7.4K per fee | Required to keep patent active |

## Trademark Strategy

### Trademark Strength Spectrum

| Strength Level | Type | Example | Protectability |
|---------------|------|---------|----------------|
| Fanciful (strongest) | Invented word | KODAK, XEROX | Easiest to register and enforce |
| Arbitrary | Real word, unrelated use | APPLE (computers) | Strong protection |
| Suggestive | Suggests quality/characteristic | NETFLIX (internet flicks) | Moderate — registrable |
| Descriptive (weak) | Describes the product/service | "Cold & Creamy" (ice cream) | Requires secondary meaning |
| Generic (none) | Common product name | "Computer" for computers | Cannot be trademarked |

### Trademark Registration Process

| Step | Timeline | Action |
|------|----------|--------|
| 1. Clearance search | 2–4 weeks | Search USPTO TESS, common law, domain names |
| 2. File application | 1 week | File via TEAS (USPTO) with specimens of use |
| 3. Examination | 3–6 months | USPTO examiner reviews, may issue office actions |
| 4. Publication | 30 days | Published for opposition in Official Gazette |
| 5. Registration | 2–3 months after publication | Registration certificate issued |
| 6. Maintenance | Years 5–6, 9–10, then every 10 | File declaration of use and renewal |

## Trade Secret Protection Framework

| Protection Measure | Implementation | Priority |
|-------------------|---------------|----------|
| Identify trade secrets | Inventory all confidential business information | Critical |
| Restrict access | Need-to-know basis, role-based access controls | Critical |
| NDA/Non-compete agreements | All employees, contractors, and partners | Critical |
| Physical security | Locked offices, secure servers, clean desk policy | High |
| Digital security | Encryption, DLP tools, access logging | High |
| Employee training | Annual trade secret awareness training | High |
| Exit procedures | Return of materials, reminder of obligations at departure | High |
| Mark documents | "Confidential" and "Trade Secret" labels | Medium |

## Freedom-to-Operate (FTO) Analysis

| Step | Activity | Output |
|------|----------|--------|
| 1. Define product/process scope | Document what your product does and how | Technical specification |
| 2. Identify relevant patent landscape | Search patents in your technology space | Patent list (100–1,000+) |
| 3. Screen for relevance | Filter to patents that could cover your product | Short list (10–50) |
| 4. Analyze claims | Compare your product features to patent claims | Claim chart per patent |
| 5. Assess risk | Rate each patent: high/medium/low infringement risk | Risk matrix |
| 6. Develop strategy | Design around, license, challenge, or accept risk | Action plan |

## Best Practices

- **File early, file often** — provisional patents are relatively cheap and secure your priority date
- **Choose strong trademarks** — fanciful and arbitrary marks are 10× easier to protect than descriptive ones
- **Protect trade secrets actively** — a trade secret loses protection the moment it becomes public
- **Conduct FTO before product launch** — discovering infringement after launch is 10× more expensive
- **Monitor the competitive landscape** — track competitor patent filings and trademark registrations quarterly
- **Budget for IP maintenance** — patents and trademarks require ongoing fees to remain active
