### Balance and Visual Weight

**Elements of Visual Weight:**
- Size (larger = heavier)
- Color (saturated = heavier)
- Position (edge = heavier)
- Complexity (detailed = heavier)
- Isolation (alone = heavier)

**Balanced Composition:**
- Elements counterweight each other
- Formal balance (symmetrical)
- Informal balance (asymmetrical)
- Consider all frame elements
