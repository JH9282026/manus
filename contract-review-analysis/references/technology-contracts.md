# Technology Contracts

Key provisions and considerations for technology agreements.

---

## SaaS Agreements

### Critical Provisions

**Service Level Agreements (SLAs)**
- Uptime commitment (99.9% = 8.76 hrs downtime/year)
- Measurement methodology
- Exclusions (maintenance windows, force majeure)
- Remedies (service credits vs. termination rights)

**Data Security**
- Security standards (SOC 2, ISO 27001)
- Encryption requirements
- Incident notification timelines
- Audit rights

**Data Ownership & Portability**
- Customer data ownership confirmation
- Data export capabilities
- Format specifications
- Transition assistance

**Termination Rights**
- Termination for convenience
- Cure periods for breach
- Data return obligations
- Survival provisions

---

## Software License Agreements

### Key Terms
- License type (perpetual vs. subscription)
- Scope of use (users, sites, deployments)
- Modification and derivative rights
- Audit provisions

### Common Issues
- Unclear scope expansion
- Onerous audit requirements
- Excessive indemnification
- Restrictive assignment

---

## Data Processing Agreements (DPA)

### Required Elements (GDPR)
- Processing instructions
- Confidentiality obligations
- Security measures
- Sub-processor requirements
- Audit rights
- Deletion/return obligations
- Breach notification

---

## Negotiation Priorities

### Must-Have
- Data ownership clarity
- Reasonable SLA credits
- Termination for convenience
- Data portability

### Should-Have
- Audit rights
- Price increase caps
- Enhanced security terms
- Broader indemnification
