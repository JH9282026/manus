# Twitch Ads Compliance Checklist

Complete compliance requirements for Twitch advertising campaigns, including creative guidelines, technical specifications, and regulatory requirements.

---

## Twitch Advertising Guidelines

### Required Creative Elements

- [ ] Advertiser logo or brand name clearly visible
- [ ] Logo appears within first 3 seconds of video ads
- [ ] Logo present on end card (minimum 3 seconds)
- [ ] Brand name mentioned verbally (video ads)
- [ ] All claims are accurate and verifiable
- [ ] Content complies with Twitch Community Guidelines
- [ ] Age-appropriate for platform (13+ minimum)
- [ ] Clear, not misleading or deceptive
- [ ] Proper disclosures for sponsored content

### Prohibited Content

**Illegal Products/Services:**
- [ ] No illegal drugs or controlled substances
- [ ] No illegal weapons or explosives
- [ ] No counterfeit goods
- [ ] No illegal services

**Restricted Products:**
- [ ] No tobacco products or vaping devices
- [ ] No recreational drugs or drug paraphernalia
- [ ] No weapons, ammunition, or explosives (some exceptions with approval)

**Adult Content:**
- [ ] No adult/sexual content or services
- [ ] No dating services with sexual focus
- [ ] No explicit imagery or language

**Gambling:**
- [ ] No online gambling (restrictions vary by region)
- [ ] No casino advertising (without approval)
- [ ] Lottery and sweepstakes require official rules

**Hate Speech/Discrimination:**
- [ ] No hate speech or discriminatory content
- [ ] No content targeting protected classes
- [ ] No promotion of violence or harm

**Shocking/Disturbing Content:**
- [ ] No graphic violence or gore
- [ ] No disturbing or shocking imagery
- [ ] No content designed to shock or scare

**Deceptive Practices:**
- [ ] No misleading claims
- [ ] No false advertising
- [ ] No clickbait or deceptive tactics
- [ ] No malware, phishing, or scams

### Restricted Content (Approval Required)

**Alcohol:**
- [ ] Age-gated (21+ in US, varies by region)
- [ ] No appeal to minors
- [ ] Responsible consumption messaging
- [ ] Regional restrictions apply
- [ ] Requires pre-approval

**Political Advertising:**
- [ ] Proper disclosures required
- [ ] "Paid for by" disclaimers
- [ ] Compliance with election laws
- [ ] Regional restrictions
- [ ] Requires pre-approval

**Financial Services:**
- [ ] Regulatory compliance (SEC, FINRA, etc.)
- [ ] Risk disclosures required
- [ ] No guaranteed returns claims
- [ ] Proper licensing verification
- [ ] Requires pre-approval

**Healthcare Products:**
- [ ] FDA compliance (if applicable)
- [ ] Substantiation for health claims
- [ ] No miracle cure claims
- [ ] Proper disclaimers
- [ ] Requires pre-approval

**Contests/Sweepstakes:**
- [ ] Official rules provided
- [ ] No purchase necessary (if sweepstakes)
- [ ] Odds disclosure
- [ ] Eligibility requirements
- [ ] Void where prohibited
- [ ] Requires pre-approval

## Technical Compliance

### Video Ad Specifications

**Premium Video:**
- [ ] Resolution: 1920x1080 minimum
- [ ] Bitrate: 4,000-10,000 kbps
- [ ] Audio: -9dB peak level
- [ ] Format: H.264 (MP4)
- [ ] Frame Rate: 24-30fps
- [ ] Length: 6s, 15s, 20s, 30s, or 60s
- [ ] File size: Under 500MB (site served)

**Vertical Video:**
- [ ] Resolution: 1080x1920
- [ ] Aspect Ratio: 9:16
- [ ] Bitrate: 4mbps minimum (site served)
- [ ] Audio: -9dB peak level
- [ ] Format: H.264, MPEG-2, or MPEG-4
- [ ] Length: 6s, 15s, 20s, or 30s
- [ ] File size: Under 500MB (site served)

**Third-Party Serving (VAST):**
- [ ] VAST 2.0 or 3.0 compliant
- [ ] All required transcodes provided
- [ ] SSL-compliant (HTTPS)
- [ ] No 4th-party tags
- [ ] No unsupported targeting parameters
- [ ] Tracking pixels SSL-compliant

### Display Ad Specifications

**General Requirements:**
- [ ] IAB standard sizes
- [ ] File size under 150KB per creative
- [ ] Format: JPEG, PNG, or GIF
- [ ] Animation: Maximum 30 seconds loop
- [ ] SSL-compliant delivery (HTTPS)
- [ ] Clickable CTA functional
- [ ] Logo prominently featured

**Specific Sizes:**
- [ ] Super Leaderboard: 970x90
- [ ] Medium Rectangle: 300x250
- [ ] Leaderboard: 728x90
- [ ] Half Page: 300x600
- [ ] Wide Skyscraper: 160x600

### SSL Compliance

- [ ] All ad components served via HTTPS
- [ ] Third-party tracking pixels use HTTPS
- [ ] No mixed content (HTTP and HTTPS)
- [ ] All vendor tags SSL-compliant
- [ ] Landing pages use HTTPS
- [ ] All assets (images, videos, scripts) use HTTPS

## FTC Compliance (Sponsored Content)

### Disclosure Requirements

**When Disclosure Required:**
- [ ] Payment received for content
- [ ] Free products received
- [ ] Any material connection with brand
- [ ] Affiliate relationships
- [ ] Employee or family member of brand

**How to Disclose:**
- [ ] Clear and conspicuous disclosure
- [ ] #ad or #sponsored in stream title
- [ ] Verbal disclosure during stream
- [ ] Twitch "Sponsored" label enabled
- [ ] Disclosure in channel panels
- [ ] Can't be buried or hidden

**Timing of Disclosure:**
- [ ] At beginning of sponsored content
- [ ] Periodically throughout (every 15-30 min recommended)
- [ ] Before any call-to-action
- [ ] In VOD description
- [ ] In social media posts about sponsored content

**What Doesn't Count:**
- [ ] Not just "Thanks to [Brand]" (must say "sponsored")
- [ ] Not buried in description
- [ ] Not hidden in hashtag list
- [ ] Not vague or ambiguous language

### Creator Responsibilities

- [ ] Creators understand disclosure requirements
- [ ] Provide clear disclosure guidelines to creators
- [ ] Monitor creator compliance
- [ ] Provide examples of compliant disclosures
- [ ] Include disclosure requirements in contracts
- [ ] Educate creators on FTC rules

## Brand Safety

### Content Category Targeting

**Review Targeting:**
- [ ] Ensure ads appear in brand-appropriate content
- [ ] Exclude mature content categories (if applicable)
- [ ] Avoid controversial streamers/content
- [ ] Align with brand values
- [ ] Consider audience sensitivities

**Exclusion Lists:**
- [ ] Create list of prohibited content categories
- [ ] Exclude specific games/streamers if needed
- [ ] Update exclusion lists regularly
- [ ] Monitor where ads actually appear
- [ ] Use Content Transparency Reports

### Creator Partnerships

**Vetting Process:**
- [ ] Review creator's content history
- [ ] Check for controversial past behavior
- [ ] Verify compliance with Twitch ToS
- [ ] Assess community reputation
- [ ] Confirm brand alignment

**Ongoing Monitoring:**
- [ ] Monitor creator content during partnership
- [ ] Address any brand safety concerns promptly
- [ ] Have termination clause in contracts
- [ ] Regular check-ins with creators
- [ ] Community sentiment monitoring

## Creative Approval Process

### Submission Timeline

- [ ] Submit creatives 5-7 business days before campaign start
- [ ] Allow 2-3 business days for review
- [ ] Plan for potential revision requests
- [ ] Rush approvals may not be available
- [ ] Factor in time for revisions

### Submission Requirements

**Creative Assets:**
- [ ] All video files (all lengths/formats)
- [ ] All display ad files (all sizes)
- [ ] High-resolution source files
- [ ] All required transcodes (third-party serving)

**Campaign Details:**
- [ ] Campaign start and end dates
- [ ] Targeting parameters
- [ ] Budget and pacing
- [ ] Placement preferences

**Additional Materials:**
- [ ] Landing page URL
- [ ] Third-party tags (if applicable)
- [ ] Substantiation for claims (if needed)
- [ ] Approval documentation (restricted content)

### Review Criteria

**Policy Compliance:**
- [ ] No prohibited content
- [ ] Restricted content has approval
- [ ] Proper disclosures
- [ ] Age-appropriate

**Technical Specifications:**
- [ ] Correct resolution and aspect ratio
- [ ] Bitrate within range
- [ ] Audio levels correct
- [ ] File format correct
- [ ] File size within limits

**Brand Safety:**
- [ ] Content appropriate for platform
- [ ] No controversial or offensive content
- [ ] Aligns with Twitch community values

**Quality Standards:**
- [ ] Professional production quality
- [ ] Clear audio and video
- [ ] No technical errors
- [ ] Functional CTAs and links

**User Experience:**
- [ ] Not deceptive or misleading
- [ ] Clear value proposition
- [ ] Appropriate for placement
- [ ] Landing page matches ad

### Common Rejection Reasons

**Policy Violations:**
- [ ] Prohibited content
- [ ] Missing disclosures
- [ ] Misleading claims
- [ ] Age-inappropriate content

**Technical Issues:**
- [ ] Wrong resolution or aspect ratio
- [ ] Bitrate too low or too high
- [ ] Audio levels incorrect
- [ ] File format not supported
- [ ] File size too large

**Quality Issues:**
- [ ] Poor video quality
- [ ] Bad audio quality
- [ ] Pixelated or blurry imagery
- [ ] Unprofessional production

**Missing Elements:**
- [ ] No logo or brand name
- [ ] Missing required disclosures
- [ ] Broken landing page link
- [ ] Missing substantiation for claims

### Revision Process

- [ ] Receive feedback from Twitch team
- [ ] Make required changes promptly
- [ ] Resubmit for approval
- [ ] Allow 1-2 business days for re-review
- [ ] Confirm final approval before campaign start
