# Complete Ad Extensions Guide

Comprehensive guide to all Google Ads extension types for Search campaigns with setup instructions and best practices.

---

## Extension Types Overview

| Extension | Purpose | Character Limits | Avg CTR Lift | Mobile Impact |
|-----------|---------|------------------|--------------|---------------|
| Sitelink | Additional landing pages | 25 char title, 35 char desc | +10-20% | High |
| Callout | Feature highlights | 25 characters | +5-10% | Medium |
| Structured Snippet | Categories/lists | Header + 3-10 values | +5-10% | Medium |
| Call | Phone number | Phone + country code | +5-15% | Very High |
| Location | Physical address | Auto from GMB | +10% | High |
| Price | Product/service pricing | 8 items, 25 char headers | +5-10% | Medium |
| Promotion | Special offers | 20 char occasion, 15 char discount | +10-15% | High |
| Image | Visual assets | 1.91:1 or 1:1 ratio | +5-10% | Medium |
| App | App download link | App name + link | +10-20% | Very High |
| Lead Form | In-ad lead capture | Custom form fields | +15-25% | Very High |

## Sitelink Extensions

**Purpose**: Direct users to specific pages on your website

**Setup**:
- **Sitelink Text**: 25 characters (e.g., \"Free Shipping\")
- **Description Line 1**: 35 characters (optional but recommended)
- **Description Line 2**: 35 characters (optional)
- **Final URL**: Destination page

**Best Practices**:
- Create 8-10 sitelinks (Google shows 2-6 depending on device/position)
- Use action-oriented text (\"Shop Now\", \"Get Quote\", \"Learn More\")
- Link to high-value pages (product categories, special offers, contact)
- Include descriptions for better CTR (+15-20% vs no descriptions)

**Example**:
```
Sitelink 1: \"Men's Running Shoes\"
Description 1: \"Shop 200+ styles for men\"
Description 2: \"Free shipping on all orders\"
URL: /mens-running-shoes

Sitelink 2: \"Women's Running Shoes\"
Description 1: \"Find your perfect fit\"
Description 2: \"Expert fitting available\"
URL: /womens-running-shoes
```

## Callout Extensions

**Purpose**: Highlight key features, benefits, or offers

**Setup**:
- **Callout Text**: 25 characters
- No URL (non-clickable)
- Create 6-10 callouts

**Best Practices**:
- Focus on unique selling points (\"24/7 Support\", \"Price Match Guarantee\")
- Use numbers when possible (\"5-Star Rated\", \"10,000+ Reviews\")
- Avoid redundancy with ad copy
- Update seasonally (\"Holiday Sale\", \"Back to School Deals\")

**Example Callouts**:
- \"Free Shipping & Returns\"
- \"Lifetime Warranty\"
- \"Expert Fitting Available\"
- \"Price Match Guarantee\"
- \"Trusted Since 1995\"
- \"4.8 Star Rating\"

## Structured Snippet Extensions

**Purpose**: Showcase specific aspects of products/services in a list format

**Setup**:
- **Header**: Predefined categories (Amenities, Brands, Courses, etc.)
- **Values**: 3-10 items, 25 characters each

**Available Headers**:
- Amenities, Brands, Courses, Degree Programs, Destinations, Featured Hotels
- Insurance Coverage, Models, Neighborhoods, Service Catalog, Shows, Styles, Types

**Best Practices**:
- Use 3-4 different headers per campaign
- Provide 8-10 values per header (Google shows 3-4)
- Keep values concise and parallel in structure

**Example**:
```
Header: Brands
Values: Nike, Adidas, Brooks, Asics, New Balance, Saucony, Hoka, On Running

Header: Types
Values: Trail Running, Road Running, Track Spikes, Racing Flats, Walking Shoes
```

## Call Extensions

**Purpose**: Display phone number, enable click-to-call on mobile

**Setup**:
- **Phone Number**: With country code
- **Call Reporting**: Enable for conversion tracking
- **Call Scheduling**: Optional (show only during business hours)

**Best Practices**:
- Use tracking numbers for call attribution
- Enable call reporting to track conversions
- Set ad schedule to business hours only (avoid after-hours calls)
- Use separate numbers for different campaigns (tracking)

**Mobile Impact**:
- 30-50% of mobile ad clicks may be calls (not website visits)
- Higher intent users often prefer calling
- Essential for local service businesses

## Location Extensions

**Purpose**: Show business address, phone, directions

**Setup**:
- Link Google My Business account to Google Ads
- Locations auto-populate from GMB
- Optional: Set radius targeting

**Best Practices**:
- Verify all GMB locations
- Keep GMB info updated (hours, phone, address)
- Use for local businesses or multi-location brands
- Combine with Call extensions for maximum mobile impact

**Performance**:
- +10-15% CTR for local searches
- Enables \"Get Directions\" clicks on mobile
- Shows distance from user's location

## Price Extensions

**Purpose**: Display product/service prices directly in ad

**Setup**:
- **Type**: Products, Services, Brands, Events, Locations, etc.
- **Items**: 3-8 price items
- **Header**: 25 characters
- **Description**: 25 characters
- **Price**: Amount + currency
- **Final URL**: Destination for each item

**Best Practices**:
- Show competitive prices (if you're price-competitive)
- Use \"From $X\" for variable pricing
- Update regularly (especially during sales)
- Link each item to specific product/service page

**Example**:
```
Type: Services
Item 1: \"Basic Package\" - $99 - \"Essential features included\"
Item 2: \"Pro Package\" - $199 - \"Advanced features + support\"
Item 3: \"Enterprise\" - $499 - \"Full suite + dedicated account manager\"
```

## Promotion Extensions

**Purpose**: Highlight sales, discounts, special offers

**Setup**:
- **Occasion**: Holiday, Event, or Generic
- **Promotion**: Discount percentage or amount
- **Start/End Dates**: Promotion period
- **Promo Code**: Optional

**Best Practices**:
- Use during sales periods (Black Friday, holidays)
- Set accurate start/end dates (auto-expires)
- Include promo code if applicable
- Update regularly (don't show expired promotions)

**Example**:
```
Occasion: Black Friday
Promotion: \"Up to 40% Off\"
Dates: 11/24/2026 - 11/27/2026
Promo Code: \"BF2026\"
```

## Image Extensions

**Purpose**: Add visual appeal to text ads

**Setup**:
- **Aspect Ratios**: 1:1 (square) or 1.91:1 (landscape)
- **Minimum Size**: 300x300 (square), 600x314 (landscape)
- **File Size**: Max 5MB
- **Format**: JPG, PNG

**Best Practices**:
- Use high-quality, relevant images
- Show products, services, or brand imagery
- Avoid text-heavy images (Google may disapprove)
- Test multiple images (Google auto-optimizes)

**Performance**:
- +5-10% CTR on average
- Higher impact for visual products (fashion, home goods)
- Less impact for B2B services

## App Extensions

**Purpose**: Promote app downloads directly from search ads

**Setup**:
- **App Store**: iOS App Store or Google Play
- **App ID**: From app store listing
- **Link Text**: \"Download\" or \"Install\" (auto-generated)

**Best Practices**:
- Use for mobile campaigns promoting apps
- Track app installs as conversions
- Combine with App campaigns for full-funnel strategy

---

## Lead Form Extensions

**Purpose**: Capture leads directly in the ad (no landing page visit)

**Setup**:
- **Form Fields**: Name, Email, Phone (customizable)
- **Privacy Policy**: Required
- **Thank You Message**: Post-submission message
- **CRM Integration**: Optional (Zapier, Salesforce, etc.)

**Best Practices**:
- Use for lead generation campaigns (B2B, services)
- Keep form short (3-5 fields max)
- Offer incentive (\"Get Free Quote\", \"Download Guide\")
- Set up auto-response email

**Performance**:
- +15-25% conversion rate vs landing page forms
- Lower quality leads (easier to submit)
- Best for top-of-funnel lead capture

## Extension Scheduling

**When to Schedule Extensions**:
- **Call Extensions**: Business hours only
- **Promotion Extensions**: Sale period only
- **Location Extensions**: Store hours (if applicable)

**How to Schedule**:
1. Create extension
2. Set \"Advanced Options\"
3. Choose \"Ad Schedule\"
4. Select days/hours

## Extension Performance Reporting

**Metrics to Track**:
- **Clicks**: Total clicks on extensions
- **CTR**: Click-through rate with extensions vs without
- **Conversions**: Conversions attributed to extensions
- **Cost**: Spend on extension clicks

**How to View**:
- Google Ads UI: Ads & Assets > Extensions
- API: `ad_group_extension_setting` resource

**Optimization**:
- Pause low-performing extensions (CTR <1%)
- Test new variations monthly
- Update seasonal extensions (promotions, callouts)
