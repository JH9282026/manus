# Consulting, Education, Government & Others

## Consulting
- Pyramid Principle: lead with recommendation, MECE structure
- Action titles: "Revenue grew 15%" not "Revenue Analysis"
- One message per slide; appendix for depth
- Strategy: situation -> options -> evaluation -> recommendation

## Education
- Learning objectives -> prior knowledge -> content -> active learning -> summary
- Chunk information (7+/-2 rule), visual aids supporting verbal
- Research: IMRaD for conference; poster with visual hierarchy

## Government
- Section 508: alt text, 4.5:1 contrast, keyboard navigation
- Plain language; accommodation for diverse audiences
- Policy: issue -> background -> options -> recommendation -> budget

## Pharmaceutical
- Drug development: need -> mechanism -> timeline -> data -> safety -> regulatory
- Fair balance of efficacy and safety; on-label only

## Engineering
- CAD drawings, material specs, tolerance standards
- Safety: OSHA hazards, risk matrices, PPE, emergency procedures

## Marketing
- Campaign: situation -> strategy -> concepts -> media -> measurement
- Visual-forward, emotional, concept mockups

## Nonprofit
- Donor: mission -> impact -> need -> budget -> ask
- Impact: achievements, transparency, stories, future plans
