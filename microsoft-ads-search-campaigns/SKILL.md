# Microsoft Advertising Search Campaigns

Create high-performing search campaigns on Microsoft Advertising Network (Bing, Yahoo, AOL, DuckDuckGo).

## Overview

This skill covers Microsoft Advertising Search campaign setup, keyword targeting, Responsive Search Ads, bidding strategies, and audience targeting. Use it for reaching Bing's 1+ billion monthly users with search advertising complementary to Google Ads.

## Why Microsoft Advertising?

### Key Advantages

| Advantage | Benefit |
|-----------|---------|
| **Lower CPCs** | 30-50% cheaper than Google Ads |
| **Less Competition** | Fewer advertisers bidding |
| **Older, Affluent Audience** | Higher household income, 45+ age |
| **Desktop-Heavy** | 60% desktop vs. 40% mobile |
| **Import from Google Ads** | Easy campaign migration |

### Audience Demographics

| Demographic | Microsoft Ads | Google Ads |
|-------------|---------------|------------|
| **Avg Age** | 45+ | 35+ |
| **Avg Income** | $75K+ | $60K+ |
| **Device** | 60% Desktop | 60% Mobile |
| **Education** | Higher college % | Mixed |

## Campaign Setup

### Campaign Types

| Type | Use Case | Complexity |
|------|----------|------------|
| **Search** | Standard search ads | Low |
| **Audience** | Display-like ads on MSN, Outlook | Medium |
| **Shopping** | Product ads (Merchant Center) | High |
| **Dynamic Search** | Auto-generated ads from website | Medium |

### Import from Google Ads

**One-Click Import**:
1. Link Google Ads account
2. Select campaigns to import
3. Choose import settings (bids, budgets, targeting)
4. Schedule automatic imports (daily, weekly)

**What Imports**:
- Campaigns, ad groups, keywords
- Ads (RSAs, ETAs)
- Extensions
- Negative keywords
- Budgets and bids

**What Doesn't Import**:
- Performance history
- Quality Scores
- Conversion data

## Responsive Search Ads

### Asset Requirements

| Asset | Quantity | Character Limit |
|-------|----------|-----------------|
| Headlines | 3-15 | 30 characters |
| Descriptions | 2-4 | 90 characters |
| Display URLs | 2 paths | 15 characters each |

### Best Practices

- Provide 10-15 headlines for optimization
- Include keywords in 5+ headlines
- Vary headline lengths (15-30 chars)
- Use all 4 description slots
- Avoid pinning (reduces performance)

## Keyword Targeting

### Match Types

| Match Type | Syntax | Behavior |
|------------|--------|----------|
| **Broad** | keyword | Widest reach, least control |
| **Phrase** | "keyword" | Moderate reach and control |
| **Exact** | [keyword] | Highest control, lowest reach |

### Keyword Research

**Microsoft Keyword Planner**:
- Search volume data
- Bid estimates
- Keyword suggestions
- Competitive insights

**Import from Google**:
- Use Google Keyword Planner
- Import keywords via campaign import
- Adjust bids (typically 30-50% lower)
