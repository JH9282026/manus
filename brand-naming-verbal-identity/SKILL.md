# Brand Naming & Verbal Identity

Create distinctive brand names and comprehensive verbal identity systems.

## Overview

This skill covers the strategic process of creating names for brands, products, and services, plus the complete verbal identity system including taglines, nomenclature, voice, and messaging architecture. Names are the primary verbal identifier—unlike visual elements that can evolve, names typically remain constant.

## Naming Types

| Type | Description | Examples |
|------|-------------|----------|
| Descriptive | Literally describes offering | General Motors, PayPal |
| Invented | Made-up words, no prior meaning | Google, Kodak, Xerox |
| Metaphorical | Evokes qualities through suggestion | Amazon, Nike, Apple |
| Acronym | Initials of longer name | IBM, BMW, KFC |
| Founder | Based on founder name | Ford, Disney, Hewlett-Packard |
| Geographic | References places | Bank of America, Patagonia |
| Compound | Two words combined | Facebook, YouTube, LinkedIn |
| Portmanteau | Parts of words blended | Microsoft, Pinterest, Netflix |

## Naming Criteria

| Criterion | Assessment |
|-----------|------------|
| Memorable | Easy to recall after single exposure |
| Meaningful | Conveys something about the brand |
| Protectable | Distinctive enough to trademark |
| Available | Domain, social, trademark clear |
| Scalable | Works for future growth |
| Pronounceable | Easy across target markets |
| Spellable | Intuitive to spell after hearing |

## Trademark Strength Spectrum

From strongest to weakest protection:

1. **Fanciful**: Invented words (Kodak, Xerox)
2. **Arbitrary**: Real words, no connection (Apple, Amazon)
3. **Suggestive**: Hints without describing (Netflix, Coppertone)
4. **Descriptive**: Describes product (weak, requires secondary meaning)
5. **Generic**: Common name for product (no protection)

## Naming Process

1. **Strategy & Brief**: Review positioning, define criteria
2. **Ideation**: Generate 100-500+ candidates
3. **Screening**: Eliminate obvious rejects, score against criteria
4. **Evaluation**: Deep analysis, stakeholder and customer testing
5. **Legal Clearance**: Comprehensive trademark search

## Verbal Identity Components

- **Name**: Primary identifier
- **Tagline/Slogan**: Memorable phrase capturing essence
- **Descriptor**: What you do/category definition
- **Boilerplate**: Standard company description
- **Key Messages**: Core message pillars
- **Voice & Tone**: Brand personality in communication
