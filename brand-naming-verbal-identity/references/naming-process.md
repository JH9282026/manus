# Naming Process

Detailed methodology for creating brand names.

---

## Ideation Techniques

### Brainstorming Methods

- Mind mapping from core concepts
- Word association chains
- Free writing and stream of consciousness
- Random word combinations
- Metaphor exploration

### Linguistic Exploration

**Root Words:**
- Latin (tech, pharma): Apex, Magna, Optimus
- Greek (science, innovation): Meta, Neo, Logos
- Sanskrit (wellness): Ananda, Veda

**Prefixes/Suffixes:**
- -ly, -ify, -ia (Spotify, Nvidia)
- Pro-, Meta-, Ultra-
- -ium, -ology, -ware

### Modification Techniques

- Deliberate misspellings (Flickr, Tumblr, Lyft)
- Letter substitutions (k for c, x for s)
- Truncation (removing syllables)
- Blending/portmanteau creation

---

## Name Generation by Type

### Creating Invented Names

1. Start with meaningful root sounds
2. Combine phonemes that feel right for brand
3. Test pronunciation across languages
4. Verify no negative meanings globally

**Sound Symbolism:**
- Front vowels (i, e): Small, light, fast
- Back vowels (o, u): Large, heavy, slow
- Hard consonants (k, t, p): Strong, sharp
- Soft consonants (l, m, s): Smooth, gentle

### Creating Compound Names

1. List words related to benefit/category
2. Combine meaningful word pairs
3. Test for clarity and memorability
4. Check domain and trademark

## Screening Framework

### Initial Filter

- Offensive in any major language?
- Already famous brand name?
- Impossible to pronounce?
- Obvious trademark conflict?

## Testing Methods

### Linguistic Testing

- Native speaker pronunciation
- Meaning in target languages
- Cultural appropriateness
- Phonetic analysis

### Audience Testing

- Surveys: Memorability, likability
- Focus groups: Emotional reactions
- A/B testing: Conversion comparison

### Questions to Ask

- Can you recall the name after hearing once?
- Can you spell it after hearing it?
- What does it make you think of?
- Do you like it? Why/why not?
- Does it feel right for this type of company?
