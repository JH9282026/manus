# Verbal Identity System

Complete framework for verbal brand elements.

---

## Tagline Development

### Tagline Types

| Type | Description | Example |
|------|-------------|--------|
| Imperative | Command/action | "Just Do It" (Nike) |
| Descriptive | What you do | "The Ultimate Driving Machine" (BMW) |
| Provocative | Challenge thinking | "Think Different" (Apple) |
| Specific | Unique claim | "15 minutes could save you 15%" (Geico) |
| Superlative | Best/most claim | "World's Greatest Shave" (Gillette) |

### Tagline Criteria

- Short (3-7 words ideal)
- Memorable
- Distinctive
- Brand-aligned
- Timeless
- Ownable (trademarkable)

## Nomenclature Systems

### Naming Architecture Models

**Apple Approach**: Prefix-based
- iPhone, iPad, iMac, iOS

**Amazon Approach**: Standalone memorable
- Prime, Alexa, Kindle, Echo

**Google Approach**: Parent + descriptor
- Google Maps, Google Drive, Google Photos

**Microsoft Approach**: Suite naming
- Microsoft 365, Microsoft Teams, Microsoft Edge

### Product Naming Strategies

| Strategy | Description | Example |
|----------|-------------|--------|
| Descriptive | States what it is | MacBook Pro |
| Alphanumeric | Codes and numbers | BMW 3 Series |
| Sequential | Numbered versions | PlayStation 5 |
| Thematic | Consistent theme | Android versions (candy) |

## Voice & Tone

### Voice Definition

**Voice**: Consistent personality (doesn't change)
**Tone**: Adaptation to context (changes by situation)

### Voice Attributes Framework

Define 3-5 attributes:

```
We are [ATTRIBUTE 1] but not [opposite extreme].
We are [ATTRIBUTE 2] but not [opposite extreme].
We are [ATTRIBUTE 3] but not [opposite extreme].
```

Example:
- "We are confident but not arrogant"
- "We are witty but not silly"
- "We are expert but not condescending"

### Tone Adaptation

| Context | Tone Shift |
|---------|------------|
| Celebration | More enthusiastic |
| Crisis | More empathetic, direct |
| Education | More patient, clear |
| Sales | More confident, urgent |

## Messaging Hierarchy

### Structure

1. **Brand Positioning Statement**: Core strategic position
2. **Tagline**: Memorable expression of essence
3. **Value Proposition**: Clear customer benefit
4. **Key Messages**: Supporting pillars (3-5)
5. **Proof Points**: Evidence for each message
6. **Elevator Pitch**: 30-60 second summary
