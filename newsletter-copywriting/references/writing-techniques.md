# Newsletter Writing Techniques

## Subject Line Optimization

### Character Count Guidelines
- **Desktop**: 60 characters visible
- **Mobile**: 30-40 characters visible
- **Sweet spot**: 6-10 words, 40-50 characters

### Power Words
| Category | Words |
|----------|-------|
| Curiosity | Secret, surprising, unexpected, hidden |
| Urgency | Now, today, limited, deadline, last chance |
| Value | Free, exclusive, proven, essential, ultimate |
| Personal | You, your, personalized |
| Numbers | Specific numbers outperform vague claims |

### A/B Testing Priorities
1. Subject line (highest impact on opens)
2. Send time and day
3. From name
4. Preview text
5. CTA placement and copy

## Email Body Writing

### Opening Hook Techniques
- **Bold claim**: Start with a surprising statement
- **Story**: Open with a brief anecdote (2-3 sentences)
- **Question**: Ask something the reader can relate to
- **Statistic**: Lead with a compelling data point
- **Controversy**: Challenge conventional wisdom

### Writing Style Guidelines
- Use short paragraphs (1-3 sentences)
- Write in first person ("I" or "we")
- Use conversational tone
- Include one clear CTA per email
- Break up text with subheadings
- Bold key phrases for scanners
- Keep total word count under 800 for weekly sends

### CTA Best Practices
| Element | Best Practice |
|---------|-------------|
| Placement | Above the fold + end of content |
| Copy | Action verb + benefit ("Get the template") |
| Design | Button format, contrasting color |
| Quantity | 1 primary CTA per email |
| Urgency | Add deadline or scarcity when appropriate |

## Voice and Tone Development

### Voice Attributes Framework
Define 3-4 consistent voice attributes:
- Example: Knowledgeable, approachable, witty, direct

### Tone Adaptation
| Content Type | Tone Adjustment |
|-------------|----------------|
| Educational | Helpful, clear, encouraging |
| News/trends | Authoritative, timely, concise |
| Promotional | Enthusiastic, benefit-focused |
| Personal story | Authentic, vulnerable, relatable |
| Urgent/crisis | Direct, empathetic, action-oriented |

## Engagement Recovery

### Re-engagement Sequence (For Inactive Subscribers)
1. **Email 1**: "We miss you" + best recent content
2. **Email 2**: Survey — "What would you like to read?"
3. **Email 3**: Special offer or exclusive content
4. **Final**: "Should we remove you?" (last chance)
5. **Cleanup**: Remove non-responders after 30 days

### Engagement Boosters
- Ask questions and invite replies
- Run polls and surveys
- Share subscriber wins and stories
- Create referral incentives
- Experiment with send times
