# Historical Analysis Methods

## Frameworks
- **Chronological narrative**: reconstruct events with causal connections; identify turning points
- **Comparative**: select cases, examine parallels/divergences, synthesize patterns
- **Periodization**: divide timeline by technology eras, regulatory regimes, leadership tenures
- **Counterfactual**: test causal claims via "what if" scenarios

## Pattern Recognition
- Cycles: recurring phases (boom/bust, innovation cycles)
- Trends: long-term directional changes
- Disruptions: sudden breaks from established patterns
- Path dependencies: how early decisions constrain later options

## Quantitative Historical Analysis
- Compile time-series data across decades
- Statistical trend analysis for structural breaks
- Build historical analogues for current conditions
- Pattern-informed scenario projections

## Quality Standards
- Full archival references for primary sources
- Reputable secondary scholarship
- Historical context for terminology and institutions
- Explicit limitation acknowledgment
