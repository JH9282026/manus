# Multi-Camera Editing

Workflow for editing multi-angle footage.

---

## Pre-Editing Setup

### Import and Organize

1. Import all camera angles
2. Label clips clearly (Camera A, B, C)
3. Organize in bins by camera
4. Note any sync issues

### Synchronization Methods

| Method | Accuracy | Use Case |
|--------|----------|----------|
| Timecode | Most accurate | Professional shoots |
| Audio waveform | Very good | Most situations |
| In/out points | Manual | Visual markers |
| Markers | Manual | Sync points |

---

## Creating Multi-Cam Sequence

### Setup Steps

1. Select all synced clips
2. Create multi-camera source sequence
3. Verify synchronization accuracy
4. Set playback to multi-camera view

### Sequence Settings

- Match primary camera settings
- Enable multi-camera playback
- Configure preview layout

---

## Switching Approaches

### Live Switching

Watch multi-cam in real-time, cut as you watch:
- Creates first rough cut quickly
- Natural timing and reactions
- May miss optimal moments
- Good for initial assembly

### Post-Switching

Review each angle separately, decide deliberately:
- More control over timing
- Slower but more precise
- Better for final cut
- Catches missed moments

### Hybrid Approach

- Live switch for rough cut
- Refine in post-switching mode
- Most common professional workflow

---

## Angle Selection Best Practices

### Cut for Content

- Prioritize meaningful changes
- Match emotional intensity to angle
- Avoid cutting just for variety
- Every cut should have purpose

### Common Mistakes

- Cutting too frequently
- Missing best angles
- Inconsistent eye lines
- Jump cuts between similar angles

---

## Continuity Management

### Track Elements

- Speaker position
- Hand gestures
- Props and objects
- Background activity

### Screen Direction

- Maintain consistent direction
- Watch for crossing the line
- Use cutaways to bridge problems

---

## Audio Considerations

### Primary Audio

- Usually use single audio source
- Typically boom or lavalier
- Avoid mixing multiple room sounds

### Multi-Cam Audio Issues

- Phase problems from multiple mics
- Room tone differences
- Sync drift over long recordings

### Solutions

- Use one primary audio track
- Mix separate tracks if needed
- Check sync periodically
