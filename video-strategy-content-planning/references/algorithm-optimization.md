## Algorithm Optimization


### Platform Algorithm Fundamentals

While algorithms differ, common signals include:

**Engagement signals**:
- Click-through rate
- Watch time
- Engagement (likes, comments, shares)
- Session time (what viewers watch after)

**Quality signals**:
- Retention rate
- Return viewership
- Subscriber conversion
- Viewer satisfaction surveys


### Working With Algorithms

**Strategies that work**:
- Optimize for viewer satisfaction, not algorithm gaming
- Strong hooks and compelling thumbnails
- Consistent posting schedule
- Build genuine community engagement
- Focus on session time, not just single video performance

**What to avoid**:
- Clickbait that disappoints viewers
- Engagement bait (asking for likes before value)
- Inconsistent posting damaging momentum
- Copying without adding unique value
